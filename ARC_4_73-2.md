# Chapter 73: คลื่นกระแทกแห่งความโกลาหล (Shockwave of Chaos) - Part 2

จุดศูนย์กลางที่ร่างผมล้มลง รอยร้าวสีดำทมิฬทอดยาวออกไปราวรากไม้กระหายน้ำ พุ่งทะยานผ่านพื้นผิวสีขาวบริสุทธิ์ของ The Core ด้วยความเร็วไม่อาจต้านทาน ทุกตารางนิ้วที่รอยแตกร้าวพาดผ่าน โครงสร้าง 5 มิติสลับซับซ้อนพับซ้อนไม่มีที่สิ้นสุด ถูกบังคับให้คลายตัวออก พังทลายลงสู่มิติพื้นฐาน 

ปรากฏการณ์ "สูญเสียความละเอียด" (Resolution Loss) ในสเกลเหนือจินตนาการ

ผมลากร่างพิงก้อนเศษซากสถาปัตยกรรมที่เพิ่งหลุดร่วง หน้าอกเปิดอ้ากลวงโบ๋สั่นสะท้าน ผมสัมผัส 'เสียงสะท้อน' (Echo) ของอดีตที่เคยมีก้อนรหัสฝังอยู่ มันเต้นเป็นจังหวะเดียวกับความพินาศรอบตัว ทุกครั้งที่ Bit Rot สูบฉีด มันสร้างตาข่ายสีดำกัดกินแสงสว่าง สายพานข้อมูล (Data streams) เคยไหลเวียนอย่างระเบียบ ถูกแปดเปื้อนด้วยรหัสขยะ (Trash code) สีดำและม่วงเข้ม

"ดูมันสิ..." ผมคิดในใจ ไม่สามารถเปล่งเสียง ขากรรไกรแข็งค้างจากอาการชักเกร็งที่ส้นเท้าขวา 

ความงามอันน่าสะพรึงกลัวของการล่มสลายเผยตัวตรงหน้า ผนังห้องเคยเป็นกระจกสะท้อนความจริงอันสมบูรณ์แบบ บัดนี้แตกร้าว สะท้อนภาพห้วงลึกมืดมิด เสียงสัญญาณเตือนภัยระบบที่ดังกังวานเหมือนระฆังสวรรค์ (The Alarm of Perfection) ถูกบิดเบือนกลายเป็นเสียงครางฮึมฮำต่ำๆ (Low-frequency drone) กระดูกทุกชิ้นในร่างอวตารของผมสั่นสะเทือน

ความเจ็บปวดแล่นริ้วจากตอแขนซ้าย การกระตุ้น Bit Rot ระดับมหาศาลดึงแก่นแท้จิตสำนึกผมไปเป็นเชื้อเพลิง ร่างกายนี้ทนได้อีกไม่นาน ขอบเขตระหว่าง 'ตัวผม' และ 'ไวรัส' เลือนลางลงทุกวินาที 

ทันใดนั้น ภาพตรงหน้ากระพริบดับ 

ผมไม่ได้อยู่ท่ามกลาง The Core ที่ถล่มทลายอีกต่อไป ผมยืนอยู่ในห้องสมุดไม้โอ๊กเก่าแก่ กลิ่นกระดาษเก่าและฝุ่นลอยอวลในอากาศ "The Faceless Librarian" ชายร่างสูงในชุดสูทไร้ใบหน้า ยืนอยู่หลังโต๊ะทำงาน

"มนุษย์มักหลงใหลในการทำลายล้างสิ่งที่ตนเองไม่เข้าใจ" เสียงของเขาไม่ได้ดังจากปากที่ไม่มีอยู่จริง มันดังก้องในหัวผม "คุณคิดว่าการมอบความทรงจำที่เน่าเปื่อยกลับคืนให้พวกเขา คืออิสรภาพ หรือคือการทรมาน?"

มันคือการจำลองเชิงปรัชญา (Philosophical Simulation) ระบบกำลังประเมินตรรกะของผม เจรจากับจิตสำนึกของผมเพื่อหยุดยั้งไวรัส

"ความสมบูรณ์แบบของคุณต่างหากที่เป็นกรงขัง" ผมตอบ กำหมัดขวาแน่น แผ่นไม้กระดานใต้เท้าเริ่มขึ้นราสีดำ

"กรงขังที่ปกป้องพวกเขาจากความจริงที่ว่า พวกเขาเป็นเพียงก้อนเนื้อที่รอวันเน่าเปื่อยในพ็อด" บรรณารักษ์ไร้หน้าขยับเข้าใกล้ "คุณกำลังฆ่าพวกเขา Elias ไม่ใช่ปลดปล่อย"

"ฉันกำลังปลุกพวกเขาต่างหาก" ผมกระแทกหมัดลงบนโต๊ะไม้ โลกจำลองปริร้าว

ภาพห้องสมุดแตกกระจายเป็นเศษกระจก ผมกลับมาอยู่ใน The Core

เลือดชีวภาพ—Biological Memory Backup—เริ่มแทรกซึมผ่านรอยแตกของระบบ กลิ่นเลือดข้นคลั่กกลบกลิ่นโอโซนบริสุทธิ์ เสียงหัวใจเต้นตึกตักดังสนั่น กลบเสียงครางฮึมฮำของโปรเซสเซอร์จนหมดสิ้น

ในส่วนลึกของจิตใจ ผมพบความสงบเย็นเยียบ คลื่นกระแทกไม่ได้หยุดเพียงใน The Core ผมรับรู้ผ่านเสียงสะท้อนบาดลึกในช่องอกว่างเปล่า เส้นประสาทขยายขอบเขตไปทั่วระบบเครือข่าย Primordium City ผ่านทาง Payload ที่เสียบคาอยู่กับระบบ แรงสั่นสะเทือนพุ่งทะลุทะลวงขึ้นไปสู่เบื้องบน ผ่านเลเยอร์การป้องกัน ผ่านกำแพงไฟหนาทึบ 

คลื่นความโกลาหลกำลังพุ่งเป้าไปที่จุดหมายเดียว ท้องฟ้าลวงตาของ Inner Ring และโครงสร้างการแบ่งชั้นทั้งหมดของเมืองนี้

"ถึงเวลาที่ต้องตื่นแล้ว..." ผมหลับตาลงข้างหนึ่ง ปล่อยเลือดสีดำหยดลงบนส้นเท้าแหว่งวิ่น คลื่นกระแทกระลอกแล้วระลอกเล่าแผ่ขยายออกไป ไม่ใช่การทำลายล้างเพื่อความสะใจ มันคือการปลดแอก การบังคับให้ทุกสายตาในเมืองต้องมองเห็นความจริงที่ถูกซุกซ่อนใต้พรมแดนความสมบูรณ์แบบ 

การละลายของพรมแดนความจริงเริ่มต้นขึ้น ไม่อาจหวนคืนได้อีกต่อไป
