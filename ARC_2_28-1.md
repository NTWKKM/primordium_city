# บทที่ 28: ความทรงจำที่ถูกขโมย (Part A)

เสียงแรกที่ผมได้ยินไม่ใช่เสียงระเบิด แต่มันคือเสียงฉีกขาด—เหมือนผ้าใบผืนใหญ่ที่ถูกกรีดด้วยใบมีดที่มองไม่เห็นกลางอากาศ ผมสะดุ้งสุดตัวขณะที่คลื่นความสั่นสะเทือนทางข้อมูลกระแทกเข้าที่หน้าอกอย่างจัง แรงอัดนั้นชนเข้ากับก้อน Payload ที่ฝังอยู่จนมันสะท้อนเสียงกังวานหนักอึ้งราวกับระฆังเหล็ก ผมทะลึ่งพรวดลุกขึ้นยืนด้วยความตื่นตระหนก แต่กลับต้องผงะหงายหลังและเสียหลักล้มลงเพราะส้นเท้าขวาที่แหว่งหายไปไม่สามารถรับน้ำหนักได้ 

ความละเอียดรอบตัวกระพริบถี่ยิบ กลิ่นอับของข้อมูลที่ถูกบีบอัดผสมกับกลิ่นโอโซนไหม้คละคลุ้งไปทั่วทางเดินแคบๆ ของเขต Mid-Sectors

บนพื้นคอนกรีตสังเคราะห์ ปรากฏรอยแยกของข้อมูลที่ทิ้งคราบสีดำสนิทไว้ มันเดือดปุดๆ ราวกับกรดที่กำลังกัดกร่อนรหัสพื้นฐานของระเบียงทางเดิน 

แล้วเสียงร้องไห้ก็ดังขึ้น มันไม่ใช่เสียงสะอื้นของมนุษย์ แต่เป็นเสียงที่ถูกระบบบีบอัดจนสูญเสียย่านความถี่ต่ำไปหมด เหลือเพียงเสียงหวีดแหลมสูงปรี๊ดที่เสียดแทงทะลุแก้วหู—เสียงหวีดร้องของจิตสำนึกที่ถูกพรากบางสิ่งไปอย่างกะทันหัน

"เกิดอะไรขึ้น!" ผมตะโกน แต่น้ำเสียงของผมเองก็สั่นพร่า ร่างกายดิจิทัลของผมตอบสนองต่อความตื่นตระหนกด้วยการเกิดพิกเซลแตกซ่าน (Artifacts) ที่ปลายนิ้ว ขณะเดียวกัน แขนซ้ายที่ติดเชื้อ Bit Rot ก็กระตุกและชาหนึบ ตอบสนองช้ากว่าปกติจนแทบจะยกไม่ขึ้น

มีคนถูกปล้น ไม่ใช่ทรัพย์สิน แต่เป็นความทรงจำ การโจรกรรมข้อมูลความทรงจำครั้งใหญ่เพิ่งเกิดขึ้นใจกลางชุมชน ผู้คนวิ่งพล่านด้วยความสับสน ร่างกายของบางคนกะพริบติดดับราวกับไฟนีออนที่กำลังจะขาดใจ เสียงสแกนความถี่ต่ำของ UX Index ดังก้องอยู่ในหัวผมอีกครั้ง มันสัมผัสได้ถึงความโกลาหลและกำลังพุ่งเป้าค้นหา 'ความผิดปกติ' อย่างผมในรัศมีนี้

[Note for Editor: เน้นย้ำภาพของ "เสียงที่ถูกบีบอัด" เพื่อแสดงให้เห็นถึงกลไกทางดิจิทัลที่ส่งผลต่อการรับรู้ทางชีวภาพของมนุษย์]
