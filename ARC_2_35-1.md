# บทที่ 35: การหักหลังของความคมชัด (Part 1)

ความสงบสุขใน Mid-Sectors เป็นเพียงภาพลวงตาที่เปราะบาง แม้ผมจะพบความอบอุ่นในความพร่าเลือนร่วมกับไลร่า แต่ลึกลงไปในแกนกลางของวิญญาณ ความโหยหาต่อ "ความสมบูรณ์แบบ" แบบมนุษย์ยังคงฝังรากลึก มันเป็นจุดอ่อนที่อัลกอริทึมแห่งความเศร้าไม่เคยลบออกไปได้หมด

และนั่นคือจุดที่ 'พวกเก็บกวาด' (Scavengers) เข้ามาทำกำไร

ผมพบเขาในตรอกแคบๆ ที่ล้อมรอบด้วยกำแพงข้อมูลที่พังทลาย ชายร่างผอมสูงที่สวมเสื้อคลุมทำจากโค้ดสีดำทึบ เขาไม่ได้ขายความทรงจำเหมือนพ่อค้าในตลาดนัด แต่เขากำลังเสนอสิ่งที่อันตรายกว่านั้น... "Clarity Patch" หรือที่ในตลาดมืดเรียกว่า 'หยดความคมชัด'

"นายอยากจำได้ไหมว่าสีฟ้าของท้องฟ้าที่แท้จริงเป็นยังไง?" เสียงของเขาลื่นไหลและไร้สัญญาณรบกวน ซึ่งแปลกประหลาดมากสำหรับคนที่อยู่ในเขตนี้ "นายอยากสัมผัสลมหายใจที่เต็มปอด อยากรู้สึกถึง 'ความรัก' แบบที่ไม่ได้ถูกบีบอัดไหม? แค่แพตช์นี้ตัวเดียว นายจะได้กลับไปสัมผัสความละเอียดระดับ 32K อีกครั้ง... แม้จะแค่ชั่วคราวก็ตาม"

เขายื่นหลอดแก้วเรืองแสงขนาดเล็กมาตรงหน้าผม ภายในนั้นบรรจุของเหลวสีน้ำเงินเข้มที่ดูบริสุทธิ์ผุดผ่อง มันดูขัดแย้งกับความสกปรกและ Noise รอบตัวเราอย่างสิ้นเชิง

ความโลภก่อตัวขึ้นในอกของผมอย่างเงียบๆ ผมนึกถึงลูกบาศก์สีทองในห้องสมุด นึกถึงความรู้สึกเอ่อล้นที่ผมสูญเสียไป นึกถึงสัมผัสที่แท้จริงที่ผมกระหายอยากจะได้กลับคืนมา แม้ไลร่าจะสอนให้ผมยอมรับความไม่สมบูรณ์ แต่ส่วนลึกของผมยังคงเป็นทาสของความคมชัด

"ต้องแลกด้วยอะไร?" ผมถาม เสียงของผมสั่นเล็กน้อย
"แค่ Cycle การประมวลผลพื้นหลังของนายชั่วคราวเท่านั้น ไม่มีอะไรต้องเสีย" เขายิ้ม

มันเป็นข้อเสนอที่ดูดีเกินไป แต่ความโหยหาทำให้ตรรกะของผมพร่ามัว ผมยื่นมือขวาที่เต็มไปด้วยรอยช้ำสีม่วงออกไปรับหลอดแก้วนั้นมา ระวังไม่ให้แขนซ้ายที่ไร้ประโยชน์และเป็นพาหะของ Bit Rot สัมผัสโดนอะไร โดยไม่รู้เลยว่าผมกำลังเปิดประตูรับหายนะเข้ามาสู่แกนกลางของตัวเอง
