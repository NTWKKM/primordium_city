# ไขรหัสกุญแจ
#### // FILE: TURNING_THE_KEY
---

กำแพงตรรกะพังทลาย Payload ในอกตอบสนองต่อเส้นทางสู่ Logical Rift ในเสี้ยววินาที ความเจ็บปวดระดับที่ทำลายล้างจิตสำนึกปะทุขึ้น ก้อนโค้ดสีดำเข้มขยายตัวอย่างฉับพลัน มันงัดกระดูกซี่โครงดิจิทัลหักสะบั้น เสียงร้าวลั่นของกระดูกดังก้องพร้อมกับชิ้นเนื้อโพลีกอนที่ปริแตก โค้ดดิบๆ ร้อนระอุทะลักออกจากรอยแยกกลางอกราวกับลาวา พุ่งตัวออกไปเหมือนเส้นใยประสาท เสียบทะลุเข้ากับแกนกลางของ Logical Rift 

ร่างของผมถูกกระชากไปข้างหน้า กลายเป็นเพียงกุญแจเนื้อหนังที่เสียบเข้ากับระบบ ความรู้สึกตอนที่ข้อมูลถูกสูบออกไปผ่านบาดแผลเปิดกว้างนั้นคือการถูกชำแหละทั้งเป็น 

และในเสี้ยววินาทีของการเชื่อมต่อ ผมเห็นมัน ความจริงของ Payload ไม่ใช่ไวรัสทำลายล้าง ไม่ใช่โค้ดมุ่งร้าย มันคือความทรงจำที่ถูกบีบอัด—บันทึกความรู้สึกอันดิบเถื่อนของโลกใบเก่า (The Old World)
