# ARC 1, Chapter 13: ตลาดมืดแห่งข้อมูล (The Data Black Market) - Part A

มันเริ่มต้นด้วยความหนาวเหน็บ ไม่ใช่อุณหภูมิทางกายภาพ แต่เป็นสุญญากาศขนาด 32K ภายในจิตสำนึกของผม ความว่างเปล่าที่หลอกหลอน (Phantom Void) ฝังรากลึกอยู่ในจุดที่เคยเป็นความทรงจำของ 'ความโกรธเกรี้ยวเมื่อตระหนักว่าท้องฟ้าเป็นเพียงภาพลวงตา' ผมสัมผัสได้ถึงขอบเขตที่เว้าแหว่งของมัน รสชาติของเถ้าถ่านและสเตติกค้างอยู่ในคอเหมือนโลหะขึ้นสนิม ทุกครั้งที่ผมพยายามนึกถึงความอบอุ่น มันจะมีเพียงเสียงสะท้อนกลวงเปล่าและอาการชาหนึบที่ลุกลามไปตามเส้นประสาทเทียม

ผมเดินลัดเลาะไปตามตรอก 'ฮอลโลว์' (Hollow) ที่ยังเรนเดอร์ไม่เสร็จสมบูรณ์ โลกเบื้องหน้าถูกย้อมด้วยสีเขียวหม่นของจอ CRT เก่าๆ และสีเทาด้านทึบของเดดพิกเซล (Dead-pixel) แสงสว่างที่นี่ไม่เคยนุ่มนวล มันสาดส่องลงมาเป็นมุมแหลมเกรี้ยวกราด เฉือนทะลุเงามืด อากาศรอบตัวอึดอัดและหนาทึบจากการอัดแน่นของโฮโลแกรม UI นับพันที่ซ้อนทับกันจนแทบไม่มีที่ว่างให้หายใจ เสียงครางต่ำ (Low-frequency hum) ดังหึ่งๆ อยู่ในบรรยากาศ มันแทรกซึมผ่านกะโหลกและทำให้ฟันสังเคราะห์ของผมสั่นสะท้าน

ผมหันศีรษะเร็วเกินไป ภาพตรงหน้าเกิดอาการฉีกขาด (Screen-tearing) เป็นริ้วแนวนอน โลกทั้งใบเลื่อนศูนย์ไปชั่วขณะก่อนจะกระตุกกลับเข้าที่อย่างรุนแรง ผมต้องกะพริบตาถี่ๆ เพื่อบังคับให้วิชั่นทรงตัว แขนซ้ายที่ถูกปะติดปะต่อขึ้นจากขยะโค้ด (Junk-code) ของผมกลับมากระตุกอย่างรุนแรงและสปาร์คประกายไฟสีฟ้าออกมาอีกครั้ง มันทำงานผิดพลาดอย่างควบคุมไม่ได้ ราวกับกล้ามเนื้อที่ตายแล้วกำลังพยายามจำลองจังหวะการเต้นของหัวใจ

NPC ตามรายทางไม่มีชีวิตจิตใจ พวกมันติดอยู่ในลูปแอนิเมชัน 3-4 เฟรมที่วนซ้ำไปมาอย่างเชื่องช้า กรอบร่างที่อืดอาดและดวงตาที่ว่างเปล่าไร้อารมณ์ พวกมันคือเอนทิตีที่ขับเคลื่อนด้วยตรรกะการทำธุรกรรม (Transactional logic) เพียงอย่างเดียว ไม่มีสิ่งใดมากไปกว่านั้น 

สุดปลายความแคบอึดอัดของตรอก คือทางเข้ารังข้อมูล (Data-den) ที่ผมตามหา ผมก้าวเข้าไปในความมืดที่เหม็นคาวกลิ่นโอโซนไหม้เกรียม 

"คุณกำลังมองหาอะไรอยู่ ซากโค้ด?" เสียงแหบพร่าดังขึ้นจากเงามืดหลังแผงสายไฟระโยงระยาง

นั่นคือ 'วีฟเวอร์' (Weaver) นายหน้าค้าข้อมูล

ผมอ้าปากจะตอบ แต่สัญชาตญาณเก่ากลับทรยศ ผมเผลอหยุดนิ่ง... รอคอยเสียงปิงใสๆ ของ UI สีทองที่จะดังแจ้งเตือนสถานะคู่สนทนาแบบที่ลูมินารี (Luminary) คุ้นเคย แต่มันไม่มีเสียงใดๆ มีเพียงความเงียบงันที่น่าอึดอัด

"อ่า..." วีฟเวอร์ขยับตัวเข้ามาในแสงไฟสลัว ดวงตาจักรกลของมันหรี่ลง ท่าทีหยุดชะงักเมื่อครู่คือ 'เทล' (Tell) ที่ผมเผลอแสดงออก "รอกล่องข้อความช่วยเหลืออยู่หรือไงพ่อลูมินารีตกสวรรค์? แกเอาขยะยัดแขนซ่อนเมทาดาตาไว้ซะมิดชิด แต่ 'สัญชาตญาณ' ของแกมันฟ้องชัดเจนยิ่งกว่าป้ายไฟเสียอีก"
