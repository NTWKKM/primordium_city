# Chapter 58: ปฏิเสธความสมบูรณ์แบบ (Rejecting Perfection)

"หุบปาก..." ผมเค้นเสียงผ่านไรฟัน รสชาติฝาดเฝื่อนของเลือดดิจิทัลคลุ้งในปาก คลื่นแสงของ UX Index ลามเลียถึงส้นเท้า ความร้อนของมันลวกไหม้ผิวหนังจำลอง สัญญาณเตือนภัยในหัวกรีดร้องถึงการลบเลือน ขาขวาสั่นสะท้านจนแทบทรุด การฝืนเกร็งกล้ามเนื้อที่ฉีกขาดบังคับน้ำหนักทั้งหมดลงบนขาซ้าย ขาขวาที่แหว่งวิ่นทำหน้าที่เพียงค้ำยันไม่ให้ร่างร่วงหล่นลงไปในความสมบูรณ์แบบที่กำลังคืบคลานเข้ามา

"นายสู้ไปเพื่ออะไร? ทนทรมานเพื่อระบบที่ทอดทิ้งนายงั้นหรือ?" เงาในกระจกยังคงเยาะเย้ย แสงสว่างด้านหลังขับเน้นให้มันดูเหมือนเทวทูตแห่งการยอมจำนน

"เพราะฉันยัง... มีชีวิต!" ผมแผดเสียง ทรวงอกสะท้านหอบหนักหน่วง "ความเจ็บปวดนี้... แขนที่หายไป... เลือดที่ไหลริน... มันคือข้อพิสูจน์ว่าฉันไม่ใช่รหัสที่ว่างเปล่า! ฉันคือมนุษย์! และมนุษย์ไม่เคยก้มหัวให้ความตาย!" 

การแบกรับความเจ็บปวดย่อมดีกว่าการเป็นเพียงข้อมูลขยะที่ถูกฟอร์แมต
