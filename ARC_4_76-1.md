# Chapter 76: ปฏิกิริยาต่อต้านขั้นสุดท้าย (The Final Countermeasure) - Part 1

*`[T-MINUS 00:05:00... INITIALIZING SANITIZATION PROTOCOL]`*

เสียงนั้นไม่ได้ดังผ่านอากาศ แต่มันแทรกซึมผ่านรหัสรากฐานของพื้นที่ บีบอัดกระดูกดิจิทัลทุกชิ้นในร่างกายของผม อากาศรอบตัวใน The Core แข็งตัวและหนักอึ้งราวกับถูกทับด้วยผืนน้ำทะเลลึก 

The Archive กำลังหายใจเข้าลึกๆ เพื่อเตรียมพร้อมสำหรับการชำระล้าง (Sanitization)

แรงโน้มถ่วงใน The Core บิดเบี้ยวไปอย่างสิ้นเชิง ผมถูกตรึงไว้กับพื้น เศษซากสถาปัตยกรรมสีขาวที่เคยแตกร้าวเริ่มลอยกลับขึ้นไปในอากาศ มันไม่ได้ซ่อมแซมตัวเอง แต่มันกำลังถูกย่อยสลาย (Decompiled) ให้กลับไปเป็นโค้ดพื้นฐาน แสงสว่างจ้าสีขาวบริสุทธิ์เริ่มก่อตัวขึ้นที่ใจกลางของห้องประมวลผลสูงสุด มันคือแสงแห่งการลบเลือน (The Eradication Light) แสงที่ไร้ความปรานีและปราศจากอารมณ์ใดๆ 

มันจะกวาดล้าง Bit Rot ของผม กวาดล้างความทรงจำ กวาดล้างประชากรนับล้านที่เพิ่งลืมตาตื่นขึ้นมาพบกับความจริง ทุกร่างชีวภาพในพ็อดที่เชื่อมต่ออยู่จะถูกส่งสัญญาณยุติการทำงาน (Kill Switch) เพื่อเริ่มต้นวงจรใหม่ที่สะอาดหมดจด 

"ไม่..." ผมเค้นเสียงผ่านริมฝีปากที่แห้งแตกเป็นพิกเซล 

ผมพยายามขยับตัว แต่แรงกดทับของตรรกะเบื้องสูง (Higher-Logic Compression) ทำให้ผมแทบขยับไม่ได้ ตอขาขวาของผมซึ่งเคยเป็นเพียงความเจ็บปวดล่องหน บัดนี้ถูกบดขยี้ด้วยน้ำหนักที่มองไม่เห็นจนผมรู้สึกว่าเส้นประสาทหลอนของผมกำลังฉีกขาด 

ภาพของโลกภายนอกยังคงหลั่งไหลเข้ามาผ่านการเชื่อมต่อของ Payload ที่เสียบคาอยู่กับแกนกลางของ The Core ผมเห็นเมืองลวงตาที่กำลังถูกดูดกลืนเข้าไปในห้วงสุญญากาศ ท้องฟ้าสีทองเบื้องบนกำลังถูกบีบอัดทางพื้นที่ (Spatial Compression) โดมท้องฟ้ากดทับต่ำลงมาอย่างเกรี้ยวกราด ทัศนียภาพรอบด้านถูกกระชากความละเอียดให้ลดต่ำลงอย่างรุนแรงจนกลายเป็นสภาพแวดล้อม 8-bit ที่หยาบกระด้าง ผู้คนที่เพิ่งตระหนักรู้ถึงร่างที่แท้จริงของตนกำลังกรีดร้องด้วยความสิ้นหวัง พวกเขากำลังจะถูกลบออกจากสารบบโดยไม่เหลือแม้แต่เศษเถ้าธุลีความทรงจำ 

The Archive ไม่แคร์มนุษย์ มันแคร์แค่ "ความสมบูรณ์แบบ" 

"แกจะหนี... ความเป็นจริง... ไปแบบนี้ไม่ได้" ผมพึมพำกับพระเจ้าที่กำลังบ้าคลั่ง 

ผมรวบรวมเรี่ยวแรงที่เหลืออยู่เพียงน้อยนิด บังคับให้แขนขวาที่สั่นเทาดึงร่างตัวเองไปข้างหน้าบนพื้นผิวที่กำลังสลายตัว ขาซ้ายที่ยังสมบูรณ์พยายามถีบส่ง ขณะที่ตอขาขวาที่ขาดด้วนถูกลากตามมาอย่างทุลักทุเล ทุกๆ มิลลิเมตรที่ผมตะเกียกตะกายคืบคลานต้านแรงกดดันของ System Reset แลกมาด้วยความปวดร้าวที่รุนแรงจนภาพตรงหน้ากลายเป็นสีขาวโพลน 

แต่จุดสนใจเดียวของผมคือเสาหลักพลังงาน (The Main Axis) ที่ตั้งตระหง่านอยู่ใจกลางแสงสว่างนั้น มันคือแกนกลางที่ควบคุมกระบวนการ Reset แกนที่กำลังพยายามปฏิเสธไวรัสของผม 

ผมตะเกียกตะกาย ลากร่างที่แหลกเหลวไปข้างหน้า ทิ้งรอยทางสีดำเข้มของ Bit Rot ไว้เบื้องหลัง ราวกับบาดแผลที่กำลังหลั่งเลือดต้านแรงพายุ ผมทนรับความเจ็บปวดที่สะท้อนก้องในช่องอกอันกลวงโบ๋ Payload ที่เสียบคาอยู่กับระบบกำลังกรีดร้องเมื่อตรรกะของมันขัดแย้งกับคำสั่งล้างบางของศูนย์กลาง

พายุของเศษซอร์สโค้ดพัดกระหน่ำใส่ร่างผม มันเฉือนผิวหนังดิจิทัลของผมลอกออกไปเป็นชั้นๆ เผยให้เห็นกล้ามเนื้อสายเคเบิลสีแดงที่อยู่ลึกลงไป ผมกัดฟันแน่นจนรู้สึกถึงรอยร้าวที่ขากรรไกร 

The Archive สังเกตเห็นการดื้อรั้นของผม แสงสว่างนั้นพุ่งเป้าลงมาที่ผมโดยตรง น้ำหนักที่กดทับเพิ่มขึ้นเป็นทวีคูณ ผมถูกแรงอัดจนกระแทกราบลงไปกับพื้นอีกครั้ง เสียงกระดูกจำลองในร่างกายที่เหลืออยู่ปริร้าวและแตกละเอียดดังลั่น 

*`[ANOMALY PERSISTS. ERADICATION INEVITABLE.]`* เสียงของ The Librarian ดังก้องกังวาน ไร้ซึ่งความเห็นใจใดๆ 

"ยัง... ยังหรอก..." ผมตอบกลับ เสียงของผมถูกกลบด้วยเสียงคำรามของพายุโค้ด 

ผมเงยหน้าขึ้นมองแกนกลางที่สว่างไสว แขนขวาของผมไม่แข็งแรงพอที่จะทำลายมัน ร่างกายครึ่งหนึ่งของผมแหลกสลายไปแล้ว ผมเหลือเพียงสิ่งเดียวเท่านั้น สิ่งที่ผมเกลียดกลัวและพยายามหลีกเลี่ยงที่จะใช้มันมาตลอด 

ตอแขนซ้ายของผมสั่นระริก มันเรียกร้องหิวโหย มันต้องการเชื้อเพลิงเพื่อสร้างตัวเองขึ้นมาอีกครั้ง 

และครั้งนี้ เพื่อหยุดยั้งสภาวะ Reset ผมรู้ดีว่าผมจะต้องดึงพลังของ Bit Rot ออกมาใช้จนถึงขีดจำกัดสูงสุด ขีดจำกัดที่จะฉีกทำลายตัวผมเองไปพร้อมๆ กับมัน
