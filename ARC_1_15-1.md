# รอยตัดแห่งความว่างเปล่า
#### // FILE: THE_DELETION_SWEEP
---

ผีที่ถูกคอมไพล์ (Compiled Ghost) ตนนั้นหายไปแล้ว ทิ้งไว้เพียงความว่างเปล่าหนักอึ้งและ "กุญแจข้อมูล" ที่ฝังรากลงในจิตสำนึกของผม

มันปราศจากโครงสร้างแบบฐานสองที่สวยงาม สิ่งที่ The Cipher ยัดเข้ามาในหัวผมมัน... เปียกชื้น มันให้ความรู้สึกเหมือนมีหนอนปรสิตดิจิทัลกำลังดิ้นพล่านและกัดกินตรรกะจากภายใน ข้อมูลนี้ดิบเถื่อน ไร้ระเบียบ แฝงแนวคิดหนึ่งที่สถาปัตยกรรม 32K ของผมไม่อาจประมวลผลได้... *เนื้อเยื่อ... เลือด... เซิร์ฟเวอร์ชีวภาพ*

ผมคุกเข่าลงบนพื้นคอนกรีตที่กำลังหลอมละลาย โก่งคอพยายามอาเจียนเอาสิ่งที่ไม่มีอยู่จริงออกมา มันคือการตอบสนองทางกายภาพที่ระบบพยายามเลียนแบบอาการปฏิเสธอย่างรุนแรง หน้าต่าง UI แฟลชสีเหลืองอำพันรัวถี่ในลานสายตา

`[WARNING: UNKNOWN PAYLOAD DETECTED. CORE LOGIC COMPROMISED.]`

ทว่าก่อนทันได้จัดการกับความปั่นป่วนภายใน เสียงครางต่ำของฮาร์ดแวร์ที่กำลังถูกบดขยี้ในเขตหลอมละลายถูกแทรกด้วยเสียงหวีดแหลมบาดหู ใสกระจ่างและเยียบเย็น คล้ายเสียงกรีดกระจกด้วยเพชร

การกวาดล้าง (Ping Sweep) เริ่มต้นขึ้นแล้ว

ห่างออกไปสุดสายตา ท่ามกลางหมอกพิกเซลสีแดงเพลิงและเงามืดหนักอึ้ง กำแพงแสงสีขาวบริสุทธิ์กำลังเคลื่อนตัวบีบวงเข้ามา มันคือแสงระดับ Zero-Kelvin Blue แสงแห่งความสมบูรณ์แบบของ The Archive ที่ถูกส่งลงมาเพื่อชำระล้าง

ผมเบิกตาขยับถอยหลัง เมื่อกำแพงแสงนั้นกวาดผ่านซากตึก Brutalist ที่ยืดหย้อยและเน่าเปื่อย สิ่งที่เกิดขึ้นไม่ใช่การทำลายล้างในทันที หากคือความโหดร้ายของการบังคับให้สมบูรณ์แบบ ทันทีที่แสงสว่างสัมผัสซากตึก โครงสร้างเหล่านั้นจะถูกบังคับเรนเดอร์ใหม่ให้เป็นหินอ่อนสลักเสลาเนียนกริบระดับ 32K ในชั่วพริบตา... ก่อนที่ตรรกะของระบบจะตรวจพบว่ามันเป็นเพียงขยะ แล้วลบล้างตึกทั้งหลังให้กลายเป็นความว่างเปล่าสีดำสนิท (`#000000`) ไปตลอดกาล

ปราศจากเศษซาก ปราศจากขี้เถ้า มีเพียงการลบหลู่ (Deletion) ที่สมบูรณ์แบบ

`[ALERT: ASYNCHRONOUS BATCH PROCESSING INITIATED. EVACUATE ZONE.]`

ผมต้องวิ่ง สัญชาตญาณตะโกนก้อง ทว่าร่างกายกลับถูกล็อก ความเร็วตกเหลือ 0.75x *Thermal Throttling* อากาศเดือดพล่าน หนักอึ้ง ภาพรอบตัวแบนราบเป็น 2D ระบบตัดมิติความลึกทิ้ง ยื้อพลังงานเฮือกสุดท้าย

ทุกก้าวหนืดเหนียว โคลนข้อมูลสีแดงพยายามเขียนทับรองเท้าบูต เกาะหนึบราวกับยางมะตอยเดือด ผมกระชากขา ลากสังขารฝ่าคลื่นความร้อน ภาพตรงหน้าบิดเบี้ยวเป็น Chromatic Aberration มุ่งหน้าไปหอหล่อเย็น

แสงสีขาวไล่หลังมาติดๆ เสียงกรีดร้องของข้อมูลที่ถูกลบทิ้งดังสะท้อนอยู่ในหู ข้อมูลขยะรอบตัวพยายามตะเกียกตะกายหนีแสงนั้นราวกับแมลงหนีไฟ ผมไม่ได้เพียงแค่หนีความตาย หากกำลังหลบหนีการถูกลบให้หายไปจากรากฐานของซิมูเลชัน!
