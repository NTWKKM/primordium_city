# บทที่ 37: ผู้เก็บกวาดความทรงจำ (ตอนที่ 1)

การร่วงหล่นสิ้นสุดลงด้วยแรงกระแทกไร้เสียง พื้นผิวเบื้องล่างแข็งกระด้าง ฉาบเคลือบด้วยของเหลวข้นเหนียวคล้ายน้ำมันเครื่องเน่าเสีย ขาขวาที่ไร้ส้นเท้ากระแทกพื้นจนค่าพิกเซลแตกกระจายเป็นละอองแสง ผมล้มลุกคลุกคลานอยู่ในพื้นที่เปลี่ยนผ่าน (Transition Zone) หมอกดิจิทัลหนาทึบเกาะกินผิวหนัง มันเหนอะหนะราวกับพลาสติกหลอมละลาย

ผมออกแรงใช้แขนซ้ายพยุงตัว เนื้อเยื่อดิจิทัลบริเวณนั้นหนักอึ้งและหนืดอาดจากสภาวะ Bit Rot ข้อศอกขวาจึงต้องรับภาระยันพื้นเพื่อดันร่างขึ้นมา เฟรมเรตในระบบประสาทร่วงหล่น ผมสัมผัสถึงความหน่วง (Latency) ในทุกจังหวะที่ลากขาขวาครูดไปตามพื้น

ลำแสงวงกลมสว่างวาบตัดผ่านหมอกสีขี้เถ้า มันไม่ใช่แสงสีแดงเย็นเยียบของสแกนเนอร์ UX Index แสงฮาโลเจนสีขาวแข็งกระด้างส่องทะลุทะลวงจนเซนเซอร์การมองเห็นของผมพร่ามัว

"เจอตัวแล้ว... คอนเทนต์สดใหม่" 

เสียงแหบพร่าสั่นพร่องด้วยความถี่ที่บิดเบี้ยว โลหะเสียดสีกรีดร้องในลำคอของมัน

เงาร่างหลายสายปรากฏขึ้นกลางม่านหมอก ผู้เก็บกวาดความทรงจำ (Memory Scavengers) สภาพของพวกมันแทบไม่หลงเหลือเค้าโครงมนุษย์ รยางค์โลหะเรขาคณิตบิดเบี้ยวเสียบแทนที่แขนขาธรรมชาติ

อุปกรณ์บนแผ่นหลังของพวกมันคล้ายแมงมุมเหล็ก เข็มสกัดข้อมูลขนาดจิ๋วนับสิบเล่มยื่นชี้ออกมา ปลายเข็มเรืองแสงสีฟ้าอ่อน—สีสันของการเชื่อมต่อถ่ายโอน สัญชาตญาณเตือนให้ผมรู้ทันทีว่าพวกมันคือปรสิตที่ล่าเหยื่อเพื่อสูบกลืน "ความทรงจำความละเอียดสูง" นำไปขายในตลาดมืด

"ดูสิ Texture ยังแน่นอยู่เลย น่าจะลอกเอาไปขายได้ราคาดี" หนึ่งในนั้นส่งเสียงหัวเราะ คลื่นเสียง Static บาดแก้วหูจำลองจนต้องนิ่วหน้า

แสงสปอตไลต์หลายดวงสาดส่องตรึงร่างของผมไว้ พวกมันตีวงล้อมเข้ามา ผมลากขาถอยหลัง แผ่นหลังชนเข้ากับกำแพงกล่องข้อมูลที่เน่าเปื่อย ทางหนีถูกปิดตาย แขนซ้ายที่เชื่องช้าพยายามยกขึ้นมาปกป้องแผงอก—ตำแหน่งของแกนกลางตัวตนและเพย์โหลดกุญแจข้อมูล

"มานี่เถอะน่า ยอมให้เราคัดลอกส่วนที่ดีที่สุดไปเงียบๆ" รยางค์เหล็กปลายแหลมพุ่งตรงเข้ามา "เราจะเหลือเศษซากโง่ๆ ไว้ให้แกเดินเล่นต่อ"
