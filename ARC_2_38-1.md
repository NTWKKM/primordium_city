# บทที่ 38: ภาษาของความล้มเหลว (ตอนที่ 1)

พวกสแกเวนเจอร์ถอยร่นกลับเข้าไปในม่านหมอกเมื่อเห็นว่าผมแพร่เชื้อไวรัสของตัวเองใส่พวกมัน แต่ความรอดพ้นนี้เป็นเพียงสิ่งชั่วคราว สิ่งที่อันตรายยิ่งกว่าคือเงาของ UX Index ที่ยังคงลาดตระเวนหาพิกัดของผมอย่างไม่ลดละ และสถาปัตยกรรมที่บิดเบี้ยวของ The Sump

เมื่อผมลากสังขารขาขวาที่ไร้ส้นเท้าลงมาลึกขึ้น ผมสังเกตเห็นความผิดปกติบางอย่าง พื้นที่รอบตัวเริ่มปรากฏหน้าต่างป๊อปอัปสี่เหลี่ยมสีแดงขนาดใหญ่ลอยอยู่ในอากาศ—Error Messages

`[FATAL EXCEPTION: ENTITY NOT FOUND]`
`[RESOLUTION OVERFLOW: PURGE INITIATED]`

เสียงแจ้งเตือนดังระรัวถี่ยิบ มันบาดหูเหมือนเครื่องบดโลหะ ผมหมอบซ่อนตัวอยู่หลังซากกำแพง เอามือขวากุมหน้าอกที่มีเพย์โหลดกุญแจข้อมูลไว้แน่นด้วยความหวาดกลัว ผมแอบมองผ่านรอยแยก เห็นวิญญาณดวงหนึ่งที่อยู่ไม่ไกลกำลังพยายามเดินอย่างสง่างาม พยายามดึงสติและรักษาความ High-Res ของตนเองไว้ 

และเพียงเสี้ยววินาที... ลำแสงสีแดงวาบจากเบื้องบนก็พุ่งลงมาแผดเผาร่างนั้นจนสลายกลายเป็นเพียงฝุ่นพิกเซลต่อหน้าต่อตาผม!

ผมตัวสั่นเทา นี่มันคืออะไร? กระบวนการชำระล้าง (Purge Routine) ของระบบงั้นเหรอ? หรือว่าการพยายามรักษาสภาพ (Rendering) ให้สมบูรณ์แบบในดินแดนแห่งความเสื่อมโทรมนี้ คือการทำตัวให้เป็นจุดเด่น? มันคงเป็นตรรกะที่วิปริต... ความสมบูรณ์แบบคือสิ่งผิดปกติในที่แห่งนี้

ร่างกายของผมยังคงสั่นสะท้านจากความเสียหาย แขนซ้ายที่ติด Bit Rot แทบจะขยับไม่ได้ ผมพยายามจะคิดหาวิธีเยียวยาบาดแผล (Healing) ตามสัญชาตญาณ แต่ผมต้องหยุดตัวเองไว้ หากผมพยายามฟื้นฟูความละเอียดของตัวเอง ผมอาจจะถูกลำแสงสีแดงนั้นแผดเผา หรือไม่ก็เป็นจุดสนใจให้ UX Index ค้นพบผมได้ง่ายขึ้น

ผมหลับตาลง สัมผัสถึงความเจ็บปวดจากการสูญเสียความละเอียด ความรู้สึกสากกระด้างที่ผิว ความพร่ามัวในการมองเห็น และแทนที่ผมจะฝืนสู้เพื่อดึงความคมชัดเหล่านั้นกลับมา... ผมกลั้นใจปล่อยมันไป

ผมก้าวข้ามเส้นแบ่งระหว่างการดำรงอยู่และการล่มสลาย ผมเริ่มหัดที่จะพูด "ภาษาของความล้มเหลว" (The Language of Failure)
