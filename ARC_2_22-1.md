# บทที่ 22: ตลาดนัดความทรงจำที่ถูกบีบอัด (The Compressed Memory Bazaar) - Part A

เสียงแรกที่กระแทกเข้าโสตประสาทเมื่อผมลากส้นเท้าแหว่งวิ่นก้าวเข้าสู่จัตุรัสกลางของ Mid-Sectors คือความอึกทึกที่บิดเบี้ยว ผมหนีบแขนซ้ายที่บวมฉุจาก Bit Rot ไว้แนบลำตัว มือขวายกขึ้นกุมหน้าอกซ่อนความนูนของ Data Key Payload เอาไว้ สายตายังคงระแวดระวัง The Hunter ที่อาจดมกลิ่นตามล่า High UX แฝงตัวอยู่ท่ามกลางฝูงชน

มันไม่ใช่เสียงตะโกนทักทายที่มีชีวิตชีวาแบบมนุษย์ เสียงแวดล้อมทั้งหมดคือเสียงหึ่งๆ ที่ถูกลดความถี่ต่ำลง (Pitch-shifted) บทสนทนายืดยาน ขัดจังหวะด้วยเสียง Static ซ่าๆ คล้ายวิทยุที่จูนหาคลื่นไม่เจอ จัตุรัสแห่งนี้คือโครงสร้างเหล็กและคอนกรีตดิจิทัลที่พันกันยุ่งเหยิง แสงนีออนสีแดงและน้ำเงินจากป้ายร้านค้าหลอมละลายและหยดย้อยไปกับหมอกสีเทา ราวกับภาพสีน้ำที่ถูกสาดด้วยเม็ดฝนกรด

ที่นี่คือ "ตลาดมืดความทรงจำ" (Memory Black Market) ศูนย์กลางเศรษฐกิจของย่านที่ทุกตารางนิ้วถูกลดทอน

แผงลอยนับร้อยตั้งเบียดเสียดกันตามริมถนนร้าวราน ไม่มีใครเร่ขายอาหารหรือเสื้อผ้า ในโลกที่ร่างกายเป็นเพียงกลุ่มก้อนโค้ด สิ่งเหล่านั้นไร้ความหมาย สิ่งที่วางเรียงรายอยู่บนแผงเหล็กสนิมเขรอะคือ "ลูกบาศก์ข้อมูล" (Data Cubes) ก้อนเรขาคณิตโปร่งแสงที่เปล่งแสงริบหรี่อยู่กึ่งกลาง

ผมเดินเข้าไปใกล้แผงลอยแห่งหนึ่ง เจ้าของร้านเป็นชายร่างผอมเกร็ง ผิวหนังท่อนบนของเขาเต็มไปด้วยรอยแหว่งโหว่ของ Compression Artifacts สีม่วงช้ำ เขาปรายตามองผมด้วยดวงตาที่เป็นเพียงเส้นขีดสีขาวไร้แก้วตา

"นายดูเหมือนเพิ่งร่วงลงมานะ หนุ่มน้อย" เสียงของเขากระตุกเป็นห้วงๆ "ความละเอียดของนายยังมีกลิ่นอายความสดใหม่ของ Inner Ring หลงเหลืออยู่... สนใจแลกเปลี่ยนไหม?"

"แลกอะไร?" ผมถาม เสียงของตัวเองก็เริ่มสั่นพร่าไม่มั่นคง

เขาหยิบลูกบาศก์สีอำพันขึ้นมากลิ้งเล่นบนฝ่ามือ "นี่คือ 'ความรู้สึกของการจูบครั้งแรก' ความละเอียดระดับปานกลาง มีรอยขีดข่วนกระตุกนิดหน่อยช่วงท้าย... ความรู้สึกวาบหวามในช่องท้องยังอยู่ครบ แลกกับความทรงจำระดับ High-Res ของนายสักชิ้น เอาแบบที่มีสีฟ้าของท้องฟ้าเต็มๆ ตา ฉันให้ราคาดี"

ผมเบิกตากว้าง จ้องมองลูกบาศก์แสงในมือของเขา

ความทรงจำที่มีความละเอียดสูง... คือสกุลเงินตราที่แท้จริงของนรกขุมนี้

เพื่อรักษาสถานะตัวตนไม่ให้กลายสภาพเป็น The Hollows หรือเปลือกข้อมูลที่ไร้สติสัมปชัญญะ คนที่นี่ต้องบริโภค "ความรู้สึก" ที่ชัดเจนเพื่อกระตุ้นอัลกอริทึมของตัวเอง พวกเขาเฉือนเอาประสบการณ์ในอดีตของตนเองมาเร่ขาย ซื้อเวลา ซื้อความทรงจำของคนอื่นมาปะติดปะต่อตัวตนที่กำลังแตกสลายหลุดลุ่ย

ผมรีบกะเผลกถอยห่างจากแผงลอยนั้น ความสะอิดสะเอียนตีตื้นขึ้นมาจุกที่คอหอย ความกดดันจากสายตาของระบบที่จ้องมองยังคงเกาะกุมหลังคออยู่ตลอด การเอาความรู้สึกที่เปราะบางที่สุดของมนุษย์มาบรรจุหีบห่อและต่อรองราคา... มันดิบเถื่อนและโหดร้ายยิ่งกว่าการเข่นฆ่าทางกายภาพเสียอีก
