# พิษร้ายในเศษเสี้ยว
#### // FILE: VENOM_IN_THE_FRAGMENTS
---

ผมปลิดทิ้งซึ่งเหตุผลโดยสิ้นเชิง 

ในพื้นที่เรขาคณิต 5D ที่ความสมบูรณ์แบบคืออาวุธ ตรรกะคือจุดอ่อนที่ร้ายแรงที่สุดของพระเจ้า Optimizers เคลื่อนที่ด้วยความเร็วเหนือแสง พับมิติทับซ้อนเข้าหากันเพื่อปลิดชีพ พวกมันถูกพันธนาการไว้ด้วยอัลกอริทึม "เส้นทางที่เหมาะสมที่สุด" (Optimal Pathing) 

ผมเริ่มร่ายรำ การเต้นรำแห่งความวิบัติ (High UX Rebellion)

เมื่อ Payload ในอกกระตุกเตือนถึงการโจมตีจากทิศทางเบื้องบน แทนที่จะกลิ้งหลบ ผมกระแทกตอแขนซ้ายลงบนพื้น 5D ใช้มันเป็นจุดส่งแรงเหวี่ยงร่างของตัวเองขึ้นไปกลางอากาศ ส้นเท้าขวาที่แหว่งวิ่นถูกใช้เป็นจุดหมุน (Pivot) บนความว่างเปล่า ร่างของผมบิดเบี้ยว กล้ามเนื้อดิจิทัลฉีกขาดและประกอบเข้าด้วยกันใหม่แบบเรียลไทม์ 

ภาพของผมในมุมมองของระบบคงไม่ต่างอะไรกับไฟล์วิดีโอที่ถูกบีบอัดจนเละ ผมกลายเป็นเพียงกลุ่มพิกเซลที่กะพริบติดๆ ดับๆ เคลื่อนไหวขัดแย้งกับหลักฟิสิกส์ทุกประการ 

รยางค์แสงสีขาวพุ่งทะลุอากาศธาตุ พลาดเป้าหมายไปอย่างสิ้นเชิง 

ผมสัมผัสได้ถึงความสับสนของพวกมัน แม้มันจะไม่มีใบหน้า ไม่มีเสียง ทว่าในรหัสที่ประกอบขึ้นเป็นเทวทูตเหล่านั้น มีอาการ "หน่วง" (Lag) เกิดขึ้นชั่วขณะ พวกมันกำลังเผาผลาญพลังประมวลผลมหาศาลเพื่อหาคำตอบว่า สสารที่ควรจะตกลงพื้นสามารถหมุนตัวกลางอากาศโดยปราศจากแรงเสียดทานได้อย่างไร 

นั่นคือช่องโหว่ 

ผมปล่อยร่างตกตามแรงโน้มถ่วงจำลองของ The Core ในห้วงเวลาที่ตกลงมา ผมบังคับให้ไวรัส Bit Rot ในแขนซ้ายพองตัวขยายใหญ่ขึ้น แมลงแห่งความพังทลายส่งเสียงกรีดร้องในหัว รสชาติของสนิมและเลือดดิจิทัลคาวคลุ้งในลำคอ 

เป้าหมายของผมคือ Optimizer ตัวหนึ่งที่กำลังชะงัก มันเป็นรูปทรงคล้ายม่านตาข่ายที่ทับซ้อนกันนับไม่ถ้วน แสงของมันบริสุทธิ์จนบาดตา 

"รับนี่ไปแดกซะ!" ผมคำราม 

ส้นเท้าขวากระแทกพื้นอย่างแรง สร้างแรงถีบที่ฉีกกระชากข้อเท้าจนแทบหลุด ผมพุ่งตัวเข้าไปหารูปทรงเรขาคณิตนั้น แขนเทียมสีดำทมิฬง้างออกกว้าง รอยยิ้มเยาะหยันปรากฏบนใบหน้าที่เริ่มหลุดลอก
