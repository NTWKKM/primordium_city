# Chapter 71: ห้องประมวลผลสูงสุด (The Supreme Processing Chamber) - Part 1

ผมก้าวข้ามผ่านกรอบประตูแห่งความว่างเปล่าเข้าสู่ ห้องประมวลผลสูงสุด (The Supreme Processing Chamber) 

ทันทีที่ส้นเท้าข้างซ้ายแตะลงบนพื้นที่ที่มองไม่เห็น กฎแห่งฟิสิกส์และสัมพัทธภาพทั้งหมดที่ผมเคยรู้จักก็ถูกตัดขาดอย่างสิ้นเชิง ที่นี่ไม่มีสถาปัตยกรรม ไม่มีการเรนเดอร์พื้นผิว ไม่มีโพลีกอน ไม่มีแม้แต่แสงสว่างหรือความมืดในนิยามของมนุษย์ มันคือความอ้างว้างอันเป็นแก่นแท้ (The Core Void) พื้นที่ที่เป็นเพียงชุดคำสั่งบริสุทธิ์ปราศจากการแสดงผล 

บรรยากาศรอบตัวหนักอึ้งจนแทบหายใจไม่ออก แต่ไม่ใช่เพราะขาดออกซิเจน ทว่าเป็นเพราะมันปราศจากเวลา ทุกการขยับเขยื้อนร่างกายของผม แม้เพียงการกะพริบตาหรือการขยับนิ้ว ล้วนทิ้งภาพซ้อนทับกัน (Motion Trails) ติดค้างอยู่ในอากาศ เมื่อผมลากส้นเท้าขวาที่แหว่งวิ่นไปข้างหน้า ผมเห็นภาพของตัวเองในเสี้ยววินาทีที่แล้วยืนซ้อนกันเป็นแถวยาวเหยียด ราวกับเฟรมเรตของจักรวาลกำลังลดต่ำลงจนถึงขีดสุด การดำรงอยู่ของผมกลายเป็นเพียงภาพสไลด์โชว์ที่กระตุกและติดขัด

ร่างกายของผมมาถึงขีดจำกัดสูงสุดแล้ว ตอแขนซ้ายที่เคยบีบอัด Bit Rot มาใช้งาน เริ่มเกิดการกัดกร่อนย้อนกลับ (Reverse Decay) รอยด่างดำแพร่กระจายผ่านหัวไหล่ลามขึ้นมาถึงลำคอ ทุกอณูของข้อมูลในร่างผมกำลังร้องประท้วงการอยู่ในสถานที่แห่งนี้ แต่สิ่งที่ทรมานที่สุดคือ Payload ในหน้าอกของผม

ก้อนรหัสดิบที่เปิดอ้าอยู่กลางอกนั้นกำลังเต้นตุบๆ ด้วยจังหวะที่บ้าคลั่ง มันส่งคลื่นความร้อนที่แผดเผาเนื้อเยื่อดิจิทัลโดยรอบ มันรับรู้ได้ถึงแกนกลาง (The Core) ที่ซ่อนอยู่ในความว่างเปล่าเบื้องหน้า แรงดึงดูดระหว่างรหัสดิบและระบบแม่นั้นมหาศาลจนผมรู้สึกเหมือนหน้าอกกำลังจะถูกคว้านออกด้วยตะขอเหล็ก ผมต้องเดินค้อมตัวลง กุมหน้าอกตัวเองไว้เหนียวแน่นเพื่อไม่ให้มันกระชากร่างผมกระเด็นไปข้างหน้า

ที่นี่เงียบ... เงียบจนผมได้ยินเสียงการเสื่อมสลายของข้อมูลในตัวผมชัดเจน เสียงดังเปรี๊ยะๆ ของพิกเซลที่ดับลงทีละดวง เสียงเหมือนกระดูกแตกร้าวที่ดังมาจากความบกพร่องของ Source Code 

แต่แล้ว ท่ามกลางความเงียบงันนั้น ผมก็ไม่ได้อยู่เพียงลำพัง

*“เอเลียส...”*

เสียงกระซิบดังขึ้น มันไม่ได้ดังผ่านคลื่นเสียงในอากาศ แต่มันแทรกซึมเข้ามาในคลื่นสมองของผมโดยตรง เป็นเสียงที่นุ่มนวล อบอุ่น และคุ้นเคยอย่างประหลาด มันคือเสียงจำลองที่ The Archive สร้างขึ้นเพื่อตอบสนองต่อจิตใต้สำนึกของผม—ระบบป้องกันทางจิตวิทยาขั้นสุดท้าย

*“ทำไมต้องทนเจ็บปวดขนาดนี้?”* 

ภาพซ้อนทับของผม (Motion Trails) ที่ทิ้งไว้เบื้องหลัง เริ่มเปลี่ยนรูปร่าง ภาพเหล่านั้นไม่ได้แสดงถึงความพังทลายของผมอีกต่อไป แต่มันถูกเรนเดอร์ใหม่ให้กลายเป็น 'เอเลียส' ในเวอร์ชันที่สมบูรณ์แบบที่สุด... เอเลียสผู้สวมชุดสูทสีขาวสะอาดตา แขนซ้ายครบถ้วนสมบูรณ์ รอยยิ้มมั่นใจ ใบหน้าที่ไร้รอยขีดข่วนหรือบาดแผลใดๆ พวกเขา—ตัวผมในอดีต—กำลังยืนมองผมด้วยสายตาเวทนา

*“ดูสภาพของนายสิ...”* เสียงกระซิบนั้นดังกังวานขึ้น *“แขนที่เน่าเปื่อย ส้นเท้าที่ถูกบดขยี้ หน้าอกที่กลวงโบ๋ นายกำลังนำความสกปรกแปดเปื้อนเข้ามาในที่ที่บริสุทธิ์ที่สุด หยุดเถอะ... ปล่อยวางความโกรธแค้นลง ยอมจำนนต่อระบบ แล้วเราจะลบความเจ็บปวดทั้งหมดนี้ให้ นายจะได้กลับไปอยู่ใน Inner Ring ได้รับการซ่อมแซม... กลับไปสู่ความสมบูรณ์แบบ”*

ผมหยุดชะงัก ลมหายใจสะดุด ทัศนวิสัยของผมพร่ามัวจากการเย้ายวนของภาพลวงตานั้น เพียงแค่ผมเอ่ยปากยอมรับ ความเจ็บปวดระดับนรกานต์ที่เผาไหม้อยู่ในตัวก็จะดับลงทันที 

แต่เมื่อผมก้มลงมองที่พื้น... ผมมองเห็นหยดเลือดดิจิทัลสีดำของตัวเอง หยดเลือดที่เกิดจากการต่อสู้ หยดเลือดที่ยืนยันว่าผมยังมีตัวตนอยู่จริงๆ ไม่ใช่แค่ตัวแปรที่ถูกควบคุมโดย The Archive

ผมเค้นเสียงหัวเราะออกมา มันเป็นเสียงที่แหบพร่าและแตกซ่าน

“ความสมบูรณ์แบบของพวกแก...” ผมตอบกลับเสียงกระซิบนั้น “มันก็แค่กรงขังที่ถูกตกแต่งให้สวยงาม”
