# บทที่ 22: ตลาดนัดความทรงจำที่ถูกบีบอัด (The Compressed Memory Bazaar) - Part B

ผมลากเท้ากะเผลกฝ่าฝูงชนที่เบียดเสียด ทุกครั้งที่ไหล่ชนกัน ผมรู้สึกถึงประกายไฟเล็กๆ (Glitch sparks) ที่แลบออกมาพร้อมกับความเจ็บลึกจากแขนซ้ายที่เน่าเปื่อย มันคือการชนกันของ Hitbox ที่คำนวณพลาด ภาพรอบตัวเต็มไปด้วยความสับสนวุ่นวาย คนซื้อและคนขายกำลังต่อรองกันอย่างเอาเป็นเอาตาย

ที่แผงลอยฝั่งตรงข้าม หญิงสาวคนหนึ่งกำลังร้องไห้ แต่แทนที่จะเป็นน้ำตา หยดน้ำที่ไหลจากตาของเธอคือพิกเซลสีฟ้าที่ร่วงหล่นแล้วแตกกระจายเป็นโค้ด 

"ฉันขอร้องล่ะ..." เสียงของเธอแตกร้าว "ขอซื้อ 'กลิ่นของฝนบนดิน' คืนเถอะ เมื่อวานฉันเพิ่งขายให้คุณไป แต่ตอนนี้... ฉันจำไม่ได้แล้วว่าฝนคืออะไร ฉันกำลังสูญเสียมัน!"

พ่อค้าส่ายหน้าอย่างไร้เยื่อใย "ขายไปแล้ว แม่หนู มีคนยอมจ่ายโควตาประมวลผลสามวันเพื่อแลกกับกลิ่นฝนนั่น ไปหาอย่างอื่นเถอะ... ฉันมี 'ความเศร้าจากการทำแก้วแตก' เอาไหม? ราคาถูกนะ ช่วยกระตุ้นระบบประสาทจำลองได้ดีทีเดียว"

หญิงสาวทรุดตัวลงกับพื้น กรีดร้องออกมาเป็นเสียงความถี่สูง (Binary screams) ก่อนที่ร่างของเธอจะกระตุกอย่างรุนแรง ผิวหนังของเธอเริ่มสูญเสีย Texture กลายเป็นหุ่นจำลองสีเทาด้านๆ... เธอรับความจริงไม่ได้ และระบบประมวลผลของเธอกำลังพังทลาย

ไม่มีใครสนใจเธอ คนอื่นๆ ยังคงก้มหน้าก้มตาเลือกซื้อเศษซากอารมณ์ต่อไป

ผมมองดูภาพนั้นด้วยความรู้สึกที่หนักอึ้งในอก ความกลัวคืบคลานเข้ามา ไม่ใช่ความกลัวความตาย แต่เป็นความกลัวที่จะ 'ลืม'

ผมยังจำความรู้สึกของสายลมที่พัดผ่านหน้าต่างใน Inner Ring ได้หรือไม่? ผมยังจำสัมผัสของผ้าปูที่นอนที่นุ่มนวลได้หรือเปล่า? หรือว่าความละเอียดเหล่านั้นกำลังถูกบีบอัดและค่อยๆ เลือนหายไปจากฐานข้อมูลส่วนตัวของผมแล้ว?

ถ้าผมอยู่ที่นี่นานเกินไป ผมเองก็คงต้องเฉือนเอาความรัก ความโกรธ หรือความหวังของตัวเองออกมาเร่ขาย เพื่อแลกกับความรู้สึกจอมปลอมของคนอื่น เพียงเพื่อพิสูจน์ว่าตัวเองยัง 'มีชีวิต' อยู่

เศรษฐกิจของ Mid-Sectors ไม่ได้ขับเคลื่อนด้วยเงินทองหรืออำนาจ แต่มันขับเคลื่อนด้วยความโหยหา 

โหยหาในความเป็นมนุษย์ที่พวกเราไม่เคยมีอยู่จริงตั้งแต่แรก

ลมหายใจของผมพ่นเอาหมอกสีเทาออกมาอีกครั้ง ผมซุกแขนซ้ายที่เต็มไปด้วย Compression Artifacts สีม่วงลงในกระเป๋าเสื้อคลุม ขณะที่มือขวายังคงแตะป้องกัน Data Key กลางอกไว้แน่น ผมรู้ดีว่าสุนัขรับใช้ของระบบกำลังสูดดมร่องรอย High UX ของผมอยู่และใกล้เข้ามาทุกที ผมหันหลังให้กับการแลกเปลี่ยนที่น่าสลดใจนั้น และฝืนลากสังขารก้าวเดินต่อไปในความพร่าเลือนของนครกระจกเงา
