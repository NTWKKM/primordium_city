# บทที่ 36: รอยแยกของความหวัง (ตอนที่ 1)

ร่างของผมกำลังร่วงหล่น ไม่ใช่แค่ร่วงหล่นลงสู่ขอบของเขต Mid-Sectors แต่มันคือการพังทลายลงมาจากภายใน ยาแห่งความคมชัด (Clarity) ที่ผมฉีดเข้าร่างกายไปเมื่อครู่แท้จริงแล้วคือมัลแวร์ที่ปลอมตัวมา มันไม่ได้มอบความสมบูรณ์แบบ แต่มันกำลังฉีกกระชากระดับความละเอียดของผมให้แหลกละเอียด ความรู้สึกสากกระด้างแล่นพล่านไปทั่วร่าง บังคับให้ผมต้องลากสังขารที่พังทลายกระแทกเข้ากับขอบเหวของรอยแยกสีดำมืด

ข้อเท้าขวาของผมไร้ส้นเท้าคอยรับน้ำหนักอีกต่อไป มันหายไปอย่างสิ้นเชิงจากการโดนลบข้อมูล ผมล้มลุกคลุกคลาน ลากขาขวาที่ไร้การซัพพอร์ตไปตามพื้นบล็อกสีเทาหยาบๆ แขนซ้ายของผมก็เช่นกัน มันถูกกัดกินด้วยอาการ Bit Rot เน่าเปื่อยและอืดอาด ความรู้สึกชาสลับกับความเจ็บปวดแสบร้อนราวกับมีเข็มดิจิทัลนับพันเล่มทิ่มแทงทุกครั้งที่ผมพยายามขยับมัน

และลึกลงไปในแผงอกของผม ผมสัมผัสได้ถึงความหนักอึ้งของก้อนข้อมูล—เพย์โหลดกุญแจข้อมูล (Data Key Payload) ที่ถูกฝังเอาไว้ มันเต้นเป็นจังหวะแห่งความลับที่ผมต้องปกป้อง 

เบื้องหลังของผม เสียงเตือนความถี่สูงดังก้องในอากาศ... UX Index พวกมันกำลังล่าผมอยู่ หมาล่าเนื้อแห่งสถาปัตยกรรมของ The Archive ไม่ยอมปล่อยให้คนที่มี Error และกุมความลับแบบผมหลุดรอดไปได้

ที่สุดปลายทางของตรอกนั้น ผมถูกต้อนมาพบกับมัน—รอยแยก

มันไม่ใช่ประตูหรือช่องทางที่ถูกสร้างขึ้น แต่มันคือ "บาดแผล" ของสถาปัตยกรรมดิจิทัล รอยร้าวขนาดใหญ่บนกำแพงที่ไม่มีพื้นผิว โค้ดดิบๆ สีเขียวอมเหลืองไหลทะลักออกมาราวกับน้ำเลือดที่ไหลออกจากบาดแผลเปิด 

ผมยื่นมือขวาที่ยังพอใช้งานได้ออกไปเกาะขอบรอยแยก ความรู้สึกสากกระด้างแล่นปราดเข้าสู่ปลายนิ้ว มันเจ็บปวด แต่นี่คือทางลงสู่ The Sump บ่อเกรอะข้อมูล... ดินแดนที่กฎเกณฑ์ของ The Archive อาจจะเอื้อมไม่ถึง

ผมหอบหายใจอย่างสิ้นหวัง ขาขวาที่ขาดส้นเท้าไม่สามารถพยุงน้ำหนักตัวได้อีกต่อไป เมื่อเสียงฝีเท้าของ UX Index ใกล้เข้ามา ผมทิ้งน้ำหนักทั้งหมด... ปล่อยให้แรงโน้มถ่วงที่บิดเบี้ยวกลืนกินตัวตนของผม ร่วงหล่นลงสู่ความมืดที่ไม่มีจุดสิ้นสุด
