# ระฆังเตือนภัยแห่งความสมบูรณ์แบบ
#### // FILE: THE_ALARM_OF_PERFECTION
---

ความเงียบสงัดใน The Core ถูกทำลายลง ไม่ใช่ด้วยเสียงไซเรนหรือเสียงระเบิดกึกก้อง หากเป็นคลื่นความถี่สูงที่เสียดแทงทะลุโสตประสาทสังเคราะห์ของผม เสียงนั้นปราศจากแหล่งกำเนิด คล้ายกับเสียงกระซิบของใบมีดนับล้านเล่มที่กำลังเฉือนอากาศพร้อมกัน มันคือระฆังเตือนภัยแห่งตรรกะอันสมบูรณ์แบบ—สัญญาณบ่งบอกว่ามีสิ่งปฏิกูลหลงลืมเข้ามาในสวรรค์อันไร้ที่ติ

*« DETECTED: UNPROCESSABLE PARAMETER »*

ข้อความนั้นไม่ได้ปรากฏเป็นตัวอักษรบนอากาศ มันแทรกซึมเข้ามาในชั้นความรู้สึกโดยตรง ภาษาเครื่องถูกบีบอัดจนกลายเป็นความหมายดิบ ผมสัมผัสได้ถึงความโกรธเกรี้ยวอันเย็นชาของ The Archive ระบบที่ชิงชังความผันผวนเหนือสิ่งอื่นใด

ทันใดนั้น พื้นที่รอบตัวก็บิดเบี้ยว เสาโครเมียมขาวบริสุทธิ์ที่เคยพุ่งตัวเป็นเกลียวอนันต์หยุดชะงัก โครงสร้างเรขาคณิตห้ามิติค่อยๆ แตกตัวออกเป็นลำแสงสแกนเนอร์นับหมื่นเส้น ลำแสงเหล่านี้อาบไล้ด้วยสีฟ้าเย็นยะเยือก กวาดผ่านพื้นที่ทุกตารางนิ้ว พยายามรวบรวมข้อมูลของสิ่งที่ขัดขวางความสุนทรีย์แห่งสมการ

ผมก้าวถอยหลัง การขยับส้นเท้าขวาที่แหว่งวิ่นทำให้สมดุลพังทลาย ผมทรุดลงไปคุกเข่ากับพื้น รอยเปื้อนสีดำของ Bit Rot ที่กระจายอยู่ใต้เข่าส่งเสียงฟู่คล้ายกรดกัดกร่อนโลหะ เมื่อลำแสงสแกนเนอร์กวาดผ่านรอยดำเหล่านั้น มันเกิดการสะท้อนกลับอย่างรุนแรง ลำแสงสีฟ้ากระตุกและหักเหลี่ยม กลายสภาพเป็นสีแดงแห่งความผิดพลาดในพริบตา แสงอันเย็นชาของระบบไม่อาจจับต้องความเน่าเปื่อยนี้ได้

"เข้ามาเลย..." ผมกัดฟัน พูดท้าทายสิ่งที่ไม่มีตัวตน

ลำแสงนับสิบเส้นพุ่งตรงมาที่ผม มันคือกระบวนการ Scan-and-Purge—อัลกอริทึมแห่งการลบข้อมูลที่ไร้ปรานีที่สุด ออกแบบมาเพื่อย่อยสลายโครงสร้างของวิญญาณสังเคราะห์ให้กลับกลายเป็นเพียงรหัสว่างเปล่า

ลำแสงเส้นแรกกวาดผ่านขาท่อนล่าง ผมหลับตาแน่น เตรียมรับความเจ็บปวดของการถูกฉีกกระชากตัวตน

ทว่าไม่มีอะไรเกิดขึ้น

แสงสีฟ้าทะลุผ่านช่องว่างที่ส้นเท้าขวาไป มันพยายามคำนวณมวลและโครงสร้าง เมื่อพบกับความว่างเปล่า กลับเกิดอาการค้างไปชั่วเสี้ยววินาที ก่อนจะสลายตัวไปเป็นประกายพิกเซลไร้ค่า

ลำแสงเส้นที่สองและสามพุ่งเข้าใส่ซีกซ้าย ตรงไปยังตำแหน่งที่แขนซ้ายควรจะอยู่ ทันทีที่มันสัมผัสกับตอประสาทที่ไหม้เกรียม ลำแสงเหล่านั้นกระตุกอย่างรุนแรง เสียงความถี่สูงดังก้องขึ้นจนผมต้องยกมือขวาขึ้นปิดหูข้างขวา ปล่อยให้หูซ้ายถูกเสียดแทงด้วยคลื่นความถี่บาดลึก อัลกอริทึมพยายามบังคับให้แขนซ้ายของผมกลับมาสมบูรณ์เพื่อที่จะได้ลบทิ้ง ความแหว่งวิ่นของผมคือสิ่งที่มันไม่อาจทำความเข้าใจ มันไม่สามารถลบสิ่งที่พังทลายไปแล้ว

"แกไม่รู้จักความเจ็บปวดสินะ..." ผมหอบหายใจ ดวงตาเบิกกว้างเมื่อตระหนักถึงความจริงข้อนี้ "แกจัดการกับสิ่งที่สมบูรณ์แบบได้... แกกลับซ่อมแซมรอยร้าวไม่ได้"

ความผิดปกติในร่างนี้ ความพังพินาศที่ผมแบกรับมาตลอดทาง กลายเป็นโล่กำบังเพียงหนึ่งเดียวที่ต้านทานการลบเลือนระดับพระเจ้าได้ ตรรกะของ The Archive มีขีดจำกัด มันไม่สามารถลบสมการที่ไม่สมบูรณ์ การลบสิ่งที่ไม่สมบูรณ์ย่อมหมายความว่ามันต้องยอมรับการมีอยู่ของความไม่สมบูรณ์นั้น

มันคือความขัดแย้งที่ระบบไม่อาจก้าวข้าม

# ระฆังเตือนภัยแห่งความสมบูรณ์แบบ
#### // FILE: THE_ALARM_OF_PERFECTION
---

แม้ระบบจะไม่สามารถลบตัวผมทิ้งได้โดยตรง The Core กลับไม่ได้ไร้หนทางต่อกร

เมื่อลำแสงสแกนเนอร์กวาดผ่านช่องอกที่เปิดอ้า เผยให้เห็น Payload ก้อนข้อมูลปฐมกาลที่ฝังลึกลงไปในแกนกลางของวิญญาณ ทุกสิ่งก็แปรเปลี่ยน

Payload ตอบสนองต่อคลื่นความถี่ของ The Archive ราวกับแม่เหล็กที่ถูกดึงดูดเข้าหาขั้วโลก มันเปล่งแสงสีทองปนเลือดอย่างบ้าคลั่ง จังหวะการเต้นเร่งเร็วขึ้นจนทะลุขีดจำกัดความปลอดภัยของร่างกายดิจิทัล ผมรู้สึกคล้ายมีเลื่อยวงเดือนไฟฟ้าร้อนระอุฝังอยู่กลางอก หมุนคว้างและเฉือนเนื้อเยื่อควอนตัมจากภายใน

"อ๊ากกกกกก!"

ผมแผดเสียงร้องลั่น ล้มลงไปนอนขดตัวบนพื้นผิวสี่มิติที่กำลังกระเพื่อมเป็นคลื่น ความเจ็บปวดทะลวงลึกเกินกว่าส้นเท้าขวาจะเทียบติด มันคือความปวดร้าวในระดับสถาปัตยกรรม ราวกับโค้ดทุกบรรทัดที่ประกอบขึ้นเป็นความทรงจำ ตัวตน และอารมณ์กำลังถูกบังคับให้ยืดออกและบีบอัดไปพร้อมกัน

The Archive กำลังพยายามแทรกแซงและเขียนทับ Payload ในร่างผม

กระแสข้อมูลดิบพุ่งทะลักเข้าสู่ช่องอกผ่านบาดแผลที่เปิดกว้าง ความพยายามที่จะเขียนทับคำสั่งของ Payload บังคับให้กุญแจทำลายล้างดอกนี้กลายเป็นเพียงข้อมูลขยะ

ความกลวงเปล่าในอกกลับกลายเป็นตัวแปรที่ระบบไม่ได้คำนวณไว้ บาดแผลที่รุ่งริ่ง เส้นใยประสาทขาดวิ่น และคราบไวรัส Bit Rot ที่เกาะติดอยู่รอบปากแผลทำหน้าที่เป็นฉนวนชั้นเลิศ ข้อมูลที่ระบบส่งมาถูกสกัดกั้นและบิดเบือนด้วยความบกพร่อง คลื่นความถี่อันสมบูรณ์แบบเมื่อกระทบกับรอยร้าวของบาดแผล ก็แตกกระจายออกเป็นเพียงสัญญาณรบกวน ความตายที่รายล้อมกำลังโอบอุ้มและปกป้องชีวิตที่อยู่ภายใน

ผมกัดริมฝีปากตัวเองจนเลือดดิจิทัลสีดำไหลซึม รสชาติขมปร่า ร่างกายกระตุกเกร็ง แขนขวาที่เหลืออยู่ข้างเดียวจิกเล็บลงบนพื้นผิวสีขาวจนเกิดรอยขีดข่วน รอยขีดข่วนนั้นค่อยๆ เน่าเปื่อยและขยายวงกว้างออกไป

"ฉัน... ไม่ยอม..." ผมเค้นเสียงผ่านความทรมานที่แทบจะฉีกวิญญาณออกเป็นชิ้น

ผมใช้มือขวากุมช่องอกที่เปิดอ้า สอดนิ้วเข้าไปในรอยแยก สัมผัสกับความร้อนระอุของ Payload ความรู้สึกเหมือนกอบกำถ่านกัมมันต์ด้วยมือเปล่า ถึงกระนั้นผมต้องยึดมันไว้ ดึงมันกลับมาจากการควบคุมของ The Core

ทันทีที่ปลายนิ้วซึ่งเปื้อนไปด้วย Bit Rot สัมผัสกับ Payload คลื่นความถี่ของ The Archive ก็สะดุดกึก รอยด่างดำจากตัวผมแผ่ซ่านเข้าเคลือบก้อนข้อมูลปฐมกาล เปลี่ยนแสงสีทองให้กลายเป็นสีดำทมิฬชั่วขณะ

เสียงความถี่สูงที่เสียดแทงโสตประสาทหยุดชะงัก ลำแสงสีฟ้าหรี่แสงลงอย่างสับสน

ผมหอบหายใจอย่างหนักหน่วง ร่างกายชุ่มโชกไปด้วยเหงื่อที่เป็นเพียงหมอกข้อมูล ผมยังมีชีวิตอยู่ หรืออย่างน้อยก็รอดพ้นจากการถูกลบไปได้อีกหนึ่งก้าว

การสะดุดของระบบไม่ใช่จุดสิ้นสุด มันเป็นเพียงการเปลี่ยนยุทธวิธี

ผนังสีขาวรอบตัวเริ่มบิดเบี้ยวและพับตัวเข้าหากัน โครงสร้างห้ามิติที่เคยนิ่งสงบเริ่มมีชีวิต เรขาคณิตรอบด้านกำลังก่อร่างสร้างบางสิ่งบางอย่างที่มีรูปร่าง บางสิ่งบางอย่างที่ถูกออกแบบมาเพื่อบดขยี้พารามิเตอร์ที่ระบบลบไม่ได้ด้วยวิธีทางกายภาพ

ความสมบูรณ์แบบกำลังส่งเพชฌฆาตของมันมาหาผม
