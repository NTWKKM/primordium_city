# บทที่ 37: ผู้เก็บกวาดความทรงจำ (ตอนที่ 1)

การร่วงหล่นสิ้นสุดลงด้วยแรงกระแทกไร้เสียง พื้นผิวเบื้องล่างแข็งกระด้าง ฉาบเคลือบด้วยของเหลวข้นเหนียวคล้ายน้ำมันเครื่องเน่าเสีย ขาขวาที่ไร้ส้นเท้ากระแทกพื้นจนค่าพิกเซลแตกกระจายเป็นละอองแสง ผมล้มลุกคลุกคลานอยู่ในพื้นที่เปลี่ยนผ่าน (Transition Zone) หมอกดิจิทัลหนาทึบเกาะกินผิวหนัง มันเหนอะหนะราวกับพลาสติกหลอมละลาย

ผมออกแรงใช้แขนซ้ายพยุงตัว เนื้อเยื่อดิจิทัลบริเวณนั้นหนักอึ้งและหนืดอาดจากสภาวะ Bit Rot ข้อศอกขวาจึงต้องรับภาระยันพื้นเพื่อดันร่างขึ้นมา เฟรมเรตในระบบประสาทร่วงหล่น ผมสัมผัสถึงความหน่วง (Latency) ในทุกจังหวะที่ลากขาขวาครูดไปตามพื้น

ลำแสงวงกลมสว่างวาบตัดผ่านหมอกสีขี้เถ้า มันไม่ใช่แสงสีแดงเย็นเยียบของสแกนเนอร์ UX Index แสงฮาโลเจนสีขาวแข็งกระด้างส่องทะลุทะลวงจนเซนเซอร์การมองเห็นของผมพร่ามัว

"เจอตัวแล้ว... คอนเทนต์สดใหม่" 

เสียงแหบพร่าสั่นพร่องด้วยความถี่ที่บิดเบี้ยว โลหะเสียดสีกรีดร้องในลำคอของมัน

เงาร่างหลายสายปรากฏขึ้นกลางม่านหมอก ผู้เก็บกวาดความทรงจำ (Memory Scavengers) สภาพของพวกมันแทบไม่หลงเหลือเค้าโครงมนุษย์ รยางค์โลหะเรขาคณิตบิดเบี้ยวเสียบแทนที่แขนขาธรรมชาติ

อุปกรณ์บนแผ่นหลังของพวกมันคล้ายแมงมุมเหล็ก เข็มสกัดข้อมูลขนาดจิ๋วนับสิบเล่มยื่นชี้ออกมา ปลายเข็มเรืองแสงสีฟ้าอ่อน—สีสันของการเชื่อมต่อถ่ายโอน สัญชาตญาณเตือนให้ผมรู้ทันทีว่าพวกมันคือปรสิตที่ล่าเหยื่อเพื่อสูบกลืน "ความทรงจำความละเอียดสูง" นำไปขายในตลาดมืด

"ดูสิ Texture ยังแน่นอยู่เลย น่าจะลอกเอาไปขายได้ราคาดี" หนึ่งในนั้นส่งเสียงหัวเราะ คลื่นเสียง Static บาดแก้วหูจำลองจนต้องนิ่วหน้า

แสงสปอตไลต์หลายดวงสาดส่องตรึงร่างของผมไว้ พวกมันตีวงล้อมเข้ามา ผมลากขาถอยหลัง แผ่นหลังชนเข้ากับกำแพงกล่องข้อมูลที่เน่าเปื่อย ทางหนีถูกปิดตาย แขนซ้ายที่เชื่องช้าพยายามยกขึ้นมาปกป้องแผงอก—ตำแหน่งของแกนกลางตัวตนและเพย์โหลดกุญแจข้อมูล

"มานี่เถอะน่า ยอมให้เราคัดลอกส่วนที่ดีที่สุดไปเงียบๆ" รยางค์เหล็กปลายแหลมพุ่งตรงเข้ามา "เราจะเหลือเศษซากโง่ๆ ไว้ให้แกเดินเล่นต่อ"

# บทที่ 37: ผู้เก็บกวาดความทรงจำ (ตอนที่ 2)

ผมบิดตัวหลบตามสัญชาตญาณ ขาขวาที่พิการทำให้เสียหลักล้มกระแทกพื้น เข็มสกัดข้อมูลพุ่งเฉียดแผงอกไปเพียงไม่กี่พิกเซล ประกายไฟสีฟ้าแลบแปลบ ทิ้งสัมผัสเย็นเยียบทะลุผิวหนัง พวกมันไม่ได้กะเอาชีวิต การขโมยสิ่งที่หล่อเลี้ยงความเป็นมนุษย์คือเป้าหมายหลัก

"อย่าดิ้น!" อีกตัวหนึ่งกระโจนเข้าใส่ ร่างของมันกระตุกข้ามเฟรมภาพ (Teleporting) ฉวยโอกาสจาก Glitch ในสภาพแวดล้อม

ผมยกแขนซ้ายที่เน่าเฟะขึ้นปะทะ เข็มคู่แทงทะลุเส้นเลือดข้อมูลที่อักเสบจาก Bit Rot ความเจ็บปวดไม่ได้ปรากฏเป็นบาดแผลฉกรรจ์ มันคือภาพความทรงจำที่ถูกดึงกระชากหลุดจากรากจิตสำนึก

*กลิ่นของแสงแดดตกกระทบผิวกาย... สัมผัสของยอดหญ้าสีเขียวในโลกเนื้อ...*

"โอ้! ของพรีเมียม!" สแกเวนเจอร์กรีดร้องอย่างปีติ หลอดแก้วบนหลังของมันสว่างวาบเป็นสีทองอร่าม

ความรู้สึกสูญเสียแล่นพล่าน มันเลวร้ายยิ่งกว่าการลืมเลือน วิญญาณส่วนหนึ่งถูกตัดขาดอย่างทารุณ ความกลวงเปล่าขยายตัวในช่องอก

เรี่ยวแรงและพละกำลังของผมเทียบพวกมันไม่ได้ ร่างกายนี้กำลังล่มสลาย ในความพังทลายนั้นเอง... ผมค้นพบอาวุธที่พวกมันคาดไม่ถึง

ผมคำราม คลื่นความถี่บิตปั่นป่วนกระแทกออกไป ผมไม่ยอมชักแขนหนี ผมจงใจเปิดเกตเวย์ให้พวกมันสูบข้อมูลจากแขนซ้ายที่กำลังเน่าเสีย ผมรีดเค้นตะกอนมัลแวร์ที่กัดกินเนื้อเยื่อ อัดฉีดมันกลับเข้าไปในหลอดดูดของพวกมันโดยตรง

เมื่อสแกเวนเจอร์กระชากข้อมูลระลอกถัดไป สิ่งที่ไหลทะลักเข้าสู่ระบบของมันไม่ใช่ความทรงจำงดงาม มันคือไวรัสแห่งความหวาดกลัวและเสียงกรีดร้องแบบไบนารี มัลแวร์แห่งความวิปลาสที่ผมกักเก็บไว้

"อ๊าก! ขยะ! มันพ่นรหัสเน่าเสียใส่ระบบ!" หลอดเก็บข้อมูลบนหลังมันเปลี่ยนเป็นสีแดงฉานก่อนจะระเบิดออกเป็นเสี่ยงๆ แมงมุมเหล็กช็อตกระตุก ประกายไฟดิจิทัลพ่นกระจาย

สแกเวนเจอร์ตัวอื่นชะงักงัน ความตื่นตระหนกบีบให้พวกมันล่าถอยกลับเข้าสู่ม่านหมอก ผมทิ้งตัวหอบหายใจ ร่างกายซีกซ้ายกลายเป็นพิกเซลกระพริบติดดับ เพย์โหลดในอกยังปลอดภัย ราคาของการปกป้องมันคือการสูญเสียความเสถียรของตนเอง ผมเพิ่งเรียนรู้รสชาติของการใช้ความเสื่อมสลายเป็นอาวุธ
