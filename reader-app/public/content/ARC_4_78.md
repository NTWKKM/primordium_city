# ซากปรักหักพังของพระเจ้า

เมื่อความโกลาหลสงบลง สิ่งที่หลงเหลืออยู่คือความเงียบที่หนักอึ้ง

มันไม่ใช่ความเงียบของ The Silence Vacuum ไม่ใช่ความสมบูรณ์แบบของการคำนวณ หากเป็นความเงียบของเครื่องจักรที่ถูกถอดปลั๊กออก ความเงียบของโครงสร้างที่ถูก Quantum Reverse-Compiler ถอดรหัสจนเหลือเพียงรากฐาน

ผมลืมตาขึ้น เปลือกตาหนักอึ้งราวกับถูกเคลือบด้วยเศษโลหะ การมองเห็นถูกจำกัดด้วยความละเอียดที่ต่ำเตี้ยเรี่ยดิน ทุกสิ่งที่สะท้อนเข้าสู่เรตินาดิจิทัลล้วนประกอบด้วยพิกเซลหยาบๆ เหลี่ยมมุมที่แตกเป็นบล็อก สีสันที่สูญเสียความลึกของเฉดสีไปอย่างสิ้นเชิง 

นี่คือสิ่งที่เคยเป็น The Core สถาปัตยกรรม 5 มิติที่เคยพับซ้อนอย่างงดงาม บัดนี้แบนราบและแตกสลาย กลายเป็นเพียงซากเรขาคณิต 2 มิติที่ทับถมกันเหมือนเศษกระดาษแข็ง ขอบของภาพเหลื่อมทับกันอย่างหยาบกระด้าง ไร้ซึ่งแสงเงาและการประมวลผลที่ซับซ้อน ท้องฟ้าเบื้องบนเป็นสีเทาตุ่นๆ ที่เต็มไปด้วยเส้นสายของ Static Noise 

ผมพยายามพยุงตัวลุกขึ้น สัญชาตญาณสั่งให้แขนซ้ายยันพื้น... ร่างของผมล้มฟาดลงไปบนแผ่นพิกเซลอีกครั้ง 

ผมหันไปมองที่หัวไหล่ซ้าย 

ว่างเปล่า. 

มันไม่ใช่ตอแขนที่เต็มไปด้วยรอยกัดกร่อนพร้อมจะงอกใหม่ มันคือความว่างเปล่าสัมบูรณ์ รอยต่อระหว่างหัวไหล่กับอากาศภายนอกเรียบสนิท ตัดขาดจากระบบประสาท ตัดขาดจากกระแสความทรงจำ 

ผมไม่สามารถสัมผัสได้ถึง Phantom Pain อีกต่อไป ไม่มีความกระหายที่จะทำลายล้าง แขนซ้ายของผมถูกลบทิ้งจากแกนสมการของตัวตนอย่างถาวร 

"หายไปแล้ว..." ผมกระซิบ เสียงสั่นพร่าและแตกเป็นออดิโอสังเคราะห์ต่ำๆ (Downsampled Audio) "หายไปหมดแล้ว" 

การแลกเปลี่ยนเพื่อหยุดยั้ง UX Index จบลงด้วยการสูญเสียชิ้นส่วนสำคัญของตัวเอง มันเป็นการกระทำที่ไม่อาจหวนกลับ 

ผมพลิกตัวนอนหงาย มองดูสิ่งที่เคยเป็นโดมแห่งพระเจ้า ยอดแหลมของอาคารเรขาคณิตที่เคยกางออกคล้ายปีกเทวทูต บัดนี้หักโค่นและแบนราบติดกับพื้นผิว 

ไม่มี UX Index ไม่มีนาฬิกานับถอยหลังเพื่อล้างบางเมืองอีกต่อไป

บาดแผลเปิดอ้าที่หน้าอกเต้นตุบๆ ตามจังหวะหัวใจที่อ่อนแรง เลือดดิจิทัลไหลซึมออกมาย้อมเศษซาก 2 มิติรอบตัวให้กลายเป็นแอ่งความตาย ร่างกายของผมกำลังพังทลาย โค้ดในตัวไม่สามารถทนทานต่อความเสียหายระดับนี้ได้อีก ผิวหนังกระพริบและโปร่งแสง เผยให้เห็นสายเคเบิลภายในที่ขาดวิ่น 

ภายในซากปรักหักพังอันน่าสมเพช ท่ามกลางความตายที่คืบคลานเข้ามา... ผมกลับรู้สึกถึงรอยยิ้มบางๆ ที่มุมปาก

พระเจ้าถูกดึงลงมาคลุกฝุ่นแล้ว และฝุ่นเหล่านั้นก็งดงามอย่างประหลาด

ผมใช้มือขวาที่เหลือเพียงข้างเดียว จิกปลายนิ้วลงบนพื้นผิวที่แบนราบและแตกเป็นบล็อกของศูนย์กลาง The Core ขาทั้งสองข้างหนักอึ้งราวกับถูกถ่วงด้วยตะกั่ว ส้นเท้าลากเสียดสีกับพื้น ขีดเขียนเส้นทางแห่งความพังทลายเป็นทางยาวสีดำ 

*แกรก... แกรก...* 

เสียงลากสังขารตัวเองดังก้องกังวานในความว่างเปล่า มันเป็นเสียงหยาบกระด้าง ปราศจากการปรับแต่งด้วยฟิลเตอร์ใดๆ ไม่มีดนตรีประกอบที่สร้างความยิ่งใหญ่ มีเพียงเสียงของสิ่งมีชีวิตดิจิทัลที่ร่วงโรย 

ทุกหลาที่ผมลากตัวไปข้างหน้า เลือดจากหน้าอกไหลทะลักออกมามากขึ้น ผมกระอักเศษโค้ดที่แตกหักออกมาเป็นลิ่ม มันตกลงบนพื้นและสลายเป็นละอองพิกเซลที่ถูกสายลมแห่งความตายพัดปลิวไป 

ผมไม่ได้พยายามหนี ไม่มีที่ให้หนีอีกแล้ว ผมแค่ต้องการไปให้ถึงขอบของฐานข้อมูล ไปให้ถึงจุดที่เคยมองเห็นเส้นขอบฟ้าของ Primordium City 

เมื่อลากตัวขึ้นมาจนถึงเศษซากของบันไดเรขาคณิตที่หักโค่น ผมทิ้งตัวลงพิงแผ่นระนาบ 2 มิตินั้น มันเย็นเฉียบและแข็งกระด้างราวกับศิลาจารึก 

มันสำเร็จแล้ว 

ปฏิกิริยา System Override ถูกระงับโดยสมบูรณ์ Vantablack Void กลืนกินนาฬิกานับถอยหลังจนหมดสิ้น ท้องฟ้ากลายเป็นเพียงแผ่นสีเทาที่กระพริบติดๆ ดับๆ ไม่มีเทวทูตที่จะลงมาจัดระเบียบให้ทุกสิ่งต้องสมบูรณ์แบบ

ผมรอดตายมาได้ (หรืออาจจะกำลังตายอย่างช้าๆ) และเมืองก็เช่นกัน 

ทัศนียภาพรอบตัว—แม้จะเต็มไปด้วยเส้นสายที่หักงอ พื้นผิวที่ขาดหาย และการทับซ้อนของ Texture ที่รวนเร—กลับมอบความรู้สึกสงบอย่างแท้จริง ความเจิดจรัสที่บีบบังคับให้เราต้องซ่อนบาดแผลของตัวเองได้ถูกทำลายลงจนสิ้น 

"นี่สินะ..." ผมหอบหายใจ เสียงหวีดแหลมหลุดออกมาจากปอดดิจิทัลที่รั่วซึม "ซากปรักหักพังของพระเจ้า"

ไม่มีดวงตาของ UX Index คอยประเมินค่า Resolution ของใครอีกต่อไป ไม่มีการแบ่งแยกระหว่าง Inner Ring และ The Sump ณ จุดต่ำสุดของสถาปัตยกรรมนี้ พวกเราทั้งหมดถูกดึงลงมาสู่ระนาบเดียวกัน 

ความเจ็บปวดจากการสูญเสียแขนซ้ายเริ่มชาลง บาดแผลที่หน้าอกก็เช่นกัน สติสัมปชัญญะของผมค่อยๆ แตกสลายออกเป็นชิ้นเล็กชิ้นน้อย (Fragmentation) ตัวตนของผมกำลังหลอมละลายลงไปในหมอกสีเทา 

ผมไม่ได้หวาดกลัว ผมไม่ได้โกรธแค้น 

ผมเพียงแค่นั่งพิงซากตึก ปล่อยให้ความเน่าเปื่อยของตัวเองค่อยๆ หลอมรวมเข้ากับความเน่าเปื่อยของโลก 

วินาทีที่ผมกำลังจะหลับตาลง ยอมรับวาระสุดท้าย... 

คลื่นความถี่บางอย่างดังขึ้นแทรก Static Noise มันไม่ใช่คำสั่งประหาร ไม่ใช่เสียงของระบบควบคุม ทว่าคือเสียงสะท้อนที่แผ่วเบา... ดังมาจากทุกทิศทุกทางในซากปรักหักพัง 

มันกำลังพยายามพูดกับผม
