# บทที่ 36: รอยแยกของความหวัง (ตอนที่ 1)

ร่างของผมกำลังร่วงหล่น ไม่ใช่แค่ร่วงหล่นลงสู่ขอบของเขต Mid-Sectors แต่มันคือการพังทลายลงมาจากภายใน ยาแห่งความคมชัด (Clarity) ที่ผมฉีดเข้าร่างกายไปเมื่อครู่แท้จริงแล้วคือมัลแวร์ที่ปลอมตัวมา มันไม่ได้มอบความสมบูรณ์แบบ แต่มันกำลังฉีกกระชากระดับความละเอียดของผมให้แหลกละเอียด ความรู้สึกสากกระด้างแล่นพล่านไปทั่วร่าง บังคับให้ผมต้องลากสังขารที่พังทลายกระแทกเข้ากับขอบเหวของรอยแยกสีดำมืด

ข้อเท้าขวาของผมไร้ส้นเท้าคอยรับน้ำหนักอีกต่อไป มันหายไปอย่างสิ้นเชิงจากการโดนลบข้อมูล ผมล้มลุกคลุกคลาน ลากขาขวาที่ไร้การซัพพอร์ตไปตามพื้นบล็อกสีเทาหยาบๆ แขนซ้ายของผมก็เช่นกัน มันถูกกัดกินด้วยอาการ Bit Rot เน่าเปื่อยและอืดอาด ความรู้สึกชาสลับกับความเจ็บปวดแสบร้อนราวกับมีเข็มดิจิทัลนับพันเล่มทิ่มแทงทุกครั้งที่ผมพยายามขยับมัน

และลึกลงไปในแผงอกของผม ผมสัมผัสได้ถึงความหนักอึ้งของก้อนข้อมูล—เพย์โหลดกุญแจข้อมูล (Data Key Payload) ที่ถูกฝังเอาไว้ มันเต้นเป็นจังหวะแห่งความลับที่ผมต้องปกป้อง 

เบื้องหลังของผม เสียงเตือนความถี่สูงดังก้องในอากาศ... UX Index พวกมันกำลังล่าผมอยู่ หมาล่าเนื้อแห่งสถาปัตยกรรมของ The Archive ไม่ยอมปล่อยให้คนที่มี Error และกุมความลับแบบผมหลุดรอดไปได้

ที่สุดปลายทางของตรอกนั้น ผมถูกต้อนมาพบกับมัน—รอยแยก

มันไม่ใช่ประตูหรือช่องทางที่ถูกสร้างขึ้น แต่มันคือ "บาดแผล" ของสถาปัตยกรรมดิจิทัล รอยร้าวขนาดใหญ่บนกำแพงที่ไม่มีพื้นผิว โค้ดดิบๆ สีเขียวอมเหลืองไหลทะลักออกมาราวกับน้ำเลือดที่ไหลออกจากบาดแผลเปิด 

ผมยื่นมือขวาที่ยังพอใช้งานได้ออกไปเกาะขอบรอยแยก ความรู้สึกสากกระด้างแล่นปราดเข้าสู่ปลายนิ้ว มันเจ็บปวด แต่นี่คือทางลงสู่ The Sump บ่อเกรอะข้อมูล... ดินแดนที่กฎเกณฑ์ของ The Archive อาจจะเอื้อมไม่ถึง

ผมหอบหายใจอย่างสิ้นหวัง ขาขวาที่ขาดส้นเท้าไม่สามารถพยุงน้ำหนักตัวได้อีกต่อไป เมื่อเสียงฝีเท้าของ UX Index ใกล้เข้ามา ผมทิ้งน้ำหนักทั้งหมด... ปล่อยให้แรงโน้มถ่วงที่บิดเบี้ยวกลืนกินตัวตนของผม ร่วงหล่นลงสู่ความมืดที่ไม่มีจุดสิ้นสุด


# บทที่ 36: รอยแยกของความหวัง (ตอนที่ 2)

ทันทีที่ผมร่วงผ่านรอยแยก กฎเกณฑ์ของฟิสิกส์จำลองก็พังทลายลงอย่างสิ้นเชิง 

แรงโน้มถ่วงที่นี่ไม่ได้ดึงผมลงสู่เบื้องล่างเสมอไป มันบิดเบี้ยวและเปลี่ยนทิศทางอย่างคาดเดาไม่ได้ บางครั้งผมถูกเหวี่ยงให้กระแทกกับผนังที่มองไม่เห็น ทุกครั้งที่กระแทก ขาขวาที่ไร้ส้นเท้าของผมจะส่งข้อมูลความเจ็บปวดแปลบปลาบทะลุสถาปัตยกรรมร่าง แขนซ้ายที่ติดเชื้อ Bit Rot ชาหนึบและตอบสนองเชื่องช้าจนแทบไม่สามารถยกขึ้นมาป้องกันตัวเองได้ ร่างกายของผมเบาหวิวราวกับกลุ่มควัน แต่หน้าอกกลับรู้สึกหนักอึ้งด้วยภาระของเพย์โหลดกุญแจข้อมูลที่ฝังอยู่ภายใน

รอบตัวผมคือสายน้ำตกแห่งรหัส (Cascade of Code) ที่เน่าเสีย ตัวอักษรและตัวเลขที่ไร้ความหมายร่วงหล่นลงมาราวกับสายฝนที่เป็นกรด มันกัดกร่อนพื้นผิวอวาตาร์ของผมทุกครั้งที่สัมผัส ผมเห็นปื้นสีม่วงและเขียว—Compression Artifacts—ลุกลามไปทั่วซีกขวาของลำตัว มันผสมโรงตอกย้ำความทรมานแข่งกับแขนซ้ายที่เน่าเปื่อยจาก Bit Rot ร่างกายของผมกำลังสูญเสียความละเอียดอย่างรวดเร็ว ทุกเสี้ยววินาทีผมรู้สึกเหมือนถูกกระดาษทรายที่มองไม่เห็นขูดลอกผิวหนังดิจิทัลออกไป พิกเซลหลุดลุ่ยและแตกกระจายเป็นละอองฝุ่น ความรู้สึกแสบร้อนจากการถูกฉีกกระชากระดับพิกเซลแล่นปลาบไปทั่วเส้นประสาทจำลองราวกับถูกลอกผิวหนังสดๆ จนความหนาแน่นของร่างกายผมบางเบาลงอย่างน่าใจหาย

เสียงรอบข้างถูกบีบอัดจนกลายเป็นเสียงครางฮึมฮัมต่ำๆ ที่สั่นสะเทือนไปถึงกระดูกจำลองของผม มันไม่ใช่เสียงของลม แต่มันคือเสียงร้องของข้อมูลที่กำลังถูกทำลายกระบวนการประมวลผล 

ผมพยายามเพ่งมองฝ่าความมืดลงไปเบื้องล่าง แม้สายตาจะพร่ามัว แต่ผมรู้ดีว่าผมต้องไปต่อเบื้องบนนั้น... เหนือรอยแยกขึ้นไป เสียงสัญญาณปิง (Ping) แหลมก้องของ UX Index ยังคงสแกนหาตัวผม พวกมันตามกลิ่นแห่งความผิดพลาดของผมมาติดๆ ผมไม่มีทางถอยหลังกลับได้อีกแล้ว

จิตใจของผมเริ่มพร่าเลือน ความรู้สึกหวาดกลัวถูกลดทอนลงเหลือเพียงความอึดอัดที่ว่างเปล่า นี่คือกระบวนการ Resolution Loss ที่ผมต้องเผชิญจากการโดนมัลแวร์ ผมรู้ดีว่าถ้ายอมแพ้ต่อความเสื่อมสลายนี้ ผมจะกลายเป็นเพียง The Hollows—เปลือกข้อมูลที่ไร้สติปัญญา

แต่ในความบิดเบี้ยวนี้ ผมกลับรู้สึกถึงอิสรภาพบางอย่าง แม้จะเจ็บปวดจากอาการเน่าเปื่อยและขาที่พิการ แต่มันเป็นอิสรภาพจากการหลุดพ้นจากการควบคุม 

ผมหลับตาลง ปล่อยให้สายน้ำตกแห่งรหัสชะล้างโครงสร้างตัวตนของผมลงสู่ก้นบึ้ง สู่ความดิบเถื่อนที่รอคอยอยู่เบื้องล่าง หวังเพียงว่าความมืดนี้จะช่วยอำพรางผมจากเงื้อมมือของ UX Index
