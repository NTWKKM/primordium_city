# บทที่ 23: รสชาติของความโหยหา (ส่วนที่ 1)

บางครั้งผมก็ลืมไปแล้วว่าความหิวของมนุษย์จริงๆ เป็นอย่างไร

ในเขต Mid-Sectors ความหิวมักมาในรูปแบบของการแจ้งเตือนที่กะพริบจางๆ ตรงขอบการรับรู้ เหมือนสัญญาณเตือนแบตเตอรี่อ่อนที่ไร้เสียง ผมลากสังขารกะเผลกมานั่งพักที่โต๊ะในร้านอาหารมุมถนนแห่งหนึ่ง ซ่อนแขนซ้ายที่เน่าเปื่อยบวมฉุจาก Bit Rot ไว้ใต้โต๊ะ หอบหายใจอย่างยากลำบากเพราะความอึดอัดของ Data Key กลางอก และต้องคอยหยีตามองรอบๆ ด้วยความหวาดระแวงว่า The Hunter จะตามรอย High UX ของผมมาถึง ผิวโต๊ะที่เคยจำลองลวดลายไม้โอ๊กอย่างประณีต บัดนี้กลับสูญเสียความละเอียดจนเหลือเพียงสีน้ำตาลหยาบๆ (Texture loss) ราวกับภาพวาดสีน้ำที่ถูกละเลงด้วยฟองน้ำเปียก

ผมสั่งแอปเปิลดิจิทัลมาหนึ่งผล หวังเพียงเพื่อกระตุ้นรหัสความทรงจำเก่าๆ ให้ทำงาน ผมพยายามจะหยิบมัน แต่แขนซ้ายที่ติดเชื้อนั้นชาหนึบและกระตุกอย่างควบคุมไม่ได้จนทำแอปเปิลเกือบหลุดมือ ผมต้องฝืนเปลี่ยนมาใช้มือขวาที่สั่นเทาหยิบมันขึ้นมาแทน แอปเปิลลูกนั้นมีรูปทรงสมบูรณ์แบบเกินไป ไร้รอยช้ำ ไร้จุดด่างดำ แต่เมื่อผมกัดลงไป... ไม่มีเสียงกรอบ ไม่มีน้ำหวานที่ควรจะซึมออกมาตามรอยฟัน มีเพียงความรู้สึกของก้อนโพลีกอนที่แตกตัวออก และรสชาติที่จืดชืด ซ้ำซาก ราวกับกำลังเคี้ยวกระดาษลังที่ชุบน้ำตาลเทียมบางๆ 

มันคือ *ความพึงพอใจ* ระดับต่ำ

ผมหลับตาลง พยายามขุดค้นเข้าไปใน The Archive ส่วนตัว เพื่อค้นหาความทรงจำเกี่ยวกับ 'ความรัก' ผมจำได้ว่าเคยมีใครคนหนึ่ง... ใครสักคนที่ผมเคยจับมือ ท่ามกลางแสงไฟอบอุ่นใน Inner Ring แต่เมื่อผมพยายามดึงความรู้สึกนั้นขึ้นมา สิ่งที่ได้กลับไม่ใช่ความอบอุ่นวาบหวามในช่องท้อง หรืออัตราการเต้นของหัวใจที่รัวเร็ว 

ระบบของผมถูกบีบอัด (Emotional Compression) 

ความรักอันซับซ้อน มิติของความลุ่มหลง ความกลัวที่จะสูญเสีย... ทั้งหมดถูกรีดเร้นและบีบอัดจนเหลือเพียงสถานะเดียว: **"พึงพอใจ"** (Satisfaction) มันเหมือนกับการติ๊กเครื่องหมายถูกลงในช่องว่างของแบบฟอร์ม 'คุณรักเธอไหม?' - 'ใช่' 

แต่มันไม่มีรสชาติ ไม่มีน้ำหนัก ไม่มีแม้กระทั่งความเจ็บปวด

ผมลืมตาขึ้น มองดูพิกเซลที่ขอบแอปเปิลเริ่มหลุดลอยและเบลอเข้ากับอากาศรอบๆ กลิ่นอับของข้อมูลที่ถูกบีบอัดลอยแตะจมูก คล้ายกลิ่นโอโซนที่ผสมกับฝุ่นเหล็ก นี่คือรสชาติของความโหยหาใน Primordium... การโหยหาสิ่งที่คุณรู้ว่ามันมีอยู่ แต่คุณไม่เหลือ 'ความละเอียด' มากพอที่จะสัมผัสมันได้อีกต่อไป


# บทที่ 23: รสชาติของความโหยหา (ส่วนที่ 2)

"คุณดูเหมือนคนที่กำลังควานหาเหรียญในกระเป๋าที่ขาดนะ" 

เสียงนั้นถูกบิดเบือนความถี่ (Pitch-shifted) เล็กน้อย ทำให้ฟังดูสั่นเครือราวกับคลื่นวิทยุที่หาคลื่นไม่เจอ ผมเงยหน้าขึ้นมองพ่อค้าความทรงจำเร่ร่อนที่เดินเข้ามาใกล้ เขาเป็นชายในชุดโค้ทสีเทาตุ่นๆ ที่ชายเสื้อมีอาการ Stuttering กระตุกไปมาทุกครั้งที่เขาขยับตัว

"ผมมี 'จูบแรกท่ามกลางสายฝน' นำเข้าจาก Inner Ring" เขาเสนอ พลิกฝ่ามือให้เห็นก้อนข้อมูล (Data Cube) ส่องประกายสีฟ้าจางๆ "ความละเอียด 8K รับประกันความรู้สึกเปียกชื้นของเม็ดฝน และอัตราการเต้นของหัวใจที่พุ่งทะลุ 120 ครั้งต่อนาที สนใจไหม? แลกกับความทรงจำวัยเด็กของคุณสักชิ้นก็พอ"

ผมจ้องมองก้อนข้อมูลนั้น ความโหยหาผลักดันให้ผมเผลอยื่นแขนซ้ายออกไป ปลายนิ้วที่เต็มไปด้วยรอยแหว่ง Bit Rot แตะลงบนผิวกระจกของมันเพื่อพรีวิว... การเชื่อมต่อเต็มไปด้วยความล่าช้าและขัดข้อง

ทันใดนั้น ภาพหญิงสาวใต้ร่มสีใสก็วาบขึ้นมาในหัว แต่แทนที่จะเป็นสัมผัสอันลึกซึ้ง ระบบของผมกลับกระตุก ภาพใบหน้าของเธอแตกออกเป็นโมเสกสีเหลี่ยม (Pixelation) สัมผัสที่ริมฝีปากไม่ได้ให้ความรู้สึกถึงความอ่อนนุ่ม มันกลับสากและแข็งกระด้างเหมือนการเอาใบหน้าไปถูเบาๆ กับหน้าจอแก้วที่เย็นเฉียบ

ผมรีบชักแขนซ้ายที่ปวดแปลบออก หอบหายใจเบาๆ พร้อมกับอาการเจ็บปวดที่หน้าอกซึ่ง Data Key ขยับถ่วงน้ำหนักเอาไว้ ความกดดันจากสายตาของ The Hunter ที่ตามล่าอยู่ยังคงจ่อคอหอยราวกับลมหายใจมรณะ

ไม่ใช่ว่าสินค้าของเขาด้อยคุณภาพ แต่เป็น *ตัวผมเอง* ต่างหาก 

ความจุ (Bandwidth) ในการรับรู้อารมณ์ของผมลดต่ำลงจนไม่อาจประมวลผลความรู้สึกที่ซับซ้อนเหล่านั้นได้อีกต่อไป ต่อให้ผมซื้อความรักที่สมบูรณ์แบบที่สุดมา มันก็จะถูกระบบของผมบีบอัด (Compress) ให้กลายเป็นเพียงความพึงพอใจแบนๆ อยู่ดี 

"ไม่ล่ะ..." ผมปฏิเสธ เสียงของผมแหบพร่า "มัน... มันไม่ทำงานกับผมแล้ว"

พ่อค้าไหวไหล่ เก็บก้อนข้อมูลนั้นกลับไปโดยไม่ได้พูดอะไร ทิ้งผมไว้กับความเงียบและแอปเปิลโพลีกอนที่เริ่มสลายตัวเป็นฝุ่นพิกเซลบนโต๊ะ ความน่ากลัวที่แท้จริงของการเสื่อมสลาย ไม่ใช่การสูญเสียความทรงจำ แต่คือการตระหนักว่าคุณสูญเสีย *ความสามารถในการรู้สึก* ถึงมันอย่างถาวร
