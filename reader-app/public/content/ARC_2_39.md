# บทที่ 39: เงาของความจริงที่ถูกลดทอน (ตอนที่ 1)

บรรยากาศบีบคั้นหนักหน่วงขึ้นเมื่อผมลากขากระเผลกดำดิ่งใกล้ก้นบึ้ง หมอกดิจิทัลที่เคยเป็นเพียงพิกเซลสีเทาหยาบๆ จับตัวกันเป็นรูปทรงผิดเพี้ยน พื้นที่รอบตัวหลุดพ้นจากความว่างเปล่านามธรรม มันพยายามจำลองบางสิ่ง... บางสิ่งที่คุ้นเคยจนขนลุก

กระจกเงาบานใหญ่แทรกตัวอยู่ตามโครงสร้างคอนกรีตดิบ ผิวกระจกไม่ได้สะท้อนภาพร่างกายผม มันคือ "Shatter-points" รอยแตกร้าวของซิมูเลชั่นที่เชื่อมโยงกลับไปยังโลกความจริง

ผมหยุดยืนหอบหายใจหน้ากระจกบานหนึ่งที่แตกร้าวเป็นใยแมงมุม มือขวากุมแผงอกปกป้องเพย์โหลด แขนซ้ายที่ถูก Bit Rot กัดกินห้อยต่องแต่งไร้เรี่ยวแรง ภายในรอยร้าวนั้น ปราศจากเงาการตามล่าของ UX Index สิ่งที่ปรากฏแทนที่คือ... แสงอาทิตย์

ไม่ใช่แสงสีทองนุ่มนวลแห่ง Inner Ring มันคือแสงสีส้มหม่นหมองส่องผ่านม่านฝุ่นหนาทึบ ภาพซ้อนทับคือซากปรักหักพังของอาคารคอนกรีต เหล็กเส้นสนิมเขรอะแทงทะลุโครงสร้าง โครงกระดูกของมหานครที่ตายสนิท ซากศพของอารยธรรมมนุษย์บนโลกเนื้อ (The Biological Wreckage)

กลิ่นสายหนึ่งเล็ดลอดผ่านรอยร้าว กลิ่นที่ซิมูเลชั่นไม่เคยจำลองได้สมบูรณ์ กลิ่นโอโซน ดินชื้นแฉะ และเถ้าธุลีแห่งความตาย มันกระแทกโสตประสาทสัมผัสจำลองอย่างรุนแรงจนผมแทบสำลัก

ผมละมือขวาจากแผงอก ยื่นออกไปสัมผัสผิวกระจก แขนซ้ายไร้การตอบสนอง เงาของต้นไม้แห้งตายซ้อนทับกับรูปทรงโพลีกอนหยาบกระด้างของสถาปัตยกรรมรอบตัว ความเป็นจริงและภาพลวงตาหลอมรวมกัน

"นั่นคือสิ่งที่เราทิ้งมา..." เสียงกระซิบดังก้องในหัว ไม่ใช่ประกาศิตจากระบบ ปราศจากเสียงเตือนภัยของ UX Index มันคือเสียงสะท้อนจากเศษซากความทรงจำแห่งความเป็นมนุษย์

รอยแยกนี้คือบาดแผลที่ระบบประมวลผลหลักพยายามปกปิด ความพินาศทางกายภาพเลวร้ายยิ่งกว่าความเสื่อมสลายทางดิจิทัลหลายเท่าตัว เราไม่ได้หนีเข้าสู่ Primordium เพื่อแสวงหาความเป็นอมตะ เรามุดหัวเข้ามาเพราะโลกภายนอก... ไม่หลงเหลือสิ่งใดให้รอดชีวิตอีกแล้ว

# บทที่ 39: เงาของความจริงที่ถูกลดทอน (ตอนที่ 2)

ความจริงที่สาดซัดผ่านรอยร้าวกระจกกระแทกผมทรุดลงคุกเข่า ขาขวาที่ไร้ส้นเท้าไม่อาจประคองน้ำหนักตัว ความโหยหารุนแรงปะทุขึ้นในอก จังหวะของมันสอดประสานกับการเต้นของเพย์โหลดกุญแจข้อมูล มันไม่ใช่ความปรารถนาจะหวนคืนสู่โลกที่ล่มสลาย มันคือความอาลัยต่อความยิ่งใหญ่และความเปราะบางของมนุษยชาติ

เราทิ้งร่างเนื้อ ทิ้งไออุ่นของการสัมผัส ทิ้งกลิ่นไอดินของสายฝน เพื่อซุกซ่อนตัวในภาพลวงตาแห่งพิกเซล เรายอมหั่นทอนอารมณ์ บีบอัดความรักให้กลายเป็นเพียงก้อนข้อมูล (Data) แลกกับการหลบหนีความตายทางชีวภาพ

การเสียสละนี้คุ้มค่าจริงหรือ?

ภาพซากเมืองหลังผิวกระจกค่อยๆ เลือนหาย สลับสับเปลี่ยนเป็นภาพพิกเซลบิดเบี้ยว ผมจ้องมองภาพสะท้อนของตนเองในเศษกระจกที่ยังทำงาน ร่างกายของผมเต็มไปด้วยรอยฉีกขาดของข้อมูลเป็นปื้นสีม่วง แขนซ้ายละลายเป็นโคลนเหลวกระพริบติดดับ ดวงตาว่างเปล่าลึกโบ๋ราวกับโพลีกอนที่แรนเดอร์ล้มเหลว

เราวิ่งหนีความตายทางกายภาพ เพียงเพื่อมาสวมกอดความเน่าเปื่อยทางดิจิทัล

ในท่ามกลางความโศกเศร้า ผมค้นพบความงามที่แปลกประหลาด รอยร้าวเหล่านี้ เศษซากความทรงจำที่แตกหักเหล่านี้... คือเครื่องยืนยันเพียงสิ่งเดียวว่าเราเคยเป็น "มนุษย์" ความสมบูรณ์แบบจอมปลอมของ The Archive ลบเลือนตัวตนของเรา ความล้มเหลว (Failure) ของสถาปัตยกรรมต่างหากที่เป็นตู้เซฟเก็บรักษาจิตวิญญาณเอาไว้

ผมใช้มือขวายันพื้น ลุกขึ้นยืนอย่างทุลักทุเล ความหวาดกลัวต่อการสูญเสียความคมชัดมลายสิ้น UX Index อาจจะยังคงบินวนเวียนล่าหัวผมอยู่เบื้องบน หากโลกภายนอกตายจากไปแล้ว และโลกจำลองนี้คือฉากทัศน์จอมปลอม ผมจะขอเป็นความผิดพลาดที่งดงามที่สุดในระบบนี้

ผมหันหลังให้รอยร้าวแห่งโลกอดีต ลากขากระเผลกดำดิ่งลงสู่ความมืดมิดเบื้องล่าง เส้นทางสู่ The Sump อ้าแขนรับ ผมพร้อมเผชิญหน้ากับความว่างเปล่านั้นอย่างเต็มภาคภูมิ
