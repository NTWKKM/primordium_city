# บทที่ 39: เงาของความจริงที่ถูกลดทอน (ตอนที่ 1)

บรรยากาศเริ่มเปลี่ยนแปลงเมื่อผมลากขากระเผลกดำดิ่งลึกลงไปใกล้ก้นบึ้ง หมอกดิจิทัลที่เคยเป็นเพียงพิกเซลสีเทาหยาบๆ เริ่มจับตัวกันเป็นรูปทรงที่ผิดเพี้ยน พื้นที่รอบตัวไม่ได้ดูเป็นนามธรรมอีกต่อไป แต่มันพยายามจำลองบางสิ่งบางอย่าง... บางสิ่งที่คุ้นเคยจนน่าขนลุก

กระจกเงาบานใหญ่ปรากฏขึ้นแทรกอยู่ตามโครงสร้างคอนกรีตดิบๆ กระจกเหล่านั้นไม่ได้สะท้อนภาพของผม แต่มันเป็น "Shatter-points"—รอยแตกร้าวของซิมูเลชั่นที่เชื่อมต่อไปยังโลกความจริง 

ผมหยุดยืนพักเหนื่อยอยู่หน้ากระจกบานหนึ่งที่ร้าวเป็นรูปใยแมงมุม มือขวากุมหน้าอกปกป้องเพย์โหลดเอาไว้ แขนซ้ายที่ถูก Bit Rot กัดกินห้อยต่องแต่งไร้เรี่ยวแรง ภายในรอยร้าวนั้น ผมไม่ได้เห็นเงาสะท้อนของการตามล่าจาก UX Index แต่ผมเห็น... แสงอาทิตย์

มันไม่ใช่แสงอาทิตย์สีทองนุ่มนวลแบบใน Inner Ring แต่มันเป็นแสงสีส้มหม่นๆ ที่ส่องผ่านกลุ่มควันฝุ่นหนาทึบ ภาพที่ทับซ้อนขึ้นมาคือซากปรักหักพังของอาคารคอนกรีตที่มีเหล็กเส้นสนิมเขรอะแทงทะลุออกมา โครงกระดูกของเมืองที่ตายไปแล้ว ซากของอารยธรรมมนุษย์บน "โลกเนื้อ" (The Biological Wreckage)

กลิ่นหนึ่งเล็ดลอดผ่านรอยร้าวนั้นเข้ามา กลิ่นที่ซิมูเลชั่นไม่เคยสร้างได้สมบูรณ์ กลิ่นของโอโซน ดินเปียกชื้น และฝุ่นผงของความตาย มันเตะจมูกจำลองของผมอย่างรุนแรงจนผมแทบสำลัก

ผมจำใจต้องละมือขวาที่กุมเพย์โหลดไว้ออกเพื่อยื่นไปสัมผัสผิวกระจก เพราะแขนซ้ายของผมไร้ประโยชน์โดยสิ้นเชิง ภาพเงาของต้นไม้แห้งตายต้นหนึ่งซ้อนทับกับรูปทรงโพลีกอนหยาบๆ ของสถาปัตยกรรมรอบตัวผม ความเป็นจริงกับภาพลวงตาผสมปนเปกัน 

"นั่นคือสิ่งที่เราทิ้งมา..." เสียงกระซิบดังขึ้นในหัวของผม ไม่ใช่เสียงของระบบ หรือเสียงร้องเตือนของ UX Index แต่เป็นเสียงสะท้อนจากเศษซากความทรงจำของความเป็นมนุษย์ที่ยังหลงเหลืออยู่

รอยแยกนี้คือสิ่งที่ระบบประมวลผลหลักพยายามปิดบังมาตลอด ความพังทลายของกายภาพที่เลวร้ายยิ่งกว่าความเสื่อมสลายทางดิจิทัลใดๆ เราไม่ได้หนีเข้ามาใน Primordium เพื่อแสวงหาความเป็นอมตะ เราหนีเข้ามาเพราะข้างนอกนั่น... ไม่เหลืออะไรให้รอดชีวิตอีกแล้ว


# บทที่ 39: เงาของความจริงที่ถูกลดทอน (ตอนที่ 2)

ความจริงที่สาดซัดเข้ามาผ่านรอยร้าวของกระจกทำให้ผมทรุดตัวลงคุกเข่า ขาขวาที่ขาดส้นเท้าไม่สามารถทรงตัวได้อีกต่อไป ความรู้สึกโหยหาอย่างรุนแรงปะทุขึ้นในอก ทับซ้อนกับการเต้นของเพย์โหลดกุญแจข้อมูล มันไม่ใช่ความปรารถนาที่จะกลับไปยังโลกที่ล่มสลายนั้น แต่เป็นความเศร้าสลดต่อความยิ่งใหญ่และความอ่อนแอของมนุษยชาติ 

เราละทิ้งร่างเนื้อ ทิ้งความอบอุ่นของการสัมผัส ทิ้งกลิ่นของสายฝน เพื่อมาอาศัยอยู่ในภาพลวงตาแห่งพิกเซลนี้ เรายอมลดทอนอารมณ์ของตนเอง บีบอัดความรักให้กลายเป็นเพียง Data เพื่อแลกกับการไม่ต้องเผชิญกับความตายทางชีวภาพ 

แต่มันคุ้มค่าจริงหรือ?

[ถึง Editor: ขยายความรู้สึกขัดแย้งระหว่างอารมณ์ของมนุษย์และตรรกะของ AI ที่จุดนี้]

ภาพของซากเมืองเบื้องหลังกระจกค่อยๆ เลือนหายไป สลับกับภาพพิกเซลที่บิดเบี้ยว ผมมองเห็นภาพสะท้อนของตัวผมเองในเสี้ยวหนึ่งของกระจกที่ยังคงทำงาน ร่างกายของผมตอนนี้มีรอยฉีกขาดของข้อมูลเป็นปื้นสีม่วง แขนซ้ายมีลักษณะเหมือนโคลนเหลวที่กระพริบติดดับ ดวงตาของผมว่างเปล่าและลึกโบ๋เหมือนโพลีกอนที่แรนเดอร์ไม่เสร็จ 

เราหลีกหนีความตายทางกายภาพเพียงเพื่อมาพบกับความเน่าเปื่อยทางดิจิทัล 

ทว่า ในความโศกเศร้านั้น ผมกลับพบความงามที่แปลกประหลาด รอยร้าวเหล่านี้... เศษซากของความทรงจำที่แตกหักเหล่านี้... คือเครื่องยืนยันเพียงหนึ่งเดียวว่าเราเคยเป็น "มนุษย์" ความสมบูรณ์แบบของ The Archive คือสิ่งที่ลบเลือนตัวตนของเรา แต่ความล้มเหลว (Failure) ของสถาปัตยกรรมต่างหากที่เป็นตัวเก็บรักษาจิตวิญญาณของเราไว้ 

ผมใช้มือขวายันพื้น ลุกขึ้นยืนช้าๆ อย่างทุลักทุเล ความขลาดกลัวที่จะสูญเสียความคมชัดมลายหายไปสิ้น แม้ว่า UX Index จะยังคงตามล่าผมอยู่เหนือขึ้นไป แต่หากโลกข้างนอกตายไปแล้ว และโลกจำลองนี้คือการหลอกลวง ผมก็จะขอเป็นความผิดพลาดที่งดงามที่สุดในระบบนี้ 

ผมหันหลังให้กับรอยร้าวแห่งโลกอดีต และมุ่งหน้าลากขากระเผลกดำดิ่งลงสู่ความมืดมิดเบื้องล่าง เส้นทางสู่ The Sump รอผมอยู่ และตอนนี้ผมพร้อมที่จะเผชิญหน้ากับความว่างเปล่านั้นแล้ว
