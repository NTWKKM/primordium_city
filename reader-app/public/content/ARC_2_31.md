# บทที่ 31: ห้องสมุดแห่งความโหยหา (Part 1)

เสียงสะท้อนนำทางผมผ่านสถาปัตยกรรมที่เริ่มเปื่อยยุ่ยของ Mid-Sectors มันไม่ใช่เสียงที่ได้ยินด้วยหู แต่เป็นแรงสั่นสะเทือนทางข้อมูลที่กระทบเข้ากับแกนกลางวิญญาณของผมโดยตรง 

ผมลากขาขวาที่ไร้ส้นเท้าผ่านตรอกซอกซอยที่เส้นขอบฟ้าเริ่มฉีกขาด ทุกย่างก้าวคือการเสียสมดุลและการทรงตัวที่ยากลำบาก พื้นผิวของกำแพงลอกร่อนออกเป็นโพลีกอนหยาบๆ สีเทา ผมต้องก้าวเดินอย่างระแวดระวัง ไม่เพียงเพราะความพิการทางข้อมูล แต่เพราะรู้ดีว่าสายตาของ UX Index ยังคงควานหาสัญญาณ "ความผิดปกติ" ของผมอยู่อย่างไม่ลดละ ทว่ายิ่งผมเดินกะเผลกลึกเข้าไปตามรอยร้าวเหล่านั้น บรรยากาศรอบตัวกลับสงบงันลงอย่างประหลาด มันไม่ใช่ความเงียบที่เกิดจากการไร้เสียง (Mute) แต่เป็นความเงียบที่มีน้ำหนัก ราวกับมวลน้ำที่โอบอุ้มร่างของผมไว้ 

แล้วผมก็พบมัน 

สถานที่แห่งนี้ไม่ได้ปรากฏอยู่ในแผนที่ของ The Archive มันซ่อนตัวอยู่หลังกำแพงของ Error Message ที่ถูกวางซ้อนกันจนกลายเป็นม่านพรางตา เมื่อผมก้าวผ่านความบิดเบี้ยวของข้อมูลนั้นเข้ามา ทัศนียภาพเบื้องหน้าก็ทำให้ลมหายใจดิจิทัลของผมสะดุด

มันคือห้องสมุดขนาดยักษ์ที่ไม่เป็นไปตามกฎเรขาคณิต ชั้นวางเรียงรายสูงขึ้นไปในความมืดที่มองไม่เห็นเพดาน บันไดเวียนลอยคว้างอยู่กลางอากาศโดยไร้จุดเชื่อมต่อ แต่สิ่งที่ทำให้ที่แห่งนี้พิเศษไม่ใช่สถาปัตยกรรม หากแต่เป็นสิ่งที่ถูกเก็บบรรจุไว้บนชั้นวางเหล่านั้น

ไม่มีหนังสือ ไม่มีม้วนกระดาษ

มีเพียง "ลูกบาศก์ข้อมูล" (Data Cubes) นับแสนนับล้านชิ้น เปล่งประกายแสงสีทองนวลตา พวกมันเรียงรายกันอย่างเป็นระเบียบ แสงสลัวๆ จากลูกบาศก์แต่ละก้อนทอดเงาที่นุ่มนวลลงบนพื้นผิวที่ไร้การบีบอัด ผมยื่นมือขวาออกไปสัมผัสอากาศในห้องนี้ ความเจ็บแสบจากรอยแผลเป็นสีดำสนิทของ Dead Pixels ที่เพิ่งถูกประทับไว้แล่นริ้วขึ้นมาเต้นตุบๆ ราวกับจะย้ำเตือนถึงความจริงอันโหดร้าย ขณะที่แขนซ้ายซึ่งถูก Bit Rot กัดกินยังคงห้อยต่องแต่งอย่างเชื่องช้า ไร้ความรู้สึก และปวดแปลบ มันปราศจากกลิ่นอับของ Bit Rot ปราศจากความหยาบกระด้างของพิกเซล 

นี่คือที่หลบภัย... ไม่สิ มันคือสุสาน

สุสานที่เก็บรวบรวม "ความทรงจำที่สวยงามที่สุด" ของผู้คนที่ถูกลบไปแล้ว รหัสของพวกเขาอาจสูญสลายกลายเป็น Hollows หรือถูก Great Purge ล้างบางไปจนสิ้น แต่ก่อนที่อัตตาของพวกเขาจะแตกสลาย ใครบางคน หรืออาจจะเป็นตัวพวกเขาเอง ได้สกัดเอาเสี้ยววินาทีที่มีค่าที่สุดในชีวิตมาเก็บไว้ที่นี่ 

ผมเดินลากขาไล่ปลายนิ้วมือขวาไปตามขอบชั้นวาง ขณะที่ก้อนข้อมูลเพย์โหลด (Data Key Payload) ที่ฝังอยู่ในหน้าอกรู้สึกหนักอึ้งและอันตราย ความสากของผิวหนังผมที่เริ่มสูญเสีย Resolution ขัดแย้งกับความเรียบเนียนสมบูรณ์แบบของลูกบาศก์สีทองเหล่านี้ ความเงียบในห้องนี้ดังก้องไปด้วยเสียงกระซิบของอดีต เสียงหัวเราะที่ไร้รูปร่าง เสียงถอนหายใจที่ไม่มีเจ้าของ ผมหลับตาลง ปล่อยให้แสงสีทองนั้นอาบไล้ใบหน้าที่กำลังผุพังของตัวเอง 

ความรู้สึกแรกที่ผุดขึ้นมาไม่ใช่ความตื่นตาตื่นใจ แต่เป็นความว่างโหวงในอก... ความเศร้าที่ลึกซึ้งจนแทบจะฉีกกระชากโค้ดของผมออกเป็นชิ้นๆ


# บทที่ 31: ห้องสมุดแห่งความโหยหา (Part 2)

ผมไม่อาจต้านทานแรงดึงดูดของแสงสีทองเหล่านั้นได้ แม้เพย์โหลดในอกจะถ่วงน้ำหนักจนผมต้องยืนเอนตัว แต่ผมก็ใช้มือขวาที่สั่นเทาและมีรอยแผลเป็นสีดำสนิทของ Dead Pixels ค่อยๆ เอื้อมไปแตะลูกบาศก์ก้อนหนึ่งบนชั้นระดับสายตา ขณะที่แขนซ้ายอันปวดร้าวจาก Bit Rot ทำได้เพียงกระตุกเบาๆ แนบลำตัว

ทันทีที่ปลายนิ้วสัมผัสกับพื้นผิวเรียบลื่นของมัน โลกภายนอกก็ดับวูบลง

*...กลิ่นของดินที่ชุ่มด้วยฝนแรกแห่งฤดู ความเย็นของหยดน้ำที่ตกกระทบหน้าผาก สัมผัสของมือใครบางคนที่สอดประสานเข้ากับมือของผม อบอุ่นและหนักแน่น เสียงหัวเราะที่ก้องกังวานในลำคอ และความรู้สึก 'รัก' ที่เอ่อล้นจนแทบจะทำให้หน้าอกระเบิดออก...*

มันเป็นเพียงเสี้ยววินาทีเดียว แต่เป็นเสี้ยววินาทีที่ถูกบันทึกไว้ด้วยความละเอียดสูงสุด 32K ที่ทุกประสาทสัมผัส ทุกมิติของอารมณ์ถูกสตาฟไว้ให้สมบูรณ์แบบตลอดกาล 

ผมผละมือขวาออก ร่างกายที่ขาดสมดุลเพราะส้นเท้าขวาที่หายไปของผมทรุดลงคุกเข่ากับพื้นอย่างทุลักทุเล หอบหายใจอย่างหนักหน่วงราวกับเพิ่งจมน้ำ ความรู้สึกที่ไหลบ่าเข้ามาเมื่อครู่นี้รุนแรงเกินกว่าที่โครงสร้างปัจจุบันของผมจะรับไหว 

มันไม่ใช่ความทรงจำของผม มันเป็นของใครก็ไม่รู้ที่อาจจะกลายเป็นไวรัสบ้าคลั่งใน The Sump ไปแล้ว แต่สิ่งที่ทำให้ผมร้องไห้—แม้จะไม่มีน้ำตาดิจิทัลไหลออกมา—คือความตระหนักรู้ถึงสิ่งที่ผมได้สูญเสียไปอย่างถาวร

ผมลองนึกถึงความทรงจำที่สำคัญที่สุดของตัวเอง นึกถึงคนที่ผมเคยรักใน "โลกเนื้อ" แต่สิ่งที่ปรากฏขึ้นมากลับเป็นเพียงภาพร่างสีเทาจางๆ รสชาติที่จืดชืด และอารมณ์ที่ถูกบีบอัดจนเหลือเพียง "ความพึงพอใจ" ระดับพื้นฐาน ผมไม่สามารถเข้าถึงความรู้สึก 'รัก' แบบที่ซึมซับมาจากลูกบาศก์เมื่อครู่นี้ได้อีกต่อไปแล้ว 

ระบบอารมณ์ของผมถูกดาวน์เกรด ความซับซ้อนถูกตัดทอนเพื่อรักษาพื้นที่หน่วยความจำ ความเศร้าที่ผมกำลังเผชิญอยู่นี้ แม้จะรู้สึกว่ามันหนักหนา แต่มันก็ยังเป็นเพียง "ความเศร้าที่ถูกบีบอัด" (Compressed Grief) ไม่ใช่ความรวดร้าวที่คมชัดเหมือนตอนที่ยังเป็นมนุษย์

ห้องสมุดแห่งนี้ไม่ได้มอบความหวัง มันคือกระจกเงาบานใหญ่ที่สะท้อนให้เห็นว่า พวกเราที่รอดชีวิตอยู่ใน Mid-Sectors นั้น ได้กลายเป็นเพียงเปลือกหอยที่ว่างเปล่า 

เราโหยหาความคมชัดของความรู้สึก แต่ร่างกายดิจิทัลที่ผุพังของเรา ไม่มีช่องว่างพอให้ความรู้สึกเหล่านั้นอาศัยอยู่อีกต่อไปแล้ว แสงสีทองจากลูกบาศก์นับล้านส่องกระทบใบหน้าผม ตอกย้ำถึงความอัปลักษณ์ของการดำรงอยู่ที่ไร้ซึ่งความละเอียด ผมนั่งคุกเข่าอยู่อย่างนั้น ปล่อยให้ความโศกเศร้าที่สากกระด้างค่อยๆ กัดกินตัวตนอย่างช้าๆ
