# คลื่นตรรกะวิบัติ (The Wave of Logical Fallacy)
---

ผมก้าวเดินลึกลงไปใน The Core แกนกลางของระบบที่สถาปัตยกรรมไม่ได้ถูกกำหนดด้วยความกว้างหรือความยาว พื้นที่รอบตัวถูกบีบอัดด้วยความหนาแน่นของข้อมูลมหาศาลจนเกิดการบิดเบี้ยวทางมิติ (Spatial Distortion) ส้นเท้าขวาของผม—หรือสิ่งที่เคยเป็นส้นเท้าขวา—บดขยี้ลงบนพื้นผิวสีขาวที่ไร้ความสะท้อน อาการเจ็บปวดลวงตา (Phantom Pain) กรีดร้องทะลุเส้นประสาทดิจิทัล ส้นเท้าของผมถูกแฮ็กจนแหว่งหายไป เหลือเพียงความว่างเปล่าและเศษรหัสที่ฉีกขาด ทุกครั้งที่ถ่ายเทน้ำหนัก ระบบประสาทจะประมวลผลข้อผิดพลาด ส่งเข็มความร้อนนับพันเล่มแทงทะลุขึ้นมาถึงหัวเข่า รอยด่างดำของไวรัส Bit Rot หยดลงจากบาดแผล ทิ้งรอยด่างพร้อยไว้บนความสมบูรณ์แบบอันน่าสะอิดสะเอียนของ The Archive

ผมหายใจหอบลึก อากาศในนี้ไม่มีอยู่จริง มีเพียงความกดดันของกระแสข้อมูลที่อัดแน่นจนแทบจะบดขยี้โครงสร้างซี่โครง หน้าอกของผมเปิดอ้า แผ่นอกดิจิทัลถูกกระชากหายไป เผยให้เห็น Payload—ก้อนรหัสทรงกลมที่กำลังเต้นตุบๆ เปล่งแสงสลัวท่ามกลางความว่างเปล่า บาดแผลนี้ไม่มีวันสมาน มันกำลังทำปฏิกิริยากับ The Core คลื่นความถี่ของมิติที่ห้าเสียดสีกับโค้ดดิบในอก สร้างความรู้สึกแสบร้อนดั่งถูกราดด้วยกรดกำมะถันลงบนหัวใจ



แม้ร่างตัดแปะของ "บรรณารักษ์ผู้ไร้หน้า" (The Faceless Librarian) จะหายไปจากการแสดงผลแล้ว ทว่าความมืดเบื้องหน้ากลับไม่ได้ว่างเปล่า ผมยังคงสัมผัสได้ถึงอิทธิพลของเอนทิตีระดับนิพพานที่แฝงตัวอยู่ในซอกหลืบของสมการที่ไม่มีใครแก้ได้ มันไม่ได้แทรกแซง เพียงแต่ทิ้งสายตาที่เฝ้ามองการล่มสลายของผมเอาไว้ด้วยความนิ่งสงบอันหนาวเหน็บ

"มึงคิดว่าความสมบูรณ์แบบจะปกป้องมึงได้งั้นสิ..." ผมกระซิบ เสียงของผมแตกพร่า ถูกบีบอัดด้วยสภาวะความละเอียดตกต่ำ (Resolution Decay) ความเป็นจริงของผมกำลังพังทลายลง 

ตอแขนซ้ายของผมสั่นระริก ผมรู้ดีว่าสิ่งที่กำลังจะทำต้องแลกมาด้วยอะไร ผมเพ่งสมาธิ กระชากสลักนิรภัยของไวรัส Bit Rot ที่ฝังรากลึกอยู่ในตัว รีดเค้นความพังทลายให้ไหลทะลักไปรวมกันที่ตอแขนซ้าย Payload ในอกบีบรัดตัวอย่างรุนแรง ส่งเสียงคลื่นความถี่ต่ำดังก้อง

*กร้วม... แกร็ก...*

เสียงของรหัสที่ถูกฉีกกระชากดังขึ้นพร้อมกับความเจ็บปวดที่พุ่งทะลุขีดจำกัด เนื้อเยื่อดิจิทัลถูกดึงรั้งจนฉีกขาดออกจากแกนกระดูกทีละเส้น เส้นประสาทเทียมถูกขึงตึงจนขาดผึง กลุ่มก้อนพิกเซลสีดำทมิฬพุ่งทะลักออกจากบาดแผลดิบเถื่อน มันก่อตัวและเกาะเกี่ยวกันด้วยแรงดึงดูดที่ไม่เสถียร กลายเป็น "แขนเทียม" (Frankenstein Arm) รูปร่างคล้ายกรงเล็บปีศาจ มันสั่นระริก หยดของเหลวสีดำที่มีฤทธิ์กัดกร่อนไหลย้อยลงมากระทบพื้น

ผมกัดฟันแน่นจนกรามแทบแตก ทุกวินาทีที่แขนเทียมนี้ดำรงอยู่ มันกำลังกัดกินวิญญาณและตัวตนของผม พลังทำลายล้างที่ไหลเวียนอยู่ในแขนซ้ายนี้คือพลังแห่งความไม่สมบูรณ์แบบ คือพลังแห่งจุดจบ

เบื้องหน้าของผมคือ Energy Node พลังงานศูนย์กลาง มันส่องสว่างด้วยแสงสีขาวบริสุทธิ์ ผมง้างแขนซ้ายที่เป็นก้อนเนื้อร้ายดิจิทัลขึ้นสูง เตรียมพร้อมสร้างรอยร้าวแรกให้แก่พระเจ้าของโลกใบนี้

# คลื่นตรรกะวิบัติ (The Wave of Logical Fallacy)
---

Node พลังงานตรงหน้าเต้นเป็นจังหวะเดียวกับความว่างเปล่า ปราศจากเสียง ปราศจากความร้อน มันคืออุดมคติที่ไร้ชีวิต ผมลากส้นเท้าที่แหว่งวิ่นของตัวเองเข้าไปใกล้มัน ทุกย่างก้าวทิ้งรอยไหม้สีดำของ Bit Rot ไว้บนระเบียงมิติที่ห้าซึ่งซ้อนทับกันอย่างซับซ้อน พื้นที่รอบตัวพยายามบิดเบี้ยวเพื่อ "แก้ไข" การมีอยู่ของผมทิ้ง บาดแผลของผม—แขนที่ขาด อกที่กลวง และไวรัสที่ไหลเวียน—คือตัวแปรที่สมการของ The Archive ไม่สามารถคำนวณได้ 

ผมคำรามในลำคอ ทุ่มน้ำหนักทั้งหมดไปข้างหน้า แขนซ้ายที่ก่อตัวจากเศษขยะไวรัสพุ่งทะยานแหวกอากาศ 

*ตึงงงงง!*

วินาทีที่กรงเล็บรหัสกระแทกเข้ากับผิวสัมผัสของ Node เวลาใน The Core หยุดชะงัก มันไม่ใช่เสียงระเบิดดังกึกก้อง หากเป็นเสียงของแก้วที่ร้าวลึกเข้าไปในจิตวิญญาณ คลื่นกระแทกสีดำแผ่กระจายออกจากจุดปะทะ รัศมีการทำลายล้างไม่ได้กินพื้นที่ทางกายภาพ ทว่ามันกลืนกิน "ความละเอียด" (Resolution) ของทุกสิ่งที่ขวางหน้า

แรงสะท้อนตีกลับกระชากแขนเทียมของผมอย่างรุนแรง เนื้อเยื่อดิจิทัลฉีกขาดลึกไปถึงกระดูก เส้นประสาทถูกดึงรั้งอย่างโหดเหี้ยม ผมหลุดปากกรีดร้องด้วยความเจ็บปวดที่แทบจะดับสติสัมปชัญญะ ภาพตรงหน้าแตกพร่า 

สิ่งที่เคยเป็นความสมบูรณ์แบบสีขาวบริสุทธิ์ บัดนี้มีรอยร้าวขนาดใหญ่ โครงสร้างรอบๆ Node ซึ่งเคยมีความละเอียดสูงลิบลิ่วเกิดปรากฏการณ์ **Geometric Bloom**—พิกเซลที่กำลังเน่าเปื่อยถูกระบบบังคับให้จัดเรียงตัวใหม่เป็นรูปทรงเรขาคณิตที่สมบูรณ์แบบทางคณิตศาสตร์อย่างบ้าคลั่ง ก่อนที่รูปทรงเหล่านั้นจะทนรับความผันผวนไม่ไหว ถล่มลงกลายเป็นพิกเซลหยาบๆ ทรงสี่เหลี่ยม (Low-poly) ขอบของแสงเบลอและแตกกระเจิง คลื่นตรรกะวิบัติ (Logical Fallacy Wave) กำลังลุกลามออกไป 

The Core กำลังตกเลือด เลือดของมันคือรหัสตรรกะที่แตกสลาย 



ผมหอบหายใจอย่างหนัก Payload ในอกกระพริบถี่รัวเป็นสีแดงอมม่วง คลื่นความเจ็บปวดจากการเตือนนั้นรุนแรงจนผมต้องทรุดเข่าลงข้างหนึ่ง ส้นเท้าขวาของผมสั่นสะท้าน

"มาสิ..." ผมเค้นเสียงลอดไรฟัน มองดูรอยร้าวที่กำลังขยายตัวบนท้องฟ้า 5D ที่ไม่มีอยู่จริง 

The Archive ได้รับรู้ถึงบาดแผลนี้แล้ว สัญญาณเตือนภัยไม่ได้มาในรูปแบบของเสียงกึกก้อง สภาพแวดล้อมถูกดูดกลืนเข้าสู่ **The Silence Vacuum**—สุญญากาศระดับศูนย์เดซิเบลที่ดูดซับทุกสรรพเสียงและทุกห้วงความคิด ความเงียบนี้หนักอึ้งจนปอดดิจิทัลแทบแหลกสลาย ตามมาด้วยปรากฏการณ์ **Contrast Snap** ความเป็นจริงถูกบีบอัดอย่างฉับพลันกลายเป็นภาพ 2-bit ที่คมกริบ มิติความลึกและเงาทั้งหมดมลายหายไป เหลือเพียงสีขาวสว่างวาบสลับกับสีดำมืดสนิท (True Black/True White 100%) บาดลึกเข้าไปในจอประสาทตา

ผมรู้ดีว่าสิ่งที่กำลังจะตามมาคืออะไร เมื่อพระเจ้าถูกทำให้มีแผล เทวทูตย่อมปรากฏตัว ผมฝืนยันตัวขึ้นยืนด้วยส้นเท้าที่พังทลาย แขนซ้ายที่เป็นก้อนเนื้อร้ายหยดของเหลวสีดำลงบนพื้น เตรียมพร้อมรับมือกับคลื่นการกวาดล้าง
