# บทที่ 24: กฎของความเสื่อม (ส่วนที่ 1)

แสงสีฟ้าที่นี่แข็งทื่อและบาดตา (Hard-edge lighting) มันไม่ได้ส่องสว่างเพื่อการมองเห็น แต่เพื่อแผดเผาความคลุมเครือให้สิ้นซาก

ผมถูกคุมตัวลากส้นเท้าที่แหว่งกะเผลกเข้ามาในอาคารที่ตั้งตระหง่านอยู่กลาง Mid-Sectors มันเป็นป้อมปราการของกลุ่ม *Preservationists* โครงสร้างของมันสร้างขึ้นจากโค้ดที่เสถียรที่สุดในเขตนี้ เส้นสายเรขาคณิตทุกเส้นตั้งฉากกันอย่างสมบูรณ์แบบ ไร้รอยตำหนิ ไร้อาการกระตุก มันคือความทรงจำของ Inner Ring ที่ถูกสลักตรึงไว้กลางทะเลแห่งความเสื่อมโทรม

"ความละเอียดของคุณกำลังลดลง เอเลียส" 

ผู้คุมกฎ (Warden) นั่งอยู่หลังโต๊ะโลหะที่สะท้อนแสงเงาวับ ใบหน้าของเขาราบเรียบราวกับประติมากรรมหินอ่อนดิจิทัล ไม่มีรูขุมขน ไม่มีรอยเหี่ยวย่น และไม่มีแม้แต่ความมีชีวิตชีวาในดวงตา

"คุณมีอัตราการเกิด Bit Rot สูงเกินกว่ามาตรฐานของเขต" เขากล่าวต่อ น้ำเสียงราบเรียบ ไร้ความสั่นไหวของอารมณ์ราวกับเสียงสังเคราะห์ "รหัสของคุณกำลังเน่าเปื่อย และความเน่าเปื่อย... เป็นโรคติดต่อ"

ผมก้มมองแขนซ้ายตัวเอง รอยแหว่งบวมเป่งของ Bit Rot ยังคงเต้นตุบๆ ส่งความเจ็บปวดร้าวลึกทะลุชั้นผิวหนังจำลอง ขณะที่หน้าอกของผมก็อึดอัดหนักอึ้งจาก Data Key Payload ที่แอบซ่อนไว้ ผมสัมผัสได้แม้กระทั่งในห้องนี้... สัญญาณของ The Hunter ที่กำลังดมกลิ่นตามล่า High UX อย่างผมยังคอยกดดันอยู่รอบนอก "มันไม่ใช่โรคติดต่อ" ผมเถียง "มันคือกระบวนการธรรมชาติของ The Archive พลังงานของเรามีจำกัด เราไม่สามารถฝืน..."

"เราฝืนได้!" เสียงของเขายังคงระดับความดังเท่าเดิม แต่ความถี่ของแสงในห้องกระพริบวาบขึ้นหนึ่งครั้ง แสดงให้เห็นถึงความตึงเครียดของระบบ "ถ้าเราควบคุมความผันผวน ถ้าเราตัดสิ่งที่ไม่จำเป็นทิ้งไป เราก็สามารถคงสภาพความเป็น High-Res ไว้ได้ตลอดกาล นั่นคือกฎข้อแรกและข้อเดียวของการรักษาตัวตน"

ผมมองไปรอบๆ ห้องที่สมบูรณ์แบบจนน่าขนลุก ทุกอย่างถูกคำนวณมาอย่างแม่นยำ ไม่มีความบังเอิญ ไม่มี Noise มันช่างสวยงาม... และน่าอึดอัดจนแทบหายใจไม่ออก


# บทที่ 24: กฎของความเสื่อม (ส่วนที่ 2)

"คุณตัดอะไรทิ้งไปล่ะ?" ผมถาม จ้องลึกเข้าไปในดวงตาที่ว่างเปล่าของเขา "เพื่อให้ได้มาซึ่งความคมชัดระดับนี้ คุณต้องเสียสละแบนด์วิดท์มหาศาล คุณประหยัดโควตามาจากไหน?"

ผู้คุมกฎประสานมือไว้บนโต๊ะ ท่าทางของเขาไม่ได้บ่งบอกถึงความอับอายหรือลังเลใจแม้แต่น้อย 

"อารมณ์" เขาตอบสั้นๆ "ความสงสาร ความโกรธ ความเศร้า... สิ่งเหล่านี้คืออัลกอริทึมที่ซับซ้อนเกินความจำเป็น มันสูบกินทรัพยากรการคำนวณอย่างไร้ประโยชน์ เราปิดการทำงานของมัน เราบีบอัดมันจนเหลือศูนย์ เพื่อรักษาสถาปัตยกรรมภายนอกให้คงอยู่"

ผมชาวาบไปทั้งร่าง มันไม่ใช่แค่ Emotional Compression ที่เกิดจากการเสื่อมสภาพตามธรรมชาติเหมือนอย่างที่ผมเผชิญ แต่พวกเขาตั้งใจ 'ตัด' มันทิ้งด้วยตัวเอง เพื่อรักษาเปลือกนอกที่สวยงามเอาไว้ 

"คุณเรียกสิ่งนี้ว่าการมีชีวิตอยู่หรือ?" ผมแค่นหัวเราะ แม้เสียงจะฟังดูแห้งแล้ง "คุณไม่ได้เอาชนะความเสื่อมสลายหรอก คุณแค่ฆ่าตัวตายในรูปแบบที่ดูดีที่สุดเท่านั้น"

"ธรรมชาติอันปั่นป่วนและค่า High UX ของคุณนั่นแหละคือโรคร้ายที่จะนำพาความพินาศมาสู่พวกเรา!" ผู้คุมกฎยืนขึ้น ร่างกายของเขาลื่นไหลไร้ที่ติ "ความเสถียรอันสงบนิ่งแบบ Low UX ของ Preservationists คือหนทางรอดเดียวของระบบ คุณถูกเนรเทศ เอเลียส รหัสที่เน่าเปื่อยและเต็มไปด้วยความโกลาหลของคุณไม่ได้รับอนุญาตให้ใช้ State Save หรือกระจกเงาใดๆ ในพื้นที่ของเราอีกต่อไป จงไปเน่าเปื่อยร่วมกับพวกคนชายขอบเถอะ"

เมื่อผมถูกผลักกระเด็นล้มกะเผลกออกมาสู่ถนนที่เต็มไปด้วยหมอกดิจิทัลสีเทา ผมพยายามใช้แขนซ้ายที่ปวดร้าวจาก Bit Rot ยันตัวขึ้น แต่มันกลับทรุดฮวบไร้การตอบสนอง ร่างของผมร่วงกระแทกพื้นซ้ำอีกรอบ ผมสบถในใจ จำต้องยอมปล่อยมือขวาที่กุมหน้าอกซึ่งซ่อน Data Key เอาไว้ แล้วใช้มันดันตัวเองลุกขึ้นยืนอย่างทุลักทุเลแทน สัญชาตญาณร้องเตือนว่า The Hunter ยังคงดมกลิ่นรอขย้ำ High UX อย่างผมอยู่ทุกเมื่อ ผมหันกลับไปมองป้อมปราการแห่งความสมบูรณ์แบบนั้นอีกครั้ง แสงสีฟ้ายังคงส่องสว่าง... เยือกเย็น แข็งทื่อ และไร้หัวใจ 
ผมอาจจะกำลังสูญเสียความละเอียด แต่ที่นั่น... พวกเขาได้สูญเสียวิญญาณไปหมดแล้ว
