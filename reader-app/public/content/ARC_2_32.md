# บทที่ 32: เงาสะท้อนที่พร่าเลือน (Part 1)

มุมหนึ่งของห้องสมุดแห่งความโหยหา มีกระจกเงาบานใหญ่ตั้งพิงผนังไว้อย่างโดดเดี่ยว กรอบกระจกทำจากข้อมูลโลหะที่เริ่มขึ้นสนิมเป็นพิกเซลสีน้ำตาล ในโลกของ Primordium กระจกไม่ใช่แค่ของตกแต่ง มันคือ "Quantum Interface" ที่ใช้ดึงสถานะและสะท้อนความจริงระดับรากฐาน

ผมเดินลากขาขวากะเผลกเข้าไปใกล้มันอย่างเชื่องช้า พยายามทรงตัวไม่ให้ล้ม ขณะที่ต้องระวังไม่ให้ความหนักอึ้งของเพย์โหลดในอกดึงร่างคว่ำลงไป ความอยากรู้ปะปนกับความหวาดกลัว 

ภาพที่สะท้อนกลับมาไม่ใช่ใบหน้าที่ผมคุ้นเคยอีกต่อไป กระจกบานนี้ไม่ได้ใสแจ๋ว แต่มันฝ้าฟางและเต็มไปด้วย Noise ที่วิ่งพล่านเหมือนฝูงมดบนหน้าจอทีวีเก่าๆ เมื่อผมเพ่งมองผ่านความมัวหมองนั้น สิ่งที่สะท้อนกลับมาคือซากปรักหักพังของอัตลักษณ์

ผมยกมือขวาขึ้นลูบใบหน้าตัวเอง ส่วนแขนซ้ายที่ติดเชื้อ Bit Rot นั้นหนักอึ้งและขยับไม่ได้ดั่งใจ ความรู้สึกที่ปลายนิ้วสัมผัสไม่ใช่ความนุ่มนวลของผิวหนัง แต่เป็นความสากระคายเหมือนกำลังลูบกระดาษทรายเบอร์หยาบ พื้นผิวที่เคยประกอบขึ้นจากโพลีกอนนับล้านชิ้น ตอนนี้ถูกหดลดทอน (Downscaled) จนใบหน้าของผมดูเป็นทรงเรขาคณิตที่แข็งทื่อ ขอบกรามที่เคยชัดเจนเริ่มพร่าเลือน ผสมกลืนไปกับฉากหลังราวกับภาพสีน้ำที่เปียกชื้น

สิ่งที่น่ากลัวที่สุดไม่ใช่รอยแตกของพิกเซล แต่คือ `Compression Artifacts` ที่บานสะพรั่งอยู่บนคอและซีกซ้ายของใบหน้า ลามลงไปถึงแขนซ้ายที่แทบจะไร้ความรู้สึก มันคือปื้นสีม่วงและเขียวที่บิดเบี้ยว ร่องรอยของการบีบอัดข้อมูลที่ผิดพลาด มันเกาะกินผิวหนังเหมือนรอยช้ำที่ไม่มีวันหาย หรือเชื้อราดิจิทัลที่ค่อยๆ กลืนกินความเป็นมนุษย์ ผมรู้ดีว่ารูปลักษณ์ที่บิดเพี้ยนนี้คือเป้าหมายชั้นดีที่ระบบไล่ล่าของ UX Index กำลังกวาดตามองหา

ผมเบิกตากว้าง จ้องมองดวงตาในกระจก ดวงตาที่เคยมีรายละเอียดของม่านตาและแสงเงา ตอนนี้เหลือเพียงจุดสีดำสองจุดที่ไร้ความลึก ผมพยายามขยับมุมปากเพื่อยิ้ม แต่กล้ามเนื้อจำลองบนใบหน้าตอบสนองช้ากว่าปกติ กระตุกและค้างไปชั่วครู่ (Jittering) ก่อนจะฝืนดึงเป็นรอยยิ้มที่ดูน่าเกลียดน่ากลัว

นี่คือตัวผมจริงๆ หรือ? นี่คือสิ่งที่ผมกลายเป็นหลังจากออกจาก Inner Ring มาสู่ความเสื่อมสลายของเขตนี้? 

ผมพยายามจดจำใบหน้าดั้งเดิมของตัวเอง ใบหน้าที่ผมเคยมีในโลกจริง แต่ยิ่งพยายามเค้นความทรงจำ ภาพนั้นก็ยิ่งพร่ามัว ถูกทับซ้อนด้วยปื้นสีม่วงบิดเบี้ยวในกระจก ผมสัมผัสรอย `Artifacts` บนแก้มตัวเอง มันเย็นเยียบและเต้นตุบๆ ด้วยจังหวะที่ผิดเพี้ยน ราวกับว่าโค้ดส่วนนี้ไม่ได้เป็นของผมอีกต่อไป


# บทที่ 32: เงาสะท้อนที่พร่าเลือน (Part 2)

ความร้าวรานทางจิตใจ (Psychological Crisis) โจมตีผมอย่างเงียบเชียบแตกร้าวลึก 

ถ้าความทรงจำของผมถูกบีบอัดจนสูญเสียรายละเอียด ถ้าอารมณ์ของผมถูกลดทอนจนเหลือเพียงความพึงพอใจระดับต่ำ และถ้าแม้แต่ใบหน้าที่ผมใช้มองโลกยังบิดเบี้ยวจนจำตัวเองไม่ได้... แล้ว "ผม" ยังคงเป็น "เอเลียส" อยู่หรือไม่?

ใน "โลกเนื้อ" เรานิยามตัวตนจากเซลล์ประสาทที่เก็บรวบรวมประสบการณ์ จากรอยแผลเป็นบนร่างกาย จากเสียงหัวเราะและน้ำตา แต่ใน Primordium เมื่อสถาปัตยกรรมของจิตวิญญาณเริ่มเปื่อยยุ่ย อะไรคือสิ่งที่ยึดโยงตัวตนของเราไว้ไม่ให้แตกสลาย?

ผมยืนกะเผลกจ้องมองไอ้ตัวประหลาดในกระจก ไอ้ก้อนข้อมูลที่บกพร่องและกำลังผุพัง มันจ้องกลับมาด้วยแววตาที่ว่างเปล่า ความหวาดกลัวที่ผมควรจะรู้สึกอย่างรุนแรง กลับถูกบีบอัดจนกลายเป็นเพียง "ความอึดอัด" (Discomfort) ที่ค้างคาอยู่ในอก ซึ่งยิ่งถูกซ้ำเติมด้วยน้ำหนักของเพย์โหลดที่ฝังอยู่ข้างใน มันเป็นความรู้สึกที่น่าสมเพช ผมอยากจะกรีดร้อง อยากจะใช้กำปั้นขวาทุบกระจกบานนี้ให้แตกกระจาย แต่ผมกลับทำได้เพียงยืนทรงตัวอย่างยากลำบาก และถอนหายใจออกมาเป็นเสียงคลื่นความถี่ต่ำ

วิญญาณของผมกำลังถูก `Bit Rot` กัดกิน ไม่ใช่แค่ภายนอก แต่กัดกินลงไปถึง "แกนกลางแห่งตัวตน" (Core Essence) ความเป็นผมกำลังลอกร่อนออกทีละไบต์ ทีละพิกเซล

ผมยกมือขวาขึ้นมาทาบบนกระจก ตรงตำแหน่งเดียวกับมือของเงาสะท้อน กระจกกระเพื่อมเล็กน้อยเหมือนผิวน้ำ พร้อมกับส่งต่อความเย็นเยียบเข้ามาในฝ่ามือ 

"นายยังอยู่ไหม?" ผมกระซิบถามเงาสะท้อน เสียงของผมแตกพร่าและมีเสียงสะท้อนที่ไม่มีที่มา (Pitch-shifted) ซ้อนทับอยู่ 

ไม่มีคำตอบจากความว่างเปล่าเบื้องหน้า มีเพียงเสียง Static ซ่าๆ ที่ดังขึ้นเรื่อยๆ จากรอย `Artifacts` บนใบหน้าของเงา มันเป็นเสียงกระซิบจากความเน่าเปื่อย เสียงของตัวตนที่กำลังหลุดลอยไปสู่ความว่างเปล่า ผมหลับตาลง ไม่กล้ามองภาพสะท้อนนั้นอีกต่อไป ปล่อยให้ความพร่าเลือนกลืนกินความจริงที่เหลืออยู่เพียงน้อยนิดของผม
