# บทที่ 32: เงาสะท้อนที่พร่าเลือน (Part 1)

มุมหนึ่งของห้องสมุด มีกระจกเงาบานใหญ่ตั้งพิงผนัง กรอบโลหะข้อมูลขึ้นสนิมเป็นพิกเซลสีน้ำตาล กระจกบานนี้คือ Quantum Interface ที่สะท้อนสถานะรากฐานของโค้ด

ผมลากขากะเผลกเข้าไปใกล้มัน ประคองน้ำหนักของ Payload ในอกไม่ให้ดึงร่างคว่ำลง ความหวาดกลัวกระตุ้นให้อัตราการประมวลผลทำงานหนัก ร่างกายของผมลดทอนรายละเอียดลงอีกครั้งเพื่อเปิดทางให้กับความตระหนก 

ภาพที่สะท้อนกลับมาไม่ใช่ตัวผม กระจกฝ้าฟางเต็มไปด้วย Noise ที่วิ่งพล่าน สิ่งที่ยืนอยู่ตรงนั้นคือซากปรักหักพัง

ผมยกมือขวาลูบใบหน้า สัมผัสนั้นสากระคายเหมือนกระดาษทราย พื้นผิวที่เคยละเอียดถูก Downscaled จนกลายเป็นทรงเรขาคณิตแข็งทื่อ ขอบกรามกลืนหายไปกับฉากหลัง

สิ่งที่น่าสยดสยองที่สุดคือ Compression Artifacts ปื้นสีม่วงและเขียวที่บิดเบี้ยวบานสะพรั่งอยู่บนลำคอ ลามลงสู่แขนซ้ายที่ไร้ความรู้สึก มันคือรอยช้ำของการบีบอัดข้อมูลที่ผิดพลาด เชื้อราดิจิทัลที่กำลังกลืนกินความเป็นมนุษย์ รูปลักษณ์นี้คือเป้าหมายชั้นดีสำหรับ UX Index

ผมเบิกตากว้างจ้องมองดวงตาในกระจก มันเหลือเพียงจุดสีดำไร้ความลึก กล้ามเนื้อจำลองกระตุกค้าง (Jittering) เมื่อผมพยายามฝืนยิ้ม สร้างเป็นรอยยิ้มที่บิดเบี้ยวและน่าสะอิดสะเอียน

# บทที่ 32: เงาสะท้อนที่พร่าเลือน (Part 2)

ความร้าวรานโจมตีผมอย่างเงียบเชียบ

เมื่ออารมณ์ดิ่งลึกถึงขีดสุด การดึงแบนด์วิดท์เพื่อประมวลผลความสิ้นหวังทำให้รอยฉีกขาด (Tearing) แผ่ขยายไปทั่วร่าง เงาสะท้อนของผมกระตุก สลับไปมาระหว่างภาพความละเอียดต่ำกับโครงสร้างลวดหนามโพลีกอน

ถ้าความทรงจำถูกบีบอัด ถ้าความรู้สึกเหลือเพียงเศษเสี้ยว และถ้าใบหน้าบิดเบี้ยวจนจำตัวเองไม่ได้... ผมยังคงเป็น "เอเลียส" อยู่อีกหรือ?

ผมยืนจ้องมองก้อนข้อมูลที่บกพร่องในกระจก แววตาว่างเปล่าของมันไร้ซึ่งชีวิต ความกลัวที่ควรจะพรั่งพรูถูกบีบอัดจนกลายเป็นเพียงความอึดอัดแน่นในลำคอ น้ำหนักของเพย์โหลดในอกตอกย้ำความน่าสมเพช ผมอยากใช้กำปั้นขวาทุบกระจกให้แหลกละเอียด ทว่าทำได้เพียงถอนหายใจออกมาเป็นคลื่นความถี่ต่ำ

Bit Rot กำลังกัดกินลึกลงไปถึงแกนกลางแห่งตัวตน

ผมยกมือขวาทาบบนกระจก ตรงตำแหน่งเดียวกับเงาสะท้อน ผิวกระจกกระเพื่อม ส่งผ่านความเย็นเยียบเข้าสู่รอยแผลเป็น Dead Pixels

"นายยังอยู่ไหม..." ผมกระซิบ เสียงแตกพร่าบิดเบี้ยว (Pitch-shifted)

ไม่มีคำตอบ มีเพียงเสียง Static ซ่าๆ ดังแทรกขึ้นมาจาก Compression Artifacts บนใบหน้า ผมหลับตาลง ยอมแพ้ต่อภาพสะท้อน ปล่อยให้ความพร่าเลือนกลืนกินเสี้ยวความจริงสุดท้ายที่หลงเหลืออยู่
