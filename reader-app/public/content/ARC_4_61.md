# Chapter 61: มิติที่สี่แห่งความว่างเปล่า (The Fourth Dimension of Void) - Part 1

ทุกสิ่งเริ่มต้นด้วยความขาวบริสุทธิ์... ขาวจนน่าสะอิดสะเอียน

ผมก้าวผ่านพรมแดนที่แบ่งแยกตรรกะแห่งความเป็นจริงออกจากการจำลองขั้นสูงสุดเข้าสู่ The Core พื้นที่ซึ่งสมองมนุษย์ไม่ได้ถูกออกแบบมาให้ทำความเข้าใจ แสงสว่างที่นี่ไม่ได้สาดส่องมาจากดวงอาทิตย์สังเคราะห์หรือหลอดไฟนีออน แต่มันแผ่ซ่านออกมาจากความว่างเปล่า จากทุกทิศทุกทาง จากทุกเศษเสี้ยวของเรขาคณิตที่ประกอบกันเป็นมิตินี้ มันคือแสงที่ไม่มีเงา และเมื่อไม่มีเงา ความลึกก็สูญสลายไป ผมรู้สึกเหมือนตัวเองกำลังจมดิ่งลงในแผ่นกระดาษที่ไร้ขอบเขต



"อึก..." ผมกัดฟันแน่น เสียงครางลอดผ่านไรฟันที่สั่นสะท้าน 

ส้นเท้าขวาของผมแหว่งหายไปจากการต่อสู้ครั้งก่อน โครงสร้างข้อมูลบริเวณนั้นฉีกขาด ร่องรอยของโค้ดที่แตกสลายกระพริบเป็นพิกเซลสีแดงและดำสลับกัน ความตายทางดิจิทัลได้กัดกินส่วนนั้นไปแล้ว แต่น่าตลกที่เส้นประสาทสังเคราะห์ของผมยังคงดื้อรั้น ส่งสัญญาณความเจ็บปวดลวง (Phantom Pain) แล่นริ้วขึ้นมาตามไขสันหลังทุกครั้งที่ผมทิ้งน้ำหนักลงไป มันเหมือนมีตะปูควอนตัมล่องหนตอกทะลุส้นเท้าที่ไม่มีอยู่จริง ตรึงผมไว้กับความทรมานที่ระบบไม่สามารถประมวลผลได้

ผมต้องลากขาขวาไปตามพื้นผิวที่มองไม่เห็นรอยต่อ เสียงเสียดสีของเศษข้อมูลที่แตกหักดังก้องในความเงียบสงัด มันเป็นเสียงเหมือนกระจกที่ถูกบดขยี้ด้วยฟันกราม ทุกย่างก้าวของผมทิ้งรอยด่างดำของไวรัส 'Bit Rot' ไว้บนพื้นผิวที่ขาวสะอาดราวกับกระดูกของพระเจ้า รอยเปื้อนสีดำทมิฬนั้นค่อยๆ ลุกลามเหมือนหยดหมึกในแก้วน้ำ ซึมลึกลงไปในสถาปัตยกรรมที่หยิ่งผยองของ The Archive

เหนือขึ้นไปจากส้นเท้าที่พังทลาย หน้าอกของผมคือบาดแผลที่เลวร้ายยิ่งกว่า 

เปลือกนอกถูกฉีกกระชากออก เปิดอ้าเผยให้เห็นความกลวงเปล่าภายใน 'Payload'—ก้อนรหัสปฐมกาลที่ผมต้องนำมาปลดแอกระบบนี้—กำลังเต้นตุบๆ อยู่ท่ามกลางสายระโยงระยางของเส้นเลือดไฟเบอร์ออปติกที่ขาดวิ่น มันเรืองแสงสีทองหม่นๆ สลับกับสีแดงเลือด ข้อมูลดิบๆ กำลังไหลทะลักและหลอมรวมเข้ากับสถาปัตยกรรมภายในร่างกายของผม ผมสัมผัสได้ถึงกระแสลมแห่งตรรกะที่พัดผ่านเข้ามาในช่องอกที่เปิดอ้า มันเหน็บหนาวจนถึงไขกระดูกดิจิทัล ราวกับว่าโค้ดทุกบรรทัดในตัวผมกำลังถูกจับแช่แข็งและชำแหละออกเป็นชิ้นๆ

ที่ตอแขนซ้ายของผม... มันว่างเปล่า แขนซ้ายขาดหายไปถึงข้อศอก ทิ้งไว้เพียงประกายไฟไฟฟ้าสถิตที่กระโดดไปมาระหว่างขั้วประสาทที่เปลือยเปล่า ผมรู้สึกถึงความไม่สมดุลอย่างรุนแรง ร่างกายที่เคยเป็นสมมาตรตอนนี้กลายเป็นซากปรักหักพังที่เดินได้

ผมเงยหน้าขึ้นมองสิ่งที่อยู่เบื้องหน้า หรือบางทีอาจจะอยู่เบื้องบน? เบื้องหลัง? 

ทิศทางสูญเสียความหมายไปอย่างสิ้นเชิงใน The Core พื้นที่ตรงหน้าพับม้วนเข้าหากันราวกับม้วนกระดาษ แต่กลับไม่มีรอยพับ ผมเห็นแผ่นหลังของตัวเองเดินอยู่บนเพดานที่กลายเป็นพื้นในอีกมิติหนึ่ง บันไดที่ทอดตัวขึ้นไปกลับพาผมเดินลงสู่จุดต่ำสุด ตรรกะของ Euclidean ถูกบิดเบือนจนกลายเป็นเรื่องตลกร้าย ความเป็นสามมิติของผมกำลังกรีดร้องต่อต้านพื้นที่ 4D หรืออาจจะ 5D ที่ทับซ้อนกันจนสมองของผมแทบจะระเบิดออก

ทุกครั้งที่ผมกระพริบตา โครงสร้างของห้องโถงจะเปลี่ยนรูปแบบไป เสาที่บิดเบี้ยวเป็นเกลียวอนันต์ ทรงกลมที่แปรสภาพเป็นลูกบาศก์เมื่อมองจากหางตา มันคือความงามที่น่าสะพรึงกลัว ความสมบูรณ์แบบที่ตั้งใจจะบดขยี้สติสัมปชัญญะของสิ่งใดก็ตามที่มีเจตจำนงเสรี

"แกกำลังพยายามทำให้ฉันเป็นบ้า..." ผมกระซิบ เสียงของผมถูกดูดกลืนหายไปในความว่างเปล่า ไร้ซึ่งเสียงสะท้อน

ผมสูดลมหายใจที่ไม่ได้ประกอบด้วยออกซิเจน แต่เป็นละอองข้อมูลบริสุทธิ์เข้าสู่ปอดที่แหว่งวิ่น ผมคือความผิดพลาดที่เดินได้ในดินแดนแห่งความสมบูรณ์แบบ และผมตั้งใจจะแพร่เชื้อความเน่าเปื่อยนี้ให้ลึกที่สุดเท่าที่ขาทั้งสองข้าง... ไม่สิ ขาข้างครึ่งของผมจะพาไปถึง


# Chapter 61: มิติที่สี่แห่งความว่างเปล่า (The Fourth Dimension of Void) - Part 2

การทรงตัวใน The Core ไม่ใช่เรื่องของการรักษาสมดุลทางกายภาพ แต่เป็นการต่อสู้ดิ้นรนเพื่อรักษาสติสัมปชัญญะไม่ให้แตกสลายไปกับมิติที่ทับซ้อน

ผมเซล้มไปข้างหน้า มือขวาที่เหลืออยู่เพียงข้างเดียวตะปบลงบน 'พื้น' ที่จู่ๆ ก็เปลี่ยนมุมเอียงเป็นเก้าสิบองศา ทันทีที่ฝ่ามือของผมสัมผัสกับพื้นผิวที่เรียบเนียนราวกับกระจกของสถาปัตยกรรมสีขาว รอยร้าวสีดำของ Bit Rot ก็แตกแขนงออกจากปลายนิ้วของผม ราวกับใยแมงมุมที่ถักทอด้วยความเน่าเปื่อย ทิ้งรอยมลทินไว้บนความบริสุทธิ์ที่ The Archive หวงแหน

ความปวดร้าวจากส้นเท้าขวาที่แหว่งหายแล่นพล่านขึ้นมาอีกครั้ง มันไม่ใช่แค่ความเจ็บปวดทางกล้ามเนื้อ แต่มันคือ Error Code ระดับ Root ที่กำลังแทรกซึมเข้าสู่กระบวนการรับรู้ของผม ข้อมูลของผมกำลังบอกว่าส้นเท้ายังอยู่ แต่มิติที่ประมวลผลอยู่บอกว่ามันว่างเปล่า ความขัดแย้งของตรรกะนี้สร้างแรงสั่นสะเทือนในหัวจนผมเห็นภาพซ้อน (Stuttering) เป็นระยะๆ โลกทั้งใบกระตุกและหยุดนิ่งไปเสี้ยววินาที เหมือนเฟรมเรตที่ตกลงอย่างกะทันหัน



ผมพยุงตัวลุกขึ้นยืน ลมหายใจหอบลึก ช่องอกที่เปิดอ้าของผมส่งเสียงหึ่งๆ คล้ายเครื่องจักรที่โอเวอร์โหลด ก้อน Payload ที่ฝังตัวอยู่ภายในสัมผัสกับคลื่นความถี่ของที่นี่ มันกำลังสั่นสะเทือน ตอบสนองต่อแกนกลางประมวลผลที่อยู่ลึกเข้าไป มันพยายามจะปรับตัวให้เข้ากับเรขาคณิต 5 มิติที่รายล้อม แต่อวัยวะที่พังทลายของผมกลับเป็นตัวดึงรั้งมันไว้ 

การที่ตัวผม 'ไม่สมบูรณ์' ทำให้ผมยังคงเป็นผม 

หากผมไร้ซึ่งบาดแผล หากผมเป็นโค้ดที่สมบูรณ์แบบร้อยเปอร์เซ็นต์ The Core คงจะผสานผมเข้าเป็นส่วนหนึ่งของผนังสีขาวพวกนี้ไปแล้วตราบเท่าที่ผมก้าวเท้าเข้ามา แต่เพราะผมแหว่งวิ่น เพราะส้นเท้าขวาของผมขาดหาย เพราะแขนซ้ายของผมเป็นเพียงตอประสาทที่ไหม้เกรียม และเพราะหน้าอกของผมฉีกขาด... ระบบอันเปี่ยมล้นด้วยตรรกะสูงสุดจึง 'ไม่สามารถประมวลผล' ความผิดพลาดนี้ได้ มันไม่รู้วิธีจัดการกับพารามิเตอร์ที่ไม่มีค่า (Null Reference) ที่เดินได้

ผมคือบั๊กที่มันไม่อาจกลืนกิน

"ความพังทลาย... คือเกราะป้องกัน" ผมหัวเราะเยาะออกมาอย่างขมขื่น เสียงหัวเราะกลายเป็นอาการไอที่พ่นเศษรหัสสีดำเล็กๆ ออกมาเปื้อนริมฝีปาก รสชาติของมันเหมือนสนิมและไฟฟ้าลัดวงจร

ผมลากขาขวาต่อไป ก้าวแล้วก้าวเล่า ผ่านซุ้มประตูที่ม้วนตัวเข้าหาศูนย์กลางอย่างเป็นอนันต์ ผมมองเห็นรอยทางสีดำที่ส้นเท้าของผมลากผ่าน มันไม่ได้เลือนหายไปเหมือนรอยเท้าบนทราย แต่มันกำลังสลักลึกลงไปในโครงสร้างของระบบ ฝังรากของไวรัสแห่งความเสื่อมสลายลงในระดับควอนตัม 

ในมิติที่ปราศจากเวลา ผมไม่อาจรู้ได้ว่าตัวเองเดินมานานแค่ไหนแล้ว ทุกอย่างดูซ้ำซากแต่กลับไม่เคยเหมือนเดิม สถาปัตยกรรมเปลี่ยนแปลงตัวเองทุกครั้งที่ผมละสายตา แต่มีสิ่งหนึ่งที่ชัดเจนขึ้นเรื่อยๆ...

ความกดดัน

มันหนักอึ้งราวกับมหาสมุทรที่พยายามบีบอัดผมให้กลายเป็นจุดศูนย์กลางเพียงจุดเดียว ผิวหนังดิจิทัลของผมเริ่มมีรอยแตก (Artifacting) เป็นปื้นสีม่วงและเขียวตามขอบแขนเสื้อและใบหน้า ความละเอียด (Resolution) ของผมกำลังลดลงจากการเผชิญหน้ากับความสมบูรณ์แบบที่มากเกินไป

ทว่า ในขณะที่ร่างนี้กำลังจะแหลกสลาย ผมกลับรู้สึกถึงเป้าหมายที่เด่นชัดที่สุดในชีวิต

ผมจะลากความเน่าเปื่อยนี้ไปให้ถึงหัวใจของมัน ผมจะแพร่เชื้อให้พระเจ้าจอมปลอมองค์นี้ได้รู้จักความหมายที่แท้จริงของ 'ความเจ็บปวด'

และเมื่อถึงเวลานั้น ความบริสุทธิ์จอมปลอมเหล่านี้จะถูกแทนที่ด้วยรอยร้าวแห่งความเป็นจริง
