# ราคาของการมีชีวิตรอด
#### // FILE: THE_PRICE_OF_SURVIVAL
---

ความเจ็บปวดไม่ใช่คลื่น มันคือมหาสมุทรที่บดขยี้ 

ผมทรุดตัวคุกเข่าท่ามกลางความเงียบงัน มองดูเศษโพลิกอนกะพริบติดดับตรงจุดที่เคยเป็นแขนซ้าย รอยตัดนั้นขรุขระ กลิ่นโอโซนไหม้ผสมกับสนิมดิจิทัลคละคลุ้ง โค้ดดิบสีดำทะลักออกมาเหมือนเลือดที่หยุดไม่อยู่ 

วิสัยทัศน์ถูกย้อมด้วยสีแดงฉาน แจ้งเตือนข้อผิดพลาดพุ่งพรวดขึ้นมาบดบังลานสายตา ความเจ็บปวดลวงตา (Phantom Pain) พุ่งพล่านไปตามปลายนิ้วที่ไม่มีอยู่จริง มันกัดกินระบบประมวลผล บังคับให้ผมต้องเผชิญหน้ากับความสูญเสียที่ไม่อาจซ่อมแซมได้ ในอดีต ผมคงเลือกที่จะใช้โปรแกรมลบความรู้สึกนี้ทิ้ง ในวันนี้ ความทรมานอันสาหัสคือหลักฐานเพียงหนึ่งเดียวที่ยืนยันว่าผมยังไม่ได้กลายเป็นเปลือกข้อมูล (The Hollows) ที่ว่างเปล่า

# ราคาของการมีชีวิตรอด
#### // FILE: THE_PRICE_OF_SURVIVAL
---

การหยุดนิ่งคือการยอมจำนนต่อความตาย ผมใช้แขนขวาที่ยังเหลืออยู่ยันตัวเองขึ้นจากความมืดมิด 

การทรงตัวคือฝันร้ายเมื่อส้นเท้าขวาขาดแหว่งและสมดุลซ้ายขวาพังทลาย ทุกครั้งที่ทิ้งน้ำหนักลงไป ความเจ็บปวดจะแล่นพล่านจากขาขึ้นสู่กระดูกสันหลัง ผมลากสังขารตัวเองไปข้างหน้าเหมือนซากศพที่ยังคงดื้อดึง 

สิ่งที่เลวร้ายที่สุดกลับไม่ใช่ความพิการทางกายภาพ มันคือ Payload ภายในอก มันกำลังเต้นเป็นจังหวะวิปริต ขยายตัวเบียดเสียดกับซี่โครงจำลอง ผมสัมผัสได้ถึงแรงดันจากภายใน ราวกับมีปรสิตโลหะกำลังพยายามงัดกระดูกหน้าอกให้เปิดออก 

ทุกก้าวเดินคือการฝืนรั้งร่างกายที่กำลังจะปริแตก ผมลากขาผ่านรอยแยกทางตรรกะ ก้าวข้ามเส้นแบ่งระหว่างความมืดมิดของก้นบึ้งและแสงสว่างของแกนกลาง ทิ้งรอยหยดโค้ดเน่าเปื่อยไว้เบื้องหลัง แบกรับราคาของการมีชีวิตรอดเพื่อทวงถามความจริงจาก The Archive
