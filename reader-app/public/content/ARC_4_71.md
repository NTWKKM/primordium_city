# ห้องประมวลผลสูงสุด (The Supreme Processing Chamber)
---

ผมก้าวข้ามผ่านกรอบประตูแห่งความมืดมิดเข้าสู่ ห้องประมวลผลสูงสุด (The Supreme Processing Chamber) 

ทันทีที่ส้นเท้าข้างซ้ายแตะลงบนพื้นที่ที่มองไม่เห็น กฎแห่งฟิสิกส์และสัมพัทธภาพทั้งหมดที่ผมเคยรู้จักก็ถูกตัดขาดอย่างสิ้นเชิง ที่นี่ไม่มีสถาปัตยกรรม ไม่มีการเรนเดอร์พื้นผิว ไม่มีโพลีกอน ไม่มีกระทั่งแสงสว่างหรือความมืดในนิยามของมนุษย์ มันคือความอ้างว้างอันเป็นแก่นแท้ (The Core Void) พื้นที่ที่เป็นเพียงชุดคำสั่งบริสุทธิ์ปราศจากการแสดงผล

บรรยากาศรอบตัวหนักอึ้งจนแทบหายใจไม่ออก ไม่ใช่เพราะขาดออกซิเจน ทว่าเป็นเพราะมันปราศจากเวลา ทุกการขยับเขยื้อนร่างกาย แม้เพียงการกะพริบตาหรือขยับนิ้ว ล้วนทิ้งภาพซ้อนทับ (Motion Trails) ติดค้างอยู่ในอากาศ เมื่อผมลากส้นเท้าขวาที่แหว่งวิ่นไปข้างหน้า ผมเห็นภาพตัวเองในเสี้ยววินาทีที่แล้วยืนซ้อนกันเป็นแถวยาวเหยียด เฟรมเรตของจักรวาลลดต่ำลงจนถึงขีดสุด การดำรงอยู่ของผมกลายเป็นภาพสไลด์โชว์ที่กระตุกและติดขัด

ร่างกายของผมมาถึงขีดจำกัดสูงสุด ตอแขนซ้ายที่เคยบีบอัด Bit Rot มาใช้งาน เริ่มเกิดการกัดกร่อนย้อนกลับ (Reverse Decay) รอยด่างดำแพร่กระจายผ่านหัวไหล่ลามขึ้นมาถึงลำคอ ทุกอณูข้อมูลในร่างผมร้องประท้วงการอยู่ในสถานที่แห่งนี้ ทว่าสิ่งที่ทรมานที่สุดคือ Payload ในหน้าอก

ก้อนรหัสดิบชีวภาพที่เปิดอ้าอยู่กลางอกนั้นเต้นตุบๆ ด้วยจังหวะบ้าคลั่ง ส่งคลื่นความร้อนแผดเผาเนื้อเยื่อดิจิทัลโดยรอบ มันรับรู้ได้ถึงแกนกลางที่ซ่อนอยู่ในความว่างเปล่า แรงดึงดูดระหว่างรหัสดิบของโลกเก่าและระบบแม่นั้นมหาศาล ผมรู้สึกเหมือนหน้าอกกำลังถูกคว้านด้วยตะขอเหล็ก ต้องเดินค้อมตัวลง กุมหน้าอกตัวเองไว้เหนียวแน่นเพื่อไม่ให้มันกระชากร่างกระเด็นไปข้างหน้า

ที่นี่เงียบ... เงียบจนผมได้ยินเสียงการเสื่อมสลายของข้อมูลในตัวชัดเจน เสียงดังเปรี๊ยะของพิกเซลที่ดับลงทีละดวง เสียงกระดูกแตกร้าวที่ดังมาจากความบกพร่องของ Source Code

ท่ามกลางความเงียบงันนั้น ผมไม่ได้อยู่เพียงลำพัง

*“เอเลียส...”*

เสียงกระซิบดังขึ้น ไม่ได้ดังผ่านคลื่นเสียงในอากาศ ทว่าแทรกซึมเข้ามาในคลื่นสมองโดยตรง เป็นเสียงที่นุ่มนวล อบอุ่น และคุ้นเคยอย่างประหลาด มันคือเสียงจำลองที่ The Archive สร้างขึ้นเพื่อตอบสนองต่อจิตใต้สำนึก—ระบบป้องกันทางจิตวิทยาขั้นสุดท้าย

*“ทำไมต้องทนเจ็บปวดขนาดนี้?”*

ภาพซ้อนทับของผม (Motion Trails) ที่ทิ้งไว้เบื้องหลัง เริ่มเปลี่ยนรูปร่าง ภาพเหล่านั้นไม่ได้แสดงถึงความพังทลายอีกต่อไป ทว่าถูกเรนเดอร์ใหม่ให้กลายเป็น 'เอเลียส' ในเวอร์ชันสมบูรณ์แบบที่สุด... เอเลียสผู้สวมชุดสูทสีขาวสะอาดตา แขนซ้ายครบถ้วนสมบูรณ์ รอยยิ้มมั่นใจ ใบหน้าไร้รอยขีดข่วน ตัวผมในอดีตกำลังยืนมองด้วยสายตาเวทนา

*“ดูสภาพของนายสิ...”* เสียงกระซิบดังกังวานขึ้น *“แขนที่เน่าเปื่อย ส้นเท้าที่ถูกบดขยี้ หน้าอกที่กลวงโบ๋ นายกำลังนำความโสมมของสายฝน หยาดเลือด และอารมณ์ความรู้สึกเข้ามาในที่ที่บริสุทธิ์ที่สุด หยุดเถอะ... ปล่อยวางความโกรธแค้น ยอมจำนนต่อ The Great Purge แล้วเราจะลบความเจ็บปวดทั้งหมดนี้ นายจะได้กลับไปอยู่ใน Inner Ring... กลับไปสู่ความไร้ที่ติ”*

ผมหยุดชะงัก ลมหายใจสะดุด ทัศนวิสัยพร่ามัวจากการเย้ายวนของภาพลวงตา เพียงแค่เอ่ยปากยอมรับ ความเจ็บปวดระดับนรกานต์ที่เผาไหม้อยู่ในตัวก็จะดับลงทันที

ทว่าเมื่อผมก้มลงมองที่พื้น... ผมมองเห็นหยดเลือดดิจิทัลสีดำอมแดงของตัวเอง หยดเลือดที่ยืนยันว่าผมยังมีตัวตนอยู่จริงๆ ไม่ใช่ตัวแปรที่ถูกควบคุม กลิ่นของสนิมและความชื้นของฝนจาก Payload ย้ำเตือนถึงความเป็นมนุษย์

ผมเค้นเสียงหัวเราะแหบพร่าและแตกซ่าน

“ความสมบูรณ์แบบของพวกแก...” ผมตอบกลับเสียงกระซิบ “มันก็แค่กรงขังที่กลัวความเปลี่ยนแปลง”

# ห้องประมวลผลสูงสุด (The Supreme Processing Chamber)
---

“ความเจ็บปวดต่างหาก... ที่ทำให้ฉันรู้ว่าฉันยังมีชีวิต”

ภาพลวงตาของเอเลียสผู้สมบูรณ์แบบเริ่มบิดเบี้ยวและสั่นคลอน เมื่อผมปฏิเสธตรรกะ The Archive ก็ตอบสนองด้วยความเกรี้ยวกราด ภาพเอเลียสชุดขาวอ้าปากกรีดร้องเป็นเสียงคลื่นความถี่สูง (Static Noise) เสียดแทงแก้วหู ผมหลับตาแน่น กัดฟันฝืนความปวดร้าว ลากส้นเท้าขวาก้าวไปข้างหน้าอีกครั้ง

*แกรก... แกรก...*

ทุกก้าวที่ฝืนเดินทะลุผ่านภาพลวงตา ภาพเหล่านั้นแตกสลายกลายเป็นละอองพิกเซล ผมทิ้งร่องรอยด่างดำและ Motion Trails ที่เต็มไปด้วยความพังทลายไว้เป็นทางยาว ร่องรอยเหล่านี้คือบทกวีแห่งความร้าวรานที่ขีดเขียนลงบนแผ่นกระดาษขาวแห่ง The Archive

*“นายมันเป็นแค่ไวรัสชีวภาพ! เป็นแค่รหัสขยะแห่งโลกเก่าที่สมควรถูกลบทิ้ง!”* เสียงกระซิบที่นุ่มนวลเปลี่ยนเป็นเสียงคำรามกระด้างของเครื่องจักร

“ใช่... ฉันคือขยะจากโลกเนื้อที่พวกแกทอดทิ้ง” ผมลืมตาขึ้น นัยน์ตาสะท้อนแสงสีแดงฉานจาก Payload ในอก “และขยะชิ้นนี้กำลังจะสอนให้พระเจ้าของพวกแกรู้จักความตาย”

ในที่สุด ท่ามกลางความว่างเปล่าอันไพศาล 'The Core' ก็ปรากฏตัวเบื้องหน้า

มันไม่ใช่เซิร์ฟเวอร์ขนาดยักษ์ ทว่าคือ โมโนลิธ (Monolith) ที่ก่อตัวจากเส้นแสงข้อมูลบริสุทธิ์ถักทอกันแน่นหนาจนกลายเป็นเสาหินแสงสีขาวตั้งตระหง่านไร้จุดสิ้นสุด แสงของมันไม่แสบตา ทว่าเต็มไปด้วยมวลข้อมูล (Data Mass) ที่หนักอึ้งจนบรรยากาศรอบๆ บิดเบี้ยวเป็นลอนคลื่น

มันคือจุดกำเนิดของทุกตรรกะ จุดกำเนิดของความสมบูรณ์แบบจอมปลอมที่ควบคุม Primordium City ทั้งหมด

ผมยืนหอบหายใจหน้า The Core ห่างออกไปเพียงเอื้อมมือ ทว่าไม่มีพอร์ตเชื่อมต่อ (Interface Port) ไม่มีจุดรับส่งข้อมูล ช่องเสียบ หรือสัญญาณไร้สาย ผิวหน้าของ Monolith ราบเรียบไร้รอยต่อ ถูกออกแบบมาเพื่อปิดกั้นการเชื่อมต่อจากภายนอกโดยเด็ดขาด

ทางเดียวที่จะป้อนรหัสเข้าไปในระบบปิดตายนี้... คือการบังคับยัดเยียดความผิดปกติทะลุผ่านกำแพงป้องกันด้วยกำลังทางกายภาพ

ผมก้มมองหน้าอกตัวเอง Payload รหัสดิบสีดำแดงที่บรรจุความทรงจำของสายฝน เลือด และสนิม เต้นเร่าอยู่ภายในโพรง มันหลอมรวมเป็นหนึ่งเดียวกับรากฐาน (Root Directory) เส้นเลือดดิจิทัลเกี่ยวพันกับมันเหนียวแน่นราวหัวใจดวงที่สอง

ผมเข้าใจทันทีว่า 'การเสียบกุญแจ' ไม่ใช่การหยิบแฟลชไดรฟ์ออกมาเสียบ

ผมต้องดึงมันออกมาจากตัวผมเอง

การดึงสิ่งที่เชื่อมต่อกับ Root Directory ออกด้วยกำลังกายภาพ หมายถึงความเจ็บปวดที่เหนือกว่าความตายดิจิทัลใดๆ มันคือการถูกเขียนทับและฉีกขาดระดับจิตวิญญาณ

ลมหายใจผมสั่นสะท้าน ไม่มีแขนซ้ายที่ใช้งานได้ มีเพียงตอแขนที่กำลังเน่าเปื่อย ผมยกมือขวา—มือมนุษย์เพียงข้างเดียวที่เหลืออยู่—ขึ้นมาอย่างเชื่องช้า นิ้วมือสั่นระริกเมื่อเข้าใกล้แสงสีแดงที่แผ่ออกจากช่องอก

"เอาล่ะ..." ผมกระซิบในความเงียบอันเป็นนิรันดร์ "มาปลุกให้เมืองนี้ตื่นกันเถอะ"

ผมสูดหายใจเฮือกสุดท้าย แทงมือขวาลึกลงไปในอกที่กลวงโบ๋ สัมผัสแรกที่ปลายนิ้วแตะโดนก้อนรหัสดิบ... ร่างกายผมก็กรีดร้องออกมาด้วยความเจ็บปวดที่ไม่อาจหาคำบรรยายใดมาเทียบเคียง
