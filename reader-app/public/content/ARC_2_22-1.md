# บทที่ 22: ตลาดนัดความทรงจำที่ถูกบีบอัด (The Compressed Memory Bazaar) - Part A

เสียงแรกที่กระแทกเข้าโสตประสาทเมื่อผมลากส้นเท้าที่แหว่งหายไปก้าวเข้าสู่จัตุรัสกลางของ Mid-Sectors คือเสียงจอแจที่บิดเบี้ยว ผมพยายามหนีบแขนซ้ายที่ชาหนึบเพราะ Bit Rot ไว้ข้างลำตัว และยกมือขวากุมหน้าอกซ่อนความนูนของ Data Key Payload เอาไว้ ระแวดระวังสายตาของ The Hunter ที่ดมกลิ่นตามล่า High UX อย่างผมไม่เลิกรา

มันไม่ใช่เสียงตะโกนทักทายที่มีชีวิตชีวา แต่เป็นเสียงหึ่งๆ ที่ถูกลดความถี่ต่ำลง (Pitch-shifted) เสียงพูดคุยยืดยาน ขัดจังหวะด้วยเสียง Static ซ่าๆ คล้ายวิทยุที่หาคลื่นไม่เจอ จัตุรัสแห่งนี้คือโครงสร้างเหล็กและคอนกรีตดิจิทัลที่พันกันยุ่งเหยิง แสงนีออนสีแดงและน้ำเงินจากป้ายร้านค้าหยดย้อยและละลายไปกับหมอกสีเทา ราวกับสีน้ำที่ถูกฝนสาด

ที่นี่คือ "ตลาดมืดความทรงจำ" (Memory Black Market) ศูนย์กลางเศรษฐกิจของย่านที่ทุกอย่างถูกลดทอน

แผงลอยนับร้อยตั้งเบียดเสียดกันตามริมถนนที่แตกร้าว ไม่มีใครขายอาหารหรือเสื้อผ้า เพราะในโลกที่ร่างกายเป็นเพียงรหัส สิ่งเหล่านั้นไม่มีความหมาย สิ่งที่วางเรียงรายอยู่บนแผงเหล็กสนิมเขรอะคือ "ลูกบาศก์ข้อมูล" (Data Cubes) ก้อนเรขาคณิตโปร่งแสงที่เปล่งแสงริบหรี่อยู่ภายใน

ผมเดินเข้าไปใกล้แผงลอยแห่งหนึ่ง เจ้าของร้านเป็นชายร่างผอมที่ผิวหนังท่อนบนเต็มไปด้วยรอยแหว่งของ Compression Artifacts สีม่วงช้ำ เขาเหลือบมองผมด้วยดวงตาที่เป็นเพียงเส้นขีดสีขาว

"นายดูเหมือนเพิ่งร่วงลงมานะ หนุ่มน้อย" เสียงของเขากระตุกเป็นห้วงๆ "ความละเอียดของนายยังมีกลิ่นอายของ Inner Ring หลงเหลืออยู่... สนใจแลกเปลี่ยนไหม?"

"แลกอะไร?" ผมถาม เสียงของผมเองก็เริ่มสั่นพร่า

เขาหยิบลูกบาศก์สีอำพันขึ้นมาหมุนในมือ "นี่คือ 'ความรู้สึกของการจูบครั้งแรก' ความละเอียดระดับปานกลาง มีรอยขีดข่วนนิดหน่อยช่วงท้าย แต่ความรู้สึกวาบหวามในช่องท้องยังอยู่ครบ... แลกกับความทรงจำระดับ High-Res ของนายสักชิ้น เอาแบบที่มีสีฟ้าของท้องฟ้าเต็มๆ ตา ฉันให้ราคาดี"

ผมเบิกตากว้าง จ้องมองลูกบาศก์ในมือเขา

ความทรงจำที่มีความละเอียดสูง... คือสกุลเงินของที่นี่

เพื่อรักษาสถานะตัวตนไม่ให้กลายเป็น The Hollows หรือเปลือกข้อมูลที่ไร้สติ คนที่นี่ต้องบริโภค "ความรู้สึก" ที่ชัดเจนเพื่อกระตุ้นโค้ดของตัวเอง พวกเขาเฉือนเอาประสบการณ์ในอดีตของตนเองมาเร่ขาย เพื่อซื้อเวลา ซื้อความทรงจำของคนอื่นมาปะติดปะต่อตัวตนที่กำลังแตกสลาย

ผมรีบกะเผลกถอยห่างจากแผงลอยนั้น รู้สึกสะอิดสะเอียนอย่างบอกไม่ถูก อีกทั้งความรู้สึกเหมือนถูกระบบจ้องมองยังคอยกดดันหลังคอผมอยู่ตลอด การเอาความรู้สึกที่เปราะบางที่สุดของมนุษย์มาบรรจุหีบห่อและต่อรองราคา... มันดิบเถื่อนยิ่งกว่าการเข่นฆ่าทางกายภาพเสียอีก
