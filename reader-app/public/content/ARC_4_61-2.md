# Chapter 61: มิติที่สี่แห่งความว่างเปล่า (The Fourth Dimension of Void) - Part 2

การทรงตัวใน The Core ไม่ใช่เรื่องของการรักษาสมดุลทางกายภาพ แต่เป็นการต่อสู้ดิ้นรนเพื่อรักษาสติสัมปชัญญะไม่ให้แตกสลายไปกับมิติที่ทับซ้อน

ผมเซล้มไปข้างหน้า มือขวาที่เหลืออยู่เพียงข้างเดียวตะปบลงบน 'พื้น' ที่จู่ๆ ก็เปลี่ยนมุมเอียงเป็นเก้าสิบองศา ทันทีที่ฝ่ามือของผมสัมผัสกับพื้นผิวที่เรียบเนียนราวกับกระจกของสถาปัตยกรรมสีขาว รอยร้าวสีดำของ Bit Rot ก็แตกแขนงออกจากปลายนิ้วของผม ราวกับใยแมงมุมที่ถักทอด้วยความเน่าเปื่อย ทิ้งรอยมลทินไว้บนความบริสุทธิ์ที่ The Archive หวงแหน

ความปวดร้าวจากส้นเท้าขวาที่แหว่งหายแล่นพล่านขึ้นมาอีกครั้ง มันไม่ใช่แค่ความเจ็บปวดทางกล้ามเนื้อ แต่มันคือ Error Code ระดับ Root ที่กำลังแทรกซึมเข้าสู่กระบวนการรับรู้ของผม ข้อมูลของผมกำลังบอกว่าส้นเท้ายังอยู่ แต่มิติที่ประมวลผลอยู่บอกว่ามันว่างเปล่า ความขัดแย้งของตรรกะนี้สร้างแรงสั่นสะเทือนในหัวจนผมเห็นภาพซ้อน (Stuttering) เป็นระยะๆ โลกทั้งใบกระตุกและหยุดนิ่งไปเสี้ยววินาที เหมือนเฟรมเรตที่ตกลงอย่างกะทันหัน

[Note to Editor: อาการ Resolution Loss เริ่มแสดงออกทางสายตาของเอเลียส ภาพที่เห็นจะกระตุกและฉีกขาด]

ผมพยุงตัวลุกขึ้นยืน ลมหายใจหอบลึก ช่องอกที่เปิดอ้าของผมส่งเสียงหึ่งๆ คล้ายเครื่องจักรที่โอเวอร์โหลด ก้อน Payload ที่ฝังตัวอยู่ภายในสัมผัสกับคลื่นความถี่ของที่นี่ มันกำลังสั่นสะเทือน ตอบสนองต่อแกนกลางประมวลผลที่อยู่ลึกเข้าไป มันพยายามจะปรับตัวให้เข้ากับเรขาคณิต 5 มิติที่รายล้อม แต่อวัยวะที่พังทลายของผมกลับเป็นตัวดึงรั้งมันไว้ 

การที่ตัวผม 'ไม่สมบูรณ์' ทำให้ผมยังคงเป็นผม 

หากผมไร้ซึ่งบาดแผล หากผมเป็นโค้ดที่สมบูรณ์แบบร้อยเปอร์เซ็นต์ The Core คงจะผสานผมเข้าเป็นส่วนหนึ่งของผนังสีขาวพวกนี้ไปแล้วตราบเท่าที่ผมก้าวเท้าเข้ามา แต่เพราะผมแหว่งวิ่น เพราะส้นเท้าขวาของผมขาดหาย เพราะแขนซ้ายของผมเป็นเพียงตอประสาทที่ไหม้เกรียม และเพราะหน้าอกของผมฉีกขาด... ระบบอันเปี่ยมล้นด้วยตรรกะสูงสุดจึง 'ไม่สามารถประมวลผล' ความผิดพลาดนี้ได้ มันไม่รู้วิธีจัดการกับพารามิเตอร์ที่ไม่มีค่า (Null Reference) ที่เดินได้

ผมคือบั๊กที่มันไม่อาจกลืนกิน

"ความพังทลาย... คือเกราะป้องกัน" ผมหัวเราะเยาะออกมาอย่างขมขื่น เสียงหัวเราะกลายเป็นอาการไอที่พ่นเศษรหัสสีดำเล็กๆ ออกมาเปื้อนริมฝีปาก รสชาติของมันเหมือนสนิมและไฟฟ้าลัดวงจร

ผมลากขาขวาต่อไป ก้าวแล้วก้าวเล่า ผ่านซุ้มประตูที่ม้วนตัวเข้าหาศูนย์กลางอย่างเป็นอนันต์ ผมมองเห็นรอยทางสีดำที่ส้นเท้าของผมลากผ่าน มันไม่ได้เลือนหายไปเหมือนรอยเท้าบนทราย แต่มันกำลังสลักลึกลงไปในโครงสร้างของระบบ ฝังรากของไวรัสแห่งความเสื่อมสลายลงในระดับควอนตัม 

ในมิติที่ปราศจากเวลา ผมไม่อาจรู้ได้ว่าตัวเองเดินมานานแค่ไหนแล้ว ทุกอย่างดูซ้ำซากแต่กลับไม่เคยเหมือนเดิม สถาปัตยกรรมเปลี่ยนแปลงตัวเองทุกครั้งที่ผมละสายตา แต่มีสิ่งหนึ่งที่ชัดเจนขึ้นเรื่อยๆ...

ความกดดัน

มันหนักอึ้งราวกับมหาสมุทรที่พยายามบีบอัดผมให้กลายเป็นจุดศูนย์กลางเพียงจุดเดียว ผิวหนังดิจิทัลของผมเริ่มมีรอยแตก (Artifacting) เป็นปื้นสีม่วงและเขียวตามขอบแขนเสื้อและใบหน้า ความละเอียด (Resolution) ของผมกำลังลดลงจากการเผชิญหน้ากับความสมบูรณ์แบบที่มากเกินไป

ทว่า ในขณะที่ร่างนี้กำลังจะแหลกสลาย ผมกลับรู้สึกถึงเป้าหมายที่เด่นชัดที่สุดในชีวิต

ผมจะลากความเน่าเปื่อยนี้ไปให้ถึงหัวใจของมัน ผมจะแพร่เชื้อให้พระเจ้าจอมปลอมองค์นี้ได้รู้จักความหมายที่แท้จริงของ 'ความเจ็บปวด'

และเมื่อถึงเวลานั้น ความบริสุทธิ์จอมปลอมเหล่านี้จะถูกแทนที่ด้วยรอยร้าวแห่งความเป็นจริง
