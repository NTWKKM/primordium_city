# บทที่ 29: รอยเท้าของความว่างเปล่า (Part A)

รอยเท้าของความว่างเปล่าทอดยาว ทะลวงม่านหมอกลงสู่พรมแดนที่ Mid-Sectors กลืนหายไปกับความเสื่อมโทรมของ The Fringe ผมฝืนวิ่งลากส้นเท้าขวาที่แหว่งวิ่น ขยับตัวไปข้างหน้าอย่างทุลักทุเล แขนซ้ายที่ตายด้านจาก Bit Rot ห้อยแกว่งตามแรงโน้มถ่วงจำลองที่เริ่มบิดเบี้ยว ร่องรอยของพวก Nullifiers—กลุ่มก้อนความว่างเปล่าที่ขโมยความทรงจำ—เป็นเพียงรอยไหม้สีดำบนพื้นผิวข้อมูล

ยิ่งปล่อยให้ความโกรธแค้นแผดเผา โครงสร้างร่างกายของผมก็เริ่มสั่นคลอน การประมวลผลอารมณ์ดิบเถื่อนดึงแบนด์วิดท์มหาศาล (Uncompressed Emotion) ขอบร่างของผมเกิดอาการ Tearing ฉีกขาดเป็นเส้นริ้ว โมเดลโพลีกอนหดตัวลงสู่ความหยาบกระด้าง พื้นผิวรอบตัวสูญเสียความคมชัด กลายเป็นเพียงบล็อกสีขมุกขมัว โลกทั้งใบกระตุก Jittering เหมือนสัญญาณภาพที่ขาดหาย ผมกำลังสูญเสียความละเอียดทางกายภาพเพื่อแลกกับการประคองความโกรธเอาไว้

ความหวาดกลัวต่อภาวะ Vacuity เกาะกุมจิตใจ ทว่ายิ่งผมหวาดกลัว เรโซลูชันของผมยิ่งดิ่งฮวบ แขนขวาที่เคยสมบูรณ์แบบแตกระแหงเป็นโมเสก ผมกำลังเดินเข้าสู่ปากเหวแห่งการหลงลืม

# บทที่ 29: รอยเท้าของความว่างเปล่า (Part B)

สัญญาณของ UX Index ดังแว่วมาจากเบื้องหลัง มันไล่ต้อนผมลงสู่เงามืดอย่างไม่ลดละ

ผมหลับตาลง บังคับตัวเองให้ยอมรับความบิดเบี้ยวของร่างกาย ผมปลดล็อกระบบต่อต้านการสั่นไหว ปล่อยให้ร่างที่พร่ามัวผสานเข้ากับสภาพแวดล้อมที่แตกร้าว ทันทีที่หยุดดิ้นรนรักษาสถานะ High-Res น้ำหนักของ Payload ในอกก็สมดุลขึ้น ผมก้าวเดินผ่านร่องรอยสีดำของพวก Nullifiers ได้คล่องตัวกว่าเดิม ข้ามผ่านขีดจำกัดของความกลัวต่อความเสื่อมสลาย

รอยไหม้นั้นสิ้นสุดลงตรงลานกว้าง ซาก Data Cubes ถูกดูดกลืนความทรงจำจนเหือดแห้ง ทิ้งไว้เพียงเปลือกโพลีกอนด้านชา สิ่งที่รอผมอยู่ไม่ใช่กลุ่มก้อนความว่างเปล่า กลับเป็นกลุ่มเงาที่ยืนตระหง่านท่ามกลางหมอกของความเสื่อมสลาย ร่างกายของพวกเขามีเพียงรอยร้าว พิกเซลแตกซ่านเป็นฝุ่นผงล่องลอย

พวกเขาคือ The Eroded ชุมชนผู้รอดชีวิตที่หยัดยืนอยู่บนซากปรักหักพัง ผู้มองเห็นความจริงที่ซ่อนอยู่ในความพังทลาย
