# บทที่ 25: เสียงกระซิบจากรอยบีบอัด (ส่วนที่ 1)

หลังจากถูกเนรเทศออกมายังขอบชายแดนของ Mid-Sectors ความหนาแน่นของเมืองก็เบาบางลง ท้องฟ้าที่เคยเป็นสีฟ้าครามสดใส บัดนี้กลับมีเส้นตาราง (Grid) ลางๆ พาดผ่านก้อนเมฆที่ดูเหมือนภาพ JPEG ที่ถูกเซฟทับซ้ำๆ จนสูญเสียรายละเอียด 

ผมเดินกะเผลกลากขาขวาที่ไร้ส้นเท้า ก่อนจะทรุดตัวลงนั่งพิงกำแพงอิฐที่ลวดลายของมันไหลเยิ้ม ผมฝืนยกแขนซ้ายขึ้นมาดู... ร่องรอยการบีบอัด (Compression Artifacts) ปรากฏขึ้นชัดเจนกว่าเดิม มันเป็นปื้นสีม่วงและเขียวสะท้อนแสงที่บิดเบี้ยวแทรกซึมอยู่ใต้ผิวหนังจำลอง แขนข้างนี้ปวดร้าวรุนแรงและชาหนึบ ความอึดอัดจาก Data Key Payload กลางอกเตือนให้รู้ว่า The Hunter ยังคงตามล่าหาความผิดปกติของผมอยู่อย่างไม่ลดละ การมองเห็นความเสื่อมสลายของตัวตนด้วยตาเปล่านั้น กรีดลึกลงไปในสติสัมปชัญญะ

ผมเคยหวาดกลัวรอยพวกนี้ มันคือตราบาปของความล้มเหลว คือสัญญาณเตือนว่าตัวตนของผมกำลังจะพังทลายลงสู่ The Hollows

แต่ในความเงียบสงัดของชั่วโมงนี้ เมื่อปราศจากเสียงจอแจของเมืองและแสงแข็งทื่อของพวก Preservationists ผมเริ่มสังเกตเห็นอะไรบางอย่าง

ปื้นสีเหล่านั้นไม่ได้กระพริบแบบสุ่ม 

มันมีจังหวะ 

ผมหลับตาลงและพยายามเพ่งความสนใจไปที่ 'เสียง' ของการเสื่อมสลายนั้น โดยปกติแล้ว Bit Rot จะมาพร้อมกับเสียง Static ซ่าๆ เหมือนทีวีที่จูนหาช่องไม่เจอ แต่เมื่อผมปล่อยให้สติสัมปชัญญะของผมซิงค์กับการบีบอัดนั้น... เสียงซ่าเหล่านั้นก็เริ่มแยกชั้นออกเป็นย่านความถี่ต่างๆ


# บทที่ 25: เสียงกระซิบจากรอยบีบอัด (ส่วนที่ 2)

มันไม่ใช่เสียงรบกวน (Noise) ที่ไร้ความหมาย แต่มันคือภาษา

เสียงกระตุกถี่ๆ (Stuttering) รวมตัวกันกลายเป็นจังหวะเพอร์คัสชันที่แปลกประหลาด เสียงความถี่ต่ำที่ดังหึ่งๆ เหมือนหม้อแปลงไฟฟ้าเก่าๆ กลับซ่อนเส้นทำนองที่โศกเศร้าและลึกล้ำ มันคือบทเพลงแห่งความบิดเบี้ยว... เสียงกระซิบจากรหัสที่ถูก The Archive ทอดทิ้ง

*"...ลงไป... ลงไปใต้กระจก..."*

ผมสะดุ้ง ลืมตาโพลง เสียงนั้นไม่ได้มาจากหู แต่ดังก้องอยู่ในโสตประสาทที่ลึกที่สุด มันถูกส่งผ่านรอยบีบอัดบนแขนของผม 

ผมทรงตัวลุกขึ้นยืนอย่างทุลักทุเลบนขาขวาที่ไร้ส้นเท้า กวาดสายตามองไปยังพื้นที่รกร้างเบื้องหน้า ข้อมูลที่เน่าเสียเหล่านี้ไม่ใช่เพียงแค่ขยะดิจิทัล แต่มันคือข้อความที่ถูกบีบอัดซ่อนไว้โดยใครบางคน... หรือบางสิ่งบางอย่าง ที่อาศัยอยู่เบื้องล่าง ในบ่อเกรอะข้อมูล (The Sump) 

พวก Preservationists หวาดกลัวความเสื่อมสลายเพราะพวกเขาคิดว่ามันคือจุดจบ แต่พวกเขาคิดผิด อาการ Artifacting ไม่ใช่แค่การพังทลายของโลกเก่า แต่มันคือกุญแจที่ใช้เปิดประตูไปสู่ความจริงที่ซ่อนอยู่หลังภาพลวงตาอันสมบูรณ์แบบนี้ 

ผมใช้มือขวากดหน้าอกที่มี Data Key ซ่อนอยู่ แล้วแตะลูบปื้นสีม่วงบนแขนซ้ายที่ปวดร้าว สัมผัสถึงจังหวะชีพจรของรหัสที่กำลังกลายพันธุ์ ความหวาดกลัวจากการถูก The Hunter ดมกลิ่นตามล่า ค่อยๆ ถูกแทนที่ด้วยความสงสัยอันแรงกล้า 

ถ้าความละเอียดต่ำคือข้อแลกเปลี่ยนในการมองเห็นความจริง... ผมก็พร้อมที่จะปล่อยให้ตัวเองพร่าเลือน
