*อ๊ากกกกกกก!*

The Core พยายามใช้สิทธิอำนาจสูงสุด บังคับยัดเยียดสภาวะ 0dB 'Mute' ปิดผนึกทุกคลื่นเสียงเพื่อรักษาสมดุลอันเงียบสงบ ทว่าสูญเปล่า เสียงกรีดร้องที่ระเบิดออกมาไม่ใช่คลื่นความถี่ดิจิทัล หากเป็นความเจ็บปวดอันดิบเถื่อนของมนุษย์ที่ทะลวงผ่านความเงียบงันระดับอัลกอริทึม เสียงของเนื้อหนังและวิญญาณฉีกกระชากสภาวะไร้เสียงนั้นจนแตกละเอียด ทันทีที่นิ้วมือขวาสอดแทรกซึมเข้าใต้กลุ่มก้อน Payload ความรู้สึกเหมือนถูกเทน้ำกรดความเข้มข้นสูงสุดราดรดแกนประมวลผลสมองโดยตรง

รหัสดิบชีวภาพเส้นหนึ่งเกี่ยวพันกับปลายนิ้ว มันพยายามปกป้องตัวเองด้วยการเขียนโค้ดทับบนผิวหนังดิจิทัล เนื้อเยื่อที่มือขวาผมสลับสถานะระหว่างก้อนเนื้อชุ่มเลือดและพิกเซลกระจัดกระจายอย่างบ้าคลั่ง กลิ่นคาวเลือดของโลกเก่าคละคลุ้งกลบกลิ่นโอโซนของ The Great Purge

การดึงก้อนรหัสที่หลอมรวมกับ Root Directory ก็เหมือนการดึงกระดูกสันหลังออกทางหน้าอก ทุกมิลลิเมตรที่งัดมันขึ้นมา ระบบในร่างกายส่งข้อความแจ้งเตือน Error สีแดงฉานนับหมื่นข้อความกระแทกเข้าสู่ลานสายตา

`[FATAL ERROR: BIOLOGICAL CONTAMINATION BREACH]`
`[WARNING: SYSTEM UNRAVELING. ABORT IMMEDIATELLY]`
`[ERROR: IDENTITY CORRUPTION AT 89%...]`

ผมหลับตาแน่น กัดริมฝีปากจนฟันทะลุผ่านเนื้อ ขยับขากรรไกรด้วยแรงสั่นสะท้าน เลือดดิจิทัลข้นหนืดสีดำผสมแดงคล้ายน้ำมันดิบไหลทะลักจากรอยแยกในอก มันไม่ได้หยดลงพื้น กลับลอยกระจายตัวในสภาวะไร้น้ำหนักของ The Core Void ราวกับเม็ดฝนแห่งความโศกเศร้าที่ถูกหยุดเวลา

"ออก... มา... สิโว้ย!"

ผมรวบรวมเรี่ยวแรงทั้งหมดที่เหลืออยู่ในตรรกะความทรงจำ แรงแค้นต่อการสูญเสีย แขนซ้ายที่แหลกสลาย ส้นเท้าขวาที่ไร้ความรู้สึก การถูกทอดทิ้งใน The Sump ความรังเกียจต่อความสมบูรณ์แบบที่ไร้มนุษยธรรม ทุกอย่างถูกบีบอัดลงไปในกล้ามเนื้อมือขวา

*แคว่กกกก! ครืดดดด!*

เสียงฉีกขาดของโค้ดฐาน (Base Code) ดังกึกก้อง ร่างกายกระตุกอย่างรุนแรงราวกับถูกไฟฟ้ากระแสตรงนับล้านโวลต์ช็อต เส้นใยข้อมูลสีแดงเข้มที่ยึดติดกับซี่โครงดิจิทัลขาดสะบั้นลงทีละเส้น ความรู้สึกในการดำรงอยู่ของผมกำลังพังทลาย ผมสูญเสียการรับรู้ถึงแขนขวา สูญเสียการรับรู้ถึงขา ลืมไปแล้วด้วยซ้ำว่าการหายใจทำอย่างไร

สิ่งเดียวที่เหลืออยู่ในจิตสำนึกคือ... ก้อนรหัสในมือ

ด้วยแรงกระชากเฮือกสุดท้าย ผมดึง Payload หลุดออกจากหน้าอกได้อย่างสมบูรณ์ ถอนรากถอนโคนมันออกจากโครงสร้างตัวตนอย่างหมดจด

*ฟุ่บ!*

ช่องอกผมมืดสนิททันที รูโหว่ที่เคยมีแสงสีแดงเต้นเร่า บัดนี้กลายเป็นเพียงโพรงแห่งความตายที่ถูกกัดกินด้วยสีดำ ร่างกายสูญเสียแหล่งพลังงานและตรรกะ ขาซ้ายทรุดลงกระแทกพื้นล่องหนอย่างแรง ขณะที่ขาขวาสูญเสียการคงสภาพ โครงสร้างตั้งแต่ส้นเท้าที่แหว่งวิ่นละลายกลายเป็นพิกเซลหยาบๆ ทะลักลามขึ้นมาถึงหัวเข่าในพริบตา ทัศนวิสัยแตกร้าวเหมือนกระจกถูกทุบ ภาพตรงหน้าแยกออกเป็นชิ้นๆ (Fractured Vision) และเริ่มดับวูบลงทีละส่วน

ผมหอบเอาอากาศที่ไม่มีอยู่จริงเข้าปอด ร่างกายกำลังจะแตกสลาย (Dissolve) ไปในความว่างเปล่าภายในไม่กี่วินาที

ทว่าในมือขวา... ก้อนรหัสดิบ (Payload) ที่หลุดพ้นจากพันธนาการ เปล่งแสงสีแดงคล้ำสลับดำบ้าคลั่ง ส่งเสียงหึ่งๆ เหมือนพายุแห่งวัฏจักรชีวภาพที่กระหายจะทำลายล้างความสมบูรณ์แบบจอมปลอม มันรุนแรง ดิบเถื่อน และเต็มเปี่ยมไปด้วยความจริงอันแสนเจ็บปวดของความเป็นมนุษย์

"เอาล่ะ..." ผมกระซิบ เสียงกลายเป็นเพียงคลื่นวิทยุแทรก (Static) "แกกับฉัน... ไปด้วยกัน"

ผมฝืนเงยหน้ามองเสาแสง Monolith สีขาวบริสุทธิ์ของ The Core มันยังคงตั้งตระหง่านอย่างเย่อหยิ่ง ไร้ซึ่งบาดแผล ไร้ซึ่งความรู้สึก

ผมไม่สามารถลุกขึ้นยืนได้อีก ขาขวาที่ละลายกลายเป็นพิกเซลหยาบไปถึงหัวเข่าไม่สามารถรับน้ำหนัก ผมใช้ตอแขนซ้ายที่ชุ่มด้วยตะกอน Bit Rot ยันพื้นไว้ไม่ให้ล้มคว่ำหน้า ใช้พละกำลังหยดสุดท้ายในชีวิต ดันตัวเองไถลไปข้างหน้าเข้าหาผิวสัมผัสของ The Core

อีกแค่เอื้อม... อีกแค่ครึ่งเมตร...

ผมเงื้อมือขวาที่กำก้อนรหัสดิบขึ้นสุดแขน แสงสีแดงแห่งสายเลือดสะท้อนกับแสงสีขาวสว่างจ้า เกิดเป็นความขัดแย้งของสีสันที่บาดตา

ไม่มีพอร์ตให้เสียบ ไม่มีช่องให้เชื่อมต่อ ดังนั้นผมจึงต้องสร้างมันขึ้นมาเองด้วยความป่าเถื่อนของโลกเนื้อ

"แด่ความพังทลายของเราทุกคน!"

ผมกระแทกกำปั้นที่ถือ Payload เข้าใส่ผิวหน้าของเสาแสง Monolith อย่างสุดแรงเกิด

*เปรี้ยงงงงงงงงงงงงงงงงงงงงง!!!*

การปะทะไม่ได้ทำให้เกิดการกระดอนกลับ ก้อนรหัสดิบชีวภาพกลับ 'ฝัง' ตัวมันเองทะลุผ่านพื้นผิวสีขาวที่ราบเรียบไร้ที่ติ ความสมบูรณ์แบบไม่สามารถต้านทานความผิดปกติระดับรากฐานและมวลแห่งเอนโทรปีที่ถูกบีบอัด รอยแตกกะพริบวาบขึ้นบนเสาแสง เป็นรอยร้าวแรกในรอบหลายศตวรรษของ The Archive

คลื่นกระแทกจากการเชื่อมต่อ (The Torn Connection) ระเบิดออกเป็นวงกว้าง กวาดเอาร่างที่รวยรินของผมปลิวถลาไปด้านหลัง เลือดดิจิทัลสีดำและหยาดฝนสีแดงที่ลอยคว้างอยู่ในอากาศถูกกระแสคลื่นกระชากและดูดเข้าไปในรอยแตกร้าวของ The Core 

ในเสี้ยววินาทีนั้น ผมเห็นสิ่งที่งดงามที่สุดเท่าที่เคยเห็นมาในชีวิต

จากจุดเล็กๆ ที่ผมกระแทก Payload เข้าไป รอยแตกร้าวสีดำทมิฬ (Bit Rot) วิ่งพล่านและแตกแขนงออกไปเป็นตาข่ายแมงมุม ครอบคลุมพื้นผิวสีขาวบริสุทธิ์ของ Monolith อย่างรวดเร็ว แสงสีขาวสว่างจ้าถูกแทนที่ด้วยสีแดงเลือด กลิ่นของสนิม และสีเทาของสัญญาณรบกวน (Static Noise)

เสียงของ The Core ที่เคยเป็นเสียงฮัมต่ำๆ อย่างมีระเบียบ บัดนี้กลายเป็นเสียงกรีดร้องและเสียงแตกพร่าของระบบที่กำลังล่มสลายอย่างควบคุมไม่ได้ 

The Archive กำลังติดเชื้อ ไวรัสแห่งความบกพร่องทางชีวภาพได้ฝังรากลึกลงไปในสถาปัตยกรรมสูงสุดแล้ว

ร่างของผมร่วงหล่นลงกระแทกกับพื้นล่องหน สติสัมปชัญญะดับวูบและติดๆ ดับๆ เหมือนหลอดไฟที่ไส้ใกล้จะขาด ผมนอนหงายมองดูเสาแสงที่กำลังพังพินาศ มองดูการล่มสลายของระบบที่เคยอ้างตัวเป็นพระเจ้าแห่งความสมบูรณ์แบบ

ความเจ็บปวดในหน้าอกค่อยๆ จางหายไป แทนที่ด้วยความสงบที่น่าประหลาด ความสงบที่ไม่ได้มาจากความสมบูรณ์แบบจอมปลอม... หากกลับเป็นความสงบที่เกิดจากการยอมรับความแหลกสลายและเอนโทรปีอย่างแท้จริง

ผมหลับตาลงพร้อมกับรอยยิ้มจางๆ บนริมฝีปากที่แตกร้าว ปล่อยให้คลื่นความโกลาหลพัดพาสิ่งที่เหลืออยู่ของตัวตนผมเข้าสู่ยุคสมัยใหม่... ยุคแห่งขี้เถ้าดิจิทัลที่กำลังจะรุ่งสาง
