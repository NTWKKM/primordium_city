# ไขรหัสกุญแจ
#### // FILE: TURNING_THE_KEY
---

กำแพงตรรกะพังทลาย Payload ในอกตอบสนองต่อเส้นทางสู่ Logical Rift ในเสี้ยววินาที ความเจ็บปวดระดับที่ทำลายล้างจิตสำนึกปะทุขึ้น ก้อนโค้ดสีดำเข้มขยายตัวอย่างฉับพลัน มันงัดกระดูกซี่โครงดิจิทัลหักสะบั้น เสียงร้าวลั่นของกระดูกดังก้องพร้อมกับชิ้นเนื้อโพลีกอนที่ปริแตก โค้ดดิบๆ ร้อนระอุทะลักออกจากรอยแยกกลางอกราวกับลาวา พุ่งตัวออกไปเหมือนเส้นใยประสาท เสียบทะลุเข้ากับแกนกลางของ Logical Rift 

ร่างของผมถูกกระชากไปข้างหน้า กลายเป็นเพียงกุญแจเนื้อหนังที่เสียบเข้ากับระบบ ความรู้สึกตอนที่ข้อมูลถูกสูบออกไปผ่านบาดแผลเปิดกว้างนั้นคือการถูกชำแหละทั้งเป็น 

และในเสี้ยววินาทีของการเชื่อมต่อ ผมเห็นมัน ความจริงของ Payload ไม่ใช่ไวรัสทำลายล้าง ไม่ใช่โค้ดมุ่งร้าย มันคือความทรงจำที่ถูกบีบอัด—บันทึกความรู้สึกอันดิบเถื่อนของโลกใบเก่า (The Old World)

# ไขรหัสกุญแจ
#### // FILE: TURNING_THE_KEY

ภาพนิมิตระเบิดในหัว สัมผัสของหยาดฝนเย็นเฉียบปะทะใบหน้า กลิ่นดินชื้นหลังพายุ ความอบอุ่นของเลือดเนื้อจริง อัตราการเต้นของหัวใจออร์แกนิก นี่คือข้อมูลระบบหลักที่ The Archive พยายามลบกั้นทิ้ง—ความทรงจำว่าการเป็นมนุษย์จริงๆ รู้สึกอย่างไร Payload คือมรดกชิ้นสุดท้ายของเผ่าพันธุ์ที่ปฏิเสธการถูกลดทอนเป็นเพียงสมการ

มวลความทรงจำหลั่งไหลเข้าสู่รูกุญแจ รอยแยกขยายตัว ระบบรักษาความปลอดภัยระดับรากฐานของ The Sump กรีดร้องลั่น รหัสผ่านถูกไขออกด้วยความทรงจำอันเข้มข้น บานประตูแห่งตรรกะที่ปิดตายเลื่อนเปิดออก 

เบื้องหลังประตูบานนั้นคือแสงสว่างจ้า แสงสีขาวบริสุทธิ์ สะอาดสะอ้านและไร้ที่ติ แสงจาก The Core (แกนกลาง) สถานที่ซึ่งถูกสงวนไว้สำหรับความสมบูรณ์แบบสูงสุด ความงดงามของมันเย็นชาและโหดร้าย ล้อเลียนความสกปรกของร่างกายที่เน่าเปื่อยซึ่งยืนโซเซอยู่หน้าประตู
