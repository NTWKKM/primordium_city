# Chapter 60: ก้าวข้ามผ่านก้นบึ้ง (Crossing the Abyss)

ผมหอบหายใจรวยริน เลือดข้อมูลสีดำยังคงไหลหยดจากแขนซ้ายที่กุดหายและช่องอกที่เปิดอ้า ใบหน้าค่อยๆ หันกลับไปมองเบื้องหลัง... The Sump ก้นบึ้งแห่งความสิ้นหวังที่กักขังมาแสนนาน 

ในความมืดมิดนั้น มีเพียงเสียงสะท้อนของ Data Ghosts และกลิ่นอายของสนิม มันคือสถานที่ที่กลืนกินชิ้นส่วนร่างกาย เป็นสถานที่ที่พราก Grit ไป ทิ้งไว้เพียงเศษเสี้ยวความทรงจำของการเสียสละ แม้จะเกลียดชังมันสุดหัวใจ แต่ความโสมมนี้ก็หล่อหลอมให้ผมกลายเป็นกุญแจที่ทำลายกฎเกณฑ์ทั้งหมด สายตาจ้องมองความมืดมิดเป็นครั้งสุดท้าย สลักรอยแผลเป็นและความสูญเสียไว้ในหน่วยความจำที่ลึกที่สุด 

[ขอให้การเสียสละของนายไม่สูญเปล่า... Grit] ผมกระซิบในใจ


ผมหันกลับมาเผชิญหน้ากับแสงสว่างอันโหดร้ายของ The Core อีกครั้ง สูดหายใจลึกๆ เอาความบริสุทธิ์ของโอโซนที่บาดคอเข้าไปในปอดจำลอง 

ลาก... ส้นเท้าขวาที่พังทลายถูกลากไปตามพื้น ขยับเท้าซ้ายไปข้างหน้า แบกรับร่างกายที่พังพินาศเกินเยียวยาข้ามผ่านธรณีประตูแห่งตรรกะ ก้าวเดินเข้าไปในแสงสว่างนั้น ทิ้งรอยเท้าที่เต็มไปด้วยโค้ดไวรัสสีดำและ Bit Rot แปดเปื้อนลงบนพื้นผิวอันบริสุทธิ์ของแดนศักดิ์สิทธิ์ ทุกย่างก้าวคือการประกาศสงคราม คือการนำพาเอาโรคระบาดแห่งความเป็นมนุษย์เข้ามากัดกินความสมบูรณ์แบบจอมปลอม 

ผมเข้าสู่ The Core แล้ว... และจะเผามันให้ราบเป็นหน้ากลอง
