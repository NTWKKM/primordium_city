# แรงโน้มถ่วงที่บิดเบี้ยว
#### // FILE: DISTORTED_GRAVITY
---

ทันทีที่ก้าวล่วงล้ำเข้ามาในความมืด พื้นที่ใต้ฝ่าเท้าก็พลันอันตรธาน การร่วงหล่นครั้งนี้ปราศจากสายลมต้าน ไม่มีการกระชากของแรงดึงดูดแบบโลกเนื้อหนัง มันคือการพังทลายของพิกัดแกน Z ร่างของผมถูกดูดกลืนลงสู่ก้นบึ้งแห่งโครงสร้างสถาปัตยกรรม ก่อนจะกระแทกเข้ากับพื้นผิวเรขาคณิตที่ปราศจากการลบเหลี่ยมมุม 

*แคร่ก—*

เสียงแตกร้าวของรหัสข้อมูลดังก้องในกะโหลกจำลอง โลกทั้งใบกระตุกค้าง (Stuttering) คลื่นสีเขียวอมม่วงแห่งความเน่าเปื่อยสาดกระเซ็น สลับกับสีดำสนิทของความว่างเปล่า ภาพเบื้องหน้าเรนเดอร์ตัวเองอย่างตะกุกตะกัก ที่นี่คือ **The Sump** บ่อเกรอะข้อมูล... ดินแดนที่อัลกอริทึมพระเจ้าทอดทิ้ง

รอบกายผมคือสถาปัตยกรรมแบบ Low-poly Brutalism ขอบถนนแหลมคมราวกับใบมีดแก้ว ขยะพิกเซลถูกบีบอัดทับถมกันเป็นกำแพงสูงตระหง่าน แสงสว่างเดียวที่มีคือประกายไฟสีขุ่นจากกระแสข้อมูลที่ลัดวงจร 

ผมฝืนยันตัวลุกขึ้น โลกทั้งใบเอียงวูบ อาการ **Digital Vertigo** พุ่งเข้ากระแทกโสตประสาท กฎของฟิสิกส์ถูกฉีกทิ้ง แรงโน้มถ่วงกระชากผมลงต่ำสลับกับเหวี่ยงไปด้านข้าง ผมตะเกียกตะกายจิกนิ้วลงบนพื้นผิวหยาบกระด้าง รอยถลอกกลายเป็นเส้นโค้ดสีแดงกะพริบถี่

จังหวะที่น้ำหนักตัวทิ้งลงบนข้อเท้าขวา ความเจ็บปวดแล่นปราดขึ้นมาตามเส้นประสาทเทียม มันคือ Phantom Pain ส้นเท้าที่แหว่งหายไปถูกแทนที่ด้วยความทรมานแหลมคม สัญญาณ Error กรีดร้องเหมือนตะปูตอกย้ำลงบนบาดแผลที่ไม่มีอยู่จริง ตอกย้ำว่าผมยังคงมีชีวิตอยู่ในนรกขุมนี้

# แรงโน้มถ่วงที่บิดเบี้ยว
#### // FILE: DISTORTED_GRAVITY
---

ผมหอบหายใจ กัดฟันแน่นจนได้ยินเสียงบดเบียดของรหัสขากรรไกร ฤทธิ์ตกค้างของมัลแวร์ **Clarity**—ยาเสพติดที่บังคับโอเวอร์คล็อกร่างอวตาร—กำลังแผดเผาผมเป็นระลอกสุดท้าย

ระบบประสาทดิ้นรนจะเค้นภาพระดับ High-Def ท่ามกลางสภาพแวดล้อมที่ไร้ทรัพยากร วิสัยทัศน์ของผมสว่างวาบเป็นสีขาวโพลน รายละเอียดของฝุ่นข้อมูลทุกเม็ดคมชัดจนบาดตา ก่อนจะดับวูบและกระตุกกลับเป็นเม็ดพิกเซลหยาบๆ ความผันผวนของการประมวลผลกระชากกระเพาะจำลอง ผมโก่งคออาเจียนสายบรรทัดโค้ดขุ่นคลั่กและเศษซากสคริปต์ที่พังทลายลงบนพื้น

กฎข้อแรกของการรอดชีวิตคือต้องหลบซ่อน ผมพยายามก้าวเดิน ส้นเท้าขวาที่ไร้การรองรับทำให้ทุกจังหวะลงน้ำหนักคือการตกลงสู่หุบเหว ผมลากซีกขวาของร่างกายไปตามถนนบิดเบี้ยว ปลายเท้าขูดกับเหลี่ยมเรขาคณิตจนเกิดประกายไฟแจ้งเตือนสีแดงลุกโชน

ผมล้มลุกคลุกคลานอยู่ในเงามืดของรอยร้าว แรงดึงดูดวิปลาสพร้อมจะพลิกแผ่นดินให้กลายเป็นเพดาน สัญญาณรบกวน (Static Noise) ดังหึ่งอยู่ในหู ผมบีบร่างตัวเองให้ลีบเล็กที่สุด ซุกตัวอยู่ใต้ซากปรักหักพังของรหัสผ่านที่เสื่อมสภาพ เป็นเพียงเศษเสี้ยวความละเอียดที่รอวันเน่าเปื่อย ท่ามกลางเสียงกระซิบของโครงสร้างที่กำลังผุพัง
