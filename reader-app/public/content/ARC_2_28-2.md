# บทที่ 28: ความทรงจำที่ถูกขโมย (Part B)

ผมลากส้นเท้าขวาที่แหว่งวิ่น วิ่งกะเผลกไปตามทางเดินที่บิดเบี้ยว ร่างของชายคนหนึ่งทรุดตัวอยู่ข้างกำแพง ร่างกายของเขากระตุกอย่างรุนแรง ศีรษะของเขามีรอยแตกร้าวที่เปล่งแสงสีแดงกะพริบ 

เมื่อผมพยายามจะใช้มือขวาที่ยังสมบูรณ์เข้าไปประคอง ความรู้สึกเย็นเยียบก็แล่นวาบผ่านกระดูกสันหลังจำลองของผม มันคือเศษซากของคลื่นกระแทกที่โจรทิ้งไว้ มันพุ่งทะลุเข้าปะทะก้อน Payload กลางอกจนชีพจรข้อมูลของผมเต้นผิดจังหวะ และในเสี้ยววินาทีนั้น ผมรู้สึกถึงความว่างเปล่าที่ถูกกระชากออกไปจากหัวของตัวเอง

ภาพจำของสายฝนแรกที่ตกกระทบผิวหนังในโลกที่ผมจากมา... มันหายไป หายไปอย่างสมบูรณ์ ทิ้งไว้เพียงช่องโหว่ทางตรรกะที่ว่างเปล่าราวกับหลุมดำ

ความโกรธ—ความรู้สึกที่รุนแรงและดิบเถื่อนอย่างที่ผมไม่เคยสัมผัสมานานนับตั้งแต่ถูกบีบอัดอารมณ์—ปะทุขึ้นในอก ประสานเข้ากับน้ำหนักที่สั่นสะเทือนของ Payload มันคือความสูญเสียที่ไม่อาจประเมินค่าได้ เศษเสี้ยวของตัวตนผมถูกพรากไป ต่อให้มันเป็นเพียงความทรงจำเล็กๆ แต่มันคือ "ความจริง" เพียงไม่กี่อย่างที่เหนี่ยวรั้งผมไว้ไม่ให้กลายเป็นความว่างเปล่า ท่ามกลางเสียงเตือนระวังภัยของ UX Index ที่เริ่มบีบวงล้อมเข้ามา

ผมมองลงไปที่พื้น คราบสีดำที่ทิ้งรอยไหม้ไว้ไม่ได้หยุดอยู่แค่จุดเดียว แต่มันหยดเป็นทาง ทิ้งรอยเท้าที่เกิดจากคลื่นรบกวน (Static noise) ซึ่งกำลังดูดกลืนความละเอียดของพื้นผิวที่มันเหยียบย่ำ

ผมกัดฟันแน่น ปล่อยให้ความโกรธขับเคลื่อนรหัสของผม เมินเฉยต่อส้นเท้าที่พังทลาย และเริ่มออกวิ่งลากขาตามรอยเท้านั้นไป
