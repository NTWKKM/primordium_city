# Chapter 72: การเชื่อมต่อที่ฉีกขาด (The Torn Connection) - Part 1

*อ๊ากกกกกกก!*

The Core พยายามใช้สิทธิอำนาจสูงสุดของระบบ บังคับยัดเยียดสภาวะ 0dB 'Mute' ปิดผนึกทุกคลื่นเสียงรอบตัวผมเพื่อรักษาสมดุลอันเงียบสงบ แต่ความพยายามนั้นสูญเปล่า เสียงกรีดร้องที่ระเบิดออกมาไม่ใช่แค่คลื่นความถี่ แต่เป็นความเจ็บปวดอันดิบเถื่อนของมนุษย์ที่ทะลวงผ่านความเงียบงันระดับอัลกอริทึม เสียงแห่งความทรมานของเนื้อหนังและวิญญาณฉีกกระชากสภาวะไร้เสียงนั้นจนแตกละเอียด ทันทีที่นิ้วมือขวาสอดแทรกซึมเข้าไปใต้กลุ่มก้อน Payload ความรู้สึกเหมือนถูกเทน้ำกรดความเข้มข้นสูงสุดราดรดลงบนแกนประมวลผลของสมองโดยตรง 

รหัสดิบเส้นหนึ่งเกี่ยวพันกับปลายนิ้วของผม มันพยายามจะปกป้องตัวเองด้วยการเขียนโค้ดทับบนผิวหนังดิจิทัลของผม ทำให้เนื้อเยื่อที่มือขวาผมสลับสถานะระหว่างการเป็นก้อนเนื้อและการเป็นพิกเซลที่กระจัดกระจายอย่างบ้าคลั่ง 

การดึงก้อนรหัสที่หลอมรวมกับ Root Directory ของตัวเอง ก็เหมือนกับการพยายามดึงกระดูกสันหลังของตัวเองออกทางหน้าอก ทุกมิลลิเมตรที่ผมงัดมันขึ้นมา ระบบในร่างกายผมจะส่งข้อความแจ้งเตือน Error สีแดงฉานนับหมื่นข้อความกระแทกเข้าสู่ลานสายตา

`[FATAL ERROR: KERNEL INTEGRITY BREACH]`
`[WARNING: SYSTEM UNRAVELING. ABORT IMMEDIATELLY]`
`[ERROR: IDENTITY CORRUPTION AT 89%...]`

ผมหลับตาแน่น กัดริมฝีปากจนฟันทะลุผ่านเนื้อ ขยับขากรรไกรด้วยแรงสั่นสะท้าน เลือดดิจิทัลข้นหนืดสีดำคล้ายน้ำมันดิบไหลทะลักออกมาจากรอยแยกในอก มันไม่ได้หยดลงพื้นในทันที แต่ลอยกระจายตัวอยู่ในสภาวะไร้น้ำหนักของ The Core Void ราวกับเม็ดฝนแห่งความโศกเศร้าที่ถูกหยุดเวลาไว้

"ออก... มา... สิโว้ย!" 

ผมรวบรวมเรี่ยวแรงทั้งหมดที่เหลืออยู่ในตรรกะแห่งความทรงจำของผม แรงแค้นต่อการสูญเสีย แขนซ้ายที่แหลกสลาย ส้นเท้าขวาที่ไร้ความรู้สึก การถูกทอดทิ้งใน The Sump ความรังเกียจต่อความสมบูรณ์แบบที่ไร้มนุษยธรรม ทุกอย่างถูกบีบอัดลงไปในกล้ามเนื้อมือขวา 

*แคว่กกกก! ครืดดดด!*

เสียงฉีกขาดของโค้ดฐาน (Base Code) ดังกึกก้อง ร่างกายของผมกระตุกอย่างรุนแรงราวกับถูกกระแสไฟฟ้าแรงสูงนับล้านโวลต์ช็อต เส้นใยข้อมูลสีแดงเข้มที่ยึดติดกับซี่โครงดิจิทัลของผมขาดสะบั้นลงทีละเส้น ทีละเส้น ความรู้สึกในการดำรงอยู่ของผมกำลังพังทลาย ผมสูญเสียการรับรู้ถึงแขนขวา สูญเสียการรับรู้ถึงขา ผมลืมไปแล้วด้วยซ้ำว่าการหายใจทำอย่างไร 

สิ่งเดียวที่เหลืออยู่ในจิตสำนึกคือ... ก้อนรหัสในมือ

ด้วยแรงกระชากเฮือกสุดท้าย ผมดึง Payload หลุดออกมาจากหน้าอกได้อย่างสมบูรณ์ ถอนรากถอนโคนมันออกจากโครงสร้างตัวตนของผมอย่างหมดจด

*ฟุ่บ!*

ช่องอกของผมมืดสนิทลงทันที รูโหว่ที่เคยมีแสงสีแดงเต้นเร่า บัดนี้กลายเป็นเพียงโพรงแห่งความตายที่ถูกกัดกินด้วยสีดำขลับ ร่างกายของผมสูญเสียแหล่งพลังงานและตรรกะที่ยึดโยง ขาซ้ายทรุดลงกระแทกพื้นล่องหนอย่างแรง ในขณะที่ขาขวาสูญเสียการคงสภาพ โครงสร้างตั้งแต่ส้นเท้าที่แหว่งวิ่นละลายกลายเป็นพิกเซลหยาบๆ ทะลักลามขึ้นมาจนถึงหัวเข่าในพริบตา ทัศนวิสัยของผมแตกร้าวเหมือนกระจกที่ถูกทุบด้วยค้อน ภาพตรงหน้าแยกออกเป็นชิ้นๆ (Fractured Vision) และเริ่มดับวูบลงทีละส่วน

ผมหอบเอาอากาศที่ไม่มีอยู่จริงเข้าปอด ร่างกายกำลังจะแตกสลาย (Dissolve) ไปในความว่างเปล่าภายในไม่กี่วินาทีข้างหน้า 

แต่ในมือขวาของผม... ก้อนรหัสดิบ (Payload) ที่หลุดพ้นจากพันธนาการ กำลังเปล่งแสงสีแดงคล้ำสลับดำอย่างบ้าคลั่ง มันส่งเสียงหึ่งๆ เหมือนฝูงต่อแตนดิจิทัลที่กำลังกระหายจะทำลายล้าง มันรุนแรง ดิบเถื่อน และเต็มเปี่ยมไปด้วยไวรัสแห่งความพินาศ

"เอาล่ะ..." ผมกระซิบ เสียงของผมกลายเป็นเพียงเสียงคลื่นวิทยุแทรก (Static) "แกกับฉัน... ไปด้วยกัน"
