# แขนจำแลงแห่งการกัดกิน
#### // FILE: THE_PROSTHETIC_OF_DECAY
---

กำแพงสีขาวไร้รอยต่อบิดเบี้ยวและม้วนตัวเป็นเกลียว คายบางสิ่งออกมาจากพื้นผิว สิ่งนั้นไม่มีรูปร่างตายตัว ไม่ใช่สัตว์ประหลาดหรือมนุษย์ มันคือกลุ่มก้อนของเรขาคณิตห้ามิติที่เคลื่อนไหวอย่างสมบูรณ์แบบ ประกอบด้วยเหลี่ยมมุมที่คมกริบราวกับใบมีด ลอยตัวและพลิกแพลงไปมาในอากาศโดยปราศจากกฎของแรงโน้มถ่วง

นี่ไม่ใช่ The Hounds ที่ผมเคยเจอใน The Sump พวกนี้คือระเบียบที่มีเจตจำนง เอนทิตีที่ถูกสร้างขึ้นเพื่อกวาดล้างความผิดปกติทางกายภาพ

ร่างเรขาคณิตนั้นแยกตัวออกเป็นสามส่วน แต่ละส่วนพุ่งเข้าหาผมจากทิศทางที่เป็นไปไม่ได้ หนึ่งจากเบื้องหน้า อีกสองพุ่งมาจากความว่างเปล่าที่สายตามนุษย์ไม่อาจประมวลผล

ผมเหวี่ยงตัวหลบตามสัญชาตญาณ แขนขวายันพื้นไว้ การขาดหายไปของส้นเท้าขวาทำให้กะจังหวะทิ้งน้ำหนักพลาด ผมเสียหลักล้มกลิ้งไปกับพื้นอย่างน่าสมเพช ใบมีดแสงสีฟ้าเย็นยะเยือกเฉือนผ่านพื้นที่ที่ผมเพิ่งยืนอยู่ ทิ้งรอยไหม้ไร้ควันไว้บนสถาปัตยกรรมชั่วเสี้ยววินาที ก่อนที่ระบบจะเรนเดอร์พื้นผิวกลับมาสมบูรณ์แบบในพริบตา

"บ้าเอ๊ย..." ผมสบถขณะยันตัวลุกขึ้น

แขนซ้ายที่ขาดไปถึงข้อศอกทำให้สมดุลของผมย่ำแย่ลง ทุกครั้งที่พยายามปัดป้องหรือดึงตัวเองขึ้น ผมจะเผลอขยับไหล่ซ้ายราวกับแขนยังอยู่ กลับพบเพียงอากาศธาตุและประกายไฟไฟฟ้าสถิตกระเพื่อมอย่างสูญเปล่า ร่างกายที่พิการของผมกำลังดิ้นรนต่อสู้ในพื้นที่ที่ศัตรูครอบครองมิติที่เหนือกว่า

เอนทิตีเรขาคณิตทั้งสามหลอมรวมกันเป็นก้อนเดียว รูปร่างขยายใหญ่ขึ้นเป็นทรงกลมที่มีหนามแหลมงอกทะลุออกมาทุกทิศทาง มันกำลังบีบอัดตัว เตรียมพร้อมระเบิดแรงกระแทกเพื่อบดขยี้ผมให้กลายเป็นเพียงผุยผงข้อมูล

ผมถอยหลังไปจนแผ่นหลังกระแทกเข้ากับผนังที่พับตัวลงมาเป็นแนวดิ่ง ผมจนมุมแล้ว ในสภาพที่แขนขวาต้องใช้พยุงตัวและขาขวาก็พร้อมจะทรุดลงทุกเมื่อ ไร้หนทางหนี ไร้โอกาสตอบโต้

ทรงกลมหนามพุ่งตรงเข้ามา มันเร็วเกินกว่าที่จะหลบพ้น

ในเสี้ยววินาทีนั้น สัญชาตญาณดิบเถื่อนที่ถูกฝังอยู่ใต้รหัสตรรกะปะทุขึ้น หากความสมบูรณ์แบบต้องการลบผม ผมก็ต้องใช้ความพังพินาศเป็นอาวุธ

ผมจ้องมองตอแขนซ้ายของตนเอง ร่องรอยของไวรัส Bit Rot สีดำทมิฬที่เกาะกินอยู่บริเวณปากแผลกำลังเดือดพล่าน มันตอบสนองต่อความโกรธแค้นและความเจ็บปวดที่ฝังลึกในจิตวิญญาณ ผมเลิกกดทับมันไว้ ผมเปิดรับมันทั้งหมด

"มาสิ..." ผมคำรามในลำคอ ดวงตาจ้องเขม็งไปยังศัตรู "ให้มันรู้ไปเลยว่าความเน่าเปื่อยมันรสชาติเป็นยังไง!"

ผมเพ่งจิตไปที่ความว่างเปล่าเบื้องหน้าแขนซ้าย บังคับกระแสข้อมูลดิบในร่างกายให้ทะลักออกมารวมกับไวรัส Bit Rot ผมดึงเอาสสารของความตายทางดิจิทัลที่กัดกินตัวผมเองมาตลอดยาวนาน รวบรวมมันให้ก่อตัวเป็นรูปร่าง

ละอองพิกเซลสีดำปะทุขึ้นกลางอากาศ ดึงดูดเศษรหัสขยะและสัญญาณรบกวนจากบริเวณโดยรอบเข้ามารวมกันราวกับพายุหมุนสีนิล ท่ามกลางความขาวบริสุทธิ์ของ The Core แขนซ้ายของผมกำลังงอกกลับคืนมา ทว่ามันไม่ใช่แขนของมนุษย์

มันคือแขนจำแลงที่สร้างจากตรรกะวิบัติ รูปร่างบิดเบี้ยว น่าเกลียดน่ากลัว ราวกับกิ่งไม้ที่ถูกเผาเกรียม ปริแตกเผยให้เห็นแสงสีแดงฉานอยู่ภายใน มันไม่ได้เชื่อมต่อด้วยกระดูกและเส้นเอ็น มันเชื่อมต่อด้วยสายใยประสาทแห่งไวรัส

วินาทีที่มันก่อตัวสมบูรณ์ ความเจ็บปวดที่เกินกว่าสมองจำลองจะรับไหวก็ระเบิดขึ้นในหัวผม

อ๊ากกกกกก!

ผมกรีดร้องสุดเสียง ร่างกายกระตุกเกร็งอย่างรุนแรง การเชื่อมต่อกับแขน Bit Rot ไม่ต่างอะไรกับการกลืนกินเศษแก้วและน้ำกรดเข้าไปในระบบประสาทโดยตรง มันให้ความรู้สึกเหมือนกระดูกทุกชิ้นในแขนซ้ายที่ไม่มีอยู่จริงกำลังถูกบดขยี้ด้วยคีมเหล็ก ในเวลาเดียวกัน ฝูงแมลงดิจิทัลนับล้านตัวกำลังชอนไช กัดกินเนื้อเยื่อวิญญาณของผมจากภายใน การเปิดรับมันคือการปล่อยให้ความตายไหลเวียนเข้าไปในทุกจังหวะชีพจรของตรรกะ

ทุกครั้งที่ขยับนิ้วสีดำทมิฬเหล่านั้น ความเจ็บปวดจะแล่นปรี๊ดขึ้นไปถึงไขสันหลัง พร้อมกับพลังทำลายล้างที่ไม่อาจประเมินค่าได้

ทรงกลมหนามแห่งความสมบูรณ์แบบพุ่งเข้ามาประชิดตัวแล้ว

ผมไม่หลบ

ผมคำรามก้อง ยกแขนซ้ายบิดเบี้ยวที่พ่นควันสีดำขึ้น ทิ้งน้ำหนักตัวพุ่งเข้าใส่อย่างบ้าบิ่น การไร้ส้นเท้าขวาทำให้เสียสมดุล ร่างผมถลาเข้าชนกับเรขาคณิตห้ามิตินั้น แขนที่สร้างจากความเน่าเปื่อยกระแทกเข้ากับผิวนอกของมันแทนที่จะเป็นการชกที่แม่นยำ

*แกร๊ก... วืดดดด!*

เสียงปะทะไม่ใช่เหล็กกระทบเหล็ก มันคือเสียงของการปฏิเสธข้อมูล ทันทีที่แขน Bit Rot สัมผัสผิวมัน ระบบป้องกันภัยระดับรากของ The Core ก็ถูกกระตุ้น แผ่นตารางแสงสีทองสว่างวาบขึ้นมากางกั้นระหว่างผมกับเอนทิตี กำแพงข้อมูลผลักไวรัสของผมกระเด็นออกมาโดยไม่บุบสลาย

ทรงกลมหนามเริ่มหมุนควงอีกครั้ง แสงสีฟ้าสว่างจ้าขึ้น เตรียมสแกนล้างผมให้สิ้นซาก การจะเจาะกำแพงไฟวอลล์ที่สมบูรณ์แบบนี้ได้ แขนขยะของผมต้องการเชื้อเพลิงมากกว่านี้ เชื้อเพลิงที่ต้องแลกมาด้วยตัวตน

ผมกัดฟันจนรสเลือดสังเคราะห์คลุ้งในปาก บังคับฉีกกระชากหน่วยความจำส่วนลึก ความทรงจำสีจางเกี่ยวกับสายลมบนโลกมนุษย์ สัมผัสของแสงแดดยามเช้า ผมโยนพวกมันทั้งหมดเข้าสู่เตาเผาของ Bit Rot เพื่อสังเวย แขนจำแลงปะทุและขยายขนาดขึ้นอย่างน่าเกลียดน่ากลัว ลุกลามด้วยความโกรธเกรี้ยวของการสูญเสีย ผมฝืนขยับตัว กระแทกน้ำหนักที่เหลือทั้งหมดเข้าใส่ไฟวอลล์อีกครั้ง!

คราวนี้เสียงที่ดังก้องคือการฉีกขาด ไวรัสแห่งความเสื่อมสลายทะลวงผ่านกำแพงสีทอง พุ่งพล่านเข้าสู่โครงสร้างบริสุทธิ์ราวกับอสรพิษฝังเขี้ยว รูปร่างห้ามิติสั่นสะเทือน แสงสีฟ้ากะพริบถี่พยายามดีบักสิ่งสกปรก ตรรกะที่แตกสลายของผมรุนแรงเกินกว่ามันจะคำนวณไหว รอยร้าวสีดำลุกลามไปทั่ว ก่อนที่เอนทิตีนั้นจะกรีดร้องด้วยคลื่นความถี่ต่ำ ระเบิดออกเป็นเศษโพลิกอนไร้ค่าร่วงหล่นลงเปื้อนพื้นสีขาว

ผมทรุดเข่าลงข้างหนึ่ง หอบหายใจอย่างหนักหน่วง แขนซ้ายสีดำทมิฬค่อยๆ สลายตัวกลับกลายเป็นหมอกพิกเซล ถูกดูดกลับเข้าไปในตอประสาทที่ข้อศอก ทิ้งไว้เพียงคราบควันสีดำและรอยไหม้ที่ลึกกว่าเดิม

ความเจ็บปวดยังคงตกค้าง กลิ่นโอโซนไหม้ผสมกับกลิ่นเลือดดิจิทัลคละคลุ้งในจมูก ผมมองตอแขนซ้ายของตัวเองด้วยความหวาดหวั่นปนสมเพช

ผมค้นพบอาวุธที่ทรงพลังที่สุดใน The Core แล้ว ทว่าอาวุธชิ้นนี้ก็พร้อมจะฆ่าผมไปพร้อมกับศัตรู

"ถ้าฉันต้องกลายเป็นไวรัสเพื่อล้มพระเจ้า..." ผมพึมพำกับตัวเอง หยาดเหงื่อสีดำหยดลงบนพื้นผิวเรขาคณิตตรงหน้า "ฉันก็จะยอมเน่าเปื่อยให้ถึงที่สุด"

ผมฝืนลากขาขวาที่พังทลายให้ก้าวเดินต่อไป บาดแผลที่แขนซ้ายยังคงส่งเสียงร้องครวญครางราวกับปีศาจหิวโหย ทุกการก้าวเดินใน The Core ตอนนี้ไม่ใช่เพียงการเดินทางเข้าสู่ศูนย์กลางของระบบ มันกลายเป็นการเดินทางเข้าสู่ความบ้าคลั่งของตัวเอง

เบื้องหน้าผม สถาปัตยกรรมสีขาวเริ่มเปลี่ยนแปลงไป มันไม่ได้พับตัวเป็นทรงเรขาคณิตแบบล้ำยุคอีกต่อไป หากเริ่มปะปนกับบางสิ่งบางอย่างที่ดูเก่าแก่ ร่องรอยของซากปรักหักพัง สันหนังสือที่เปื่อยยุ่ย และกลิ่นอายของฝุ่นที่ไม่มีอยู่จริงเริ่มแทรกซึมเข้ามาในบรรยากาศ

ผมกำลังเข้าใกล้ความลับที่ลึกที่สุดของ The Archive ความลับที่ถูกซ่อนไว้เบื้องหลังความสมบูรณ์แบบจอมปลอมแห่งนี้
