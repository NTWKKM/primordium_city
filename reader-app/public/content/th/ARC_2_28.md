# บทที่ 28: ความทรงจำที่ถูกขโมย (Part A)

เสียงแรกที่กระแทกโสตประสาทไม่ใช่ระเบิด ทว่าเป็นการฉีกขาด—เหมือนผ้าใบผืนมหึมาถูกกรีดทะลวงด้วยคมมีดที่มองไม่เห็น คลื่นความสั่นสะเทือนทางข้อมูลอัดกระแทกเข้าที่หน้าอก แรงปะทะนั้นชนเข้ากับก้อน Payload จนมันส่งเสียงสะท้อนกังวานหนักอึ้งราวระฆังเหล็ก ผมทะลึ่งพรวดขึ้นด้วยความตื่นตระหนก ก่อนจะเสียหลักล้มกระแทกพื้นเพราะส้นเท้าขวาที่แหว่งหายปฏิเสธการรับน้ำหนัก

ความละเอียดรอบตัวกะพริบติดดับ กลิ่นอับของข้อมูลที่ถูกบีบอัดผสมกลิ่นโอโซนไหม้เกรียมคละคลุ้งระเบียงทางเดินแคบๆ ของเขต Mid-Sectors

พื้นคอนกรีตสังเคราะห์ปรากฏรอยแยก ทิ้งคราบข้อมูลสีดำสนิทที่เดือดปุดๆ คล้ายกรดกัดกร่อนรหัสพื้นฐาน

แล้วเสียงร้องไห้ก็ดังขึ้น มันไม่ใช่เสียงสะอื้นของมนุษย์ มันคือเสียงที่ถูกบีบอัดจนสูญเสียย่านความถี่ต่ำ เหลือเพียงเสียงหวีดแหลมเสียดแทงแก้วหู—เสียงกรีดร้องของจิตสำนึกที่ถูกกระชากบางสิ่งไปอย่างกะทันหัน

"เกิดอะไรขึ้น!" ผมตะโกน น้ำเสียงตัวเองแตกพร่า ร่างกายดิจิทัลตอบสนองต่อความตื่นตระหนกด้วยการสาดพิกเซลแตกซ่าน (Artifacts) ที่ปลายนิ้ว แขนซ้ายที่ติดเชื้อ Bit Rot กระตุกและชาหนึบ หนักอึ้งจนแทบยกไม่ขึ้น

มีคนถูกปล้น สิ่งที่หายไปไม่ใช่ทรัพย์สิน มันคือความทรงจำ การโจรกรรมข้อมูลความทรงจำครั้งใหญ่เพิ่งอุบัติขึ้นใจกลางชุมชน ผู้คนวิ่งพล่านด้วยความสับสน ร่างกายของบางคนกะพริบติดดับราวไฟนีออนใกล้ขาดใจ เสียงสแกนความถี่ต่ำของ UX Index ดังก้องในหัวผม มันสัมผัสถึงความโกลาหลและกำลังพุ่งเป้าค้นหา 'ความผิดปกติ' ในรัศมีนี้


# บทที่ 28: ความทรงจำที่ถูกขโมย (Part B)

ผมลากส้นเท้าพิการ วิ่งกะเผลกไปตามทางเดินบิดเบี้ยว ร่างของชายคนหนึ่งทรุดฮวบอยู่ข้างกำแพง ร่างกายเขากระตุกอย่างรุนแรง ศีรษะมีรอยแตกร้าวเปล่งแสงสีแดงกะพริบ 

เมื่อผมยื่นมือขวาที่ยังสมบูรณ์เข้าไปประคอง ความเย็นเยียบแล่นวาบผ่านกระดูกสันหลังจำลอง มันคือเศษซากคลื่นกระแทกที่หัวขโมยทิ้งไว้ คลื่นนั้นพุ่งทะลุเข้าปะทะก้อน Payload กลางอกจนชีพจรข้อมูลเต้นผิดจังหวะ และในเสี้ยววินาทีนั้น ผมสัมผัสถึงความว่างเปล่าที่ถูกกระชากหลุดจากหัวตัวเอง

ภาพจำของสายฝนแรกที่ตกกระทบผิวหนังในโลกที่ผมจากมา... มันระเหยหาย หายไปอย่างสมบูรณ์ ทิ้งไว้เพียงช่องโหว่ทางตรรกะที่มืดมิดราวกับหลุมดำ

ความโกรธ—ความรู้สึกดิบเถื่อนและรุนแรงอย่างที่ผมไม่เคยสัมผัสมานานตั้งแต่ถูกบีบอัดอารมณ์—ปะทุเดือดในอก ประสานเข้ากับน้ำหนักสั่นสะเทือนของ Payload มันคือความสูญเสียที่ไม่อาจประเมินค่า เศษเสี้ยวตัวตนถูกพรากไป แม้เป็นเพียงความทรงจำเล็กๆ ทว่ามันคือ "ความจริง" ไม่กี่อย่างที่เหนี่ยวรั้งผมจากความว่างเปล่า ท่ามกลางเสียงไซเรนเตือนภัยของ UX Index ที่เริ่มบีบวงล้อม

ผมก้มมองพื้น คราบสีดำที่ทิ้งรอยไหม้ไม่ได้หยุดแค่จุดเดียว มันหยดเป็นทาง ทิ้งรอยเท้าที่ก่อเกิดจากคลื่นรบกวน (Static noise) ซึ่งกำลังดูดกลืนความละเอียดของพื้นผิวที่มันเหยียบย่าง

ผมกัดฟันกรอด ปล่อยให้ความโกรธขับเคลื่อนรหัสในร่าง เพิกเฉยต่อส้นเท้าที่พังทลาย และออกวิ่งลากขาตามรอยเท้าแห่งความบิดเบี้ยวนั้นไป
