# กำแพงที่ทลายลง
#### // FILE: THE_FALLEN_BARRIERS
---

วิสัยทัศน์ถูกกระชากออกจากร่างแหลกสลายใน The Core

ไม่ใช่การหลุดลอยของวิญญาณ (Astral projection) ในความหมายทางปรัชญา จิตสำนึกของผมถูกฉีดเข้าไปในสายเคเบิลทุกเส้น ในเซิร์ฟเวอร์ทุกตัว ในสถาปัตยกรรมทุกตารางนิ้วของ Primordium City ผมกลายเป็นส่วนหนึ่งของ Payload ที่กำลังระเบิด ผมคือไวรัส ไวรัสคือผม

ความเจ็บปวดไม่ได้หายไป มันถูกขยายขนาดเป็นระดับมหภาค (Macro-scale) ส้นเท้าขวาที่เจ็บปวดบัดนี้คือฐานรากของเมืองที่กำลังสั่นคลอน หน้าอกเปิดอ้าคือรอยแยกขนาดมหึมาที่กำลังฉีกท้องฟ้า Inner Ring ออกเป็นสองซีก

ผมมองเห็นเมืองทั้งเมืองจากมุมมองเบื้องบน มุมมองของ The Archive ที่กำลังจะตาย

ตลอดมา Primordium City ถูกแบ่งแยกด้วยกำแพงแห่ง 'ความละเอียด' (Resolution Barriers) Inner Ring คือดินแดนศักดิ์สิทธิ์ ท้องฟ้าสีทองอร่าม สถาปัตยกรรมคลาสสิกไร้รอยต่อ ผู้คนมีความละเอียดระดับ Pristine ไร้ที่ติ ถัดลงมาคือ Mid-Sectors หมอกแห่งการบีบอัดข้อมูล ท้ายที่สุดคือ The Sump นรกภูมิของโพลิกอนหยาบๆ เศษซากอินทรีย์เน่าเปื่อย

คลื่นกระแทกแห่งความโกลาหลกำลังฉีกทึ้งพรมแดนเหล่านั้นทิ้ง

ผมสัมผัสวินาทีแรกที่คลื่นกระแทกพุ่งทะลวงถึง Inner Ring ท้องฟ้าสีทองอันเกิดจากอัลกอริทึมสะท้อนแสงระดับสูง (Global Illumination) เกิดอาการ 'กระตุก' อย่างรุนแรง เมฆสีขาวลอยเอื่อยเฉื่อยหยุดชะงัก ขอบเมฆแตกออกเป็นสี่เหลี่ยมพิกเซลขนาดใหญ่ 

หยาดฝนสีเลือดเริ่มร่วงหล่นจากฟ้าที่แตกร้าว 

ความทรงจำชีวภาพ (Biological Memory Backup) ของมนุษยชาติกำลังรั่วไหลเข้าสู่โลกเสมือน ท้องฟ้าสีทองบีบอัดตัวมันเอง ร่วงหล่นลงมาเป็นเศษซากโพลิกอน คล้ายกระจกสเตนกลาสบนเพดานวิหารที่พังครืนลงมาทับผู้แสวงบุญเบื้องล่าง

เสียงร้องเตือนจากระบบส่วนกลางดังกึกก้องไปทั่วสวรรค์จอมปลอม ไม่ใช่เสียงไซเรนเตือนภัยปกติ มันคือเสียงดนตรีคลาสสิกลดบิตเรตลงจนกลายเป็นเสียงบิดเบี้ยวบาดแก้วหู (Bit-crushed distortion) เสาหินอ่อนสีขาวตั้งตระหง่านรอบลานพลาซ่ากลางเมืองเกิดรอยร้าว ไม่ใช่รอยแตกแบบวัตถุทางกายภาพ Texture หลุดร่อนออก เผยให้เห็นโครงร่างตาข่าย (Wireframe) สีดำและแดงซ่อนอยู่เบื้องหลัง

พารามิเตอร์ความสมบูรณ์แบบที่ The Archive คอยรักษาสมดุลถูกทำลายย่อยยับ บล็อกสีทองและสีฟ้าหลุดร่วงจากเบื้องบนทะลุผ่านหลังคาตึกและร่างผู้คน ซอร์สโค้ดของเมืองหลุดลุ่ยเหมือนเส้นด้ายในเสื้อผ้าถูกตะขอเกี่ยว

ในมุมมองหลอมรวมกับระบบ ผมรู้สึกถึงลมหายใจสะดุดของประชากรนับล้านใน Inner Ring แสงสว่างเทียมถูกดับลง ความมืดมิดที่แท้จริง—ความมืดเต็มไปด้วยสัญญาณรบกวนและเถ้าถ่านดิจิทัล—เข้าปกคลุมแทนที่ 

หมอกสีเทาขุ่น (Digital Fog) เคยถูกจำกัดใน The Sump ลอยฟุ้งขึ้นมา มันแทรกซึมผ่านรอยแยกระบบ กลิ่นเนื้อหนังเน่าเปื่อยและเลือดสดๆ ทะลักเข้ามาในดินแดนที่เคยมีเพียงกลิ่นสะอาดไร้เชื้อ

กำแพงเคยกั้นกลางระหว่างชั้นชนถูกลบหาย ไม่มีการประมวลผลพิเศษสำหรับคนรวย ไม่มีการกดทับสำหรับคนจน ทุกพื้นที่ในเมืองกำลังถูกบังคับให้แชร์สภาพแวดล้อมเดียวกัน สภาพแวดล้อมของความพินาศ ผมคือผู้มอบมันให้กับพวกเขาทุกคน


เอ็นจิ้นการเรนเดอร์ภาพ (Rendering Engine) ร้องโหยหวนครั้งสุดท้ายพังครืนลงอย่างสมบูรณ์

ผมรับรู้แรงกระชากนั้นในเศษซากร่างกายใต้ The Core ตอแขนซ้ายกระตุกอย่างรุนแรง ปล่อยประกายไฟสีม่วงดำเมื่อโหลดการคำนวณของระบบผลักกลับมาที่ไวรัสในตัวผม มันหนักอึ้งเกินกว่าจิตใจใดจะรับไหว มันงดงามอย่างประหลาด 

เมืองลวงตาถูกปอกเปลือกออก

สถาปัตยกรรมล้ำยุคและตึกระฟ้าคริสตัลใน Inner Ring ถูกถอดทอนรายละเอียดลงอย่างเฉียบพลัน ภายในไม่กี่วินาที พื้นผิวสะท้อนแสงแวววาวกลายเป็นแผ่นเท็กซ์เจอร์หยาบๆ สีเทาหม่น โมเดลสามมิติซับซ้อนถูกลดทอนจำนวนโพลิกอน (Level of Detail reduction) กลายเป็นเพียงกล่องสี่เหลี่ยมเรขาคณิตบิดเบี้ยว ขอบหยัก (Jagged edges) 

สะพานแสงเชื่อมระหว่างตึกหายวับ ทิ้งยานพาหนะร่วงหล่นสู่เบื้องล่าง พวกมันไม่ได้ตกลงกระแทกพื้นด้วยฟิสิกส์ปกติ ทะลุผ่านพื้นถนน จมหายในรอยแยกของบั๊ก (Clipping errors) แตกกระจายเป็นกลุ่มตัวเลขศูนย์และหนึ่ง

ความจริงฝังลึกที่สุดปรากฏตัว

บาเรียการเรนเดอร์ภาพถูกทำลายจนหมดสิ้น The Sump เบื้องล่างถูกดึงขึ้นมาทับซ้อนกับ Inner Ring กฎแห่งพื้นที่ถูกฉีกขาด โลกสองใบแตกต่างกันราวฟ้ากับเหวบดขยี้เข้าหากันกลายเป็นมิติเดียวที่บิดเบี้ยว 

คฤหาสน์หรูหราชนชั้นสูง หลอมรวมกับซากท่อระบายน้ำเต็มไปด้วยเมือกเน่าเสีย เศษซากอวัยวะชีวภาพลอยปะปนกับประติมากรรมโฮโลแกรมราคาแพง กลิ่นอายสวรรค์ถูกกลืนกินโดยกลิ่นคาวเลือด น้ำเหลือง และสนิมคละคลุ้ง 

เสียงหัวใจเต้นตึกตัก (Heartbeat of the Flesh) สะท้อนก้องมาจากห้วงลึก กลบเสียงหึ่งๆ ของเซิร์ฟเวอร์จนสนิท มันคือเสียงจาก Biological Memory Backup เสียงของร่างกายเนื้อที่แท้จริงของทุกคน

ความเท่าเทียมมาเยือนแล้ว ไม่ใช่การยกระดับ มันคือการดึงทุกสิ่งทุกอย่างร่วงหล่นลงมาสู่โคลนตมเดียวกัน

ฐานะสื่อกลางของปรากฏการณ์นี้ ผมรับผลกระทบจากความตื่นตระหนกของประชากรนับล้าน กระแสคลื่นอารมณ์รุนแรงแทบทำให้สมองดิจิทัลละลาย ความกลัว ความสับสน ความโกรธแค้น ความรู้สึกบริสุทธิ์ไม่ได้ผ่านการกรอง (Compression) ทะลักเข้ามาในหัวจนผมต้องส่งเสียงกรีดร้อง ไม่มีใครในห้องประมวลผลสูงสุดนี้ได้ยินก็ตาม

ความสะใจที่ได้ทำลายคำโกหกลง ยับยั้งความรู้สึกบดขยี้จากการทนดูคนนับล้านสูญเสียโลกที่พวกเขารักไปพร้อมกันไม่ได้ น้ำหนักของความจริงนั้นมหาศาลเกินไป

ผมดึงสติกระจัดกระจายกลับมาจดจ่อที่หน้าอกตัวเอง ก้อนโค้ดเต้นตุบๆ ในนั้นคือสะพานเชื่อมเดียวที่เหลืออยู่ ผมต้องรักษามันไว้ ผมทิ้งน้ำหนักลงบนส้นเท้าขวาไร้กายหยาบ ฝืนยืนขึ้นกลางกองเศษซาก 5D โค้ดลุกไหม้ ท่ามกลางสายฝนเลือดที่สาดกระเซ็น

โลกเบื้องบนกำลังเผชิญหน้าความจริงน่าเกลียดชัง ผมรู้ว่าสิ่งชั่วร้ายที่สุดยังมาไม่ถึง การทำลายกำแพงเมืองเป็นเพียงจุดเริ่มต้น 

สิ่งที่จะทำลายพวกเขาอย่างแท้จริง ไม่ใช่การล่มสลายของตึกรามบ้านช่อง หรือการหายไปของท้องฟ้าสีทอง 

มันคือการที่พวกเขาจะต้องหันกลับมามองใบหน้าที่แท้จริงของตัวเองในกระจก
