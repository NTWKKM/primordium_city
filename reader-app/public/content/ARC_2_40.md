# บทที่ 40: ประตูสู่ก้นบึ้ง (ตอนที่ 1)

มวลอากาศรอบตัวหนักอึ้งบดขยี้การหายใจ มันไม่ใช่แรงกดดันทางฟิสิกส์ มันคือความหนาแน่นของ Data ทับซ้อนที่ถูกทิ้งขว้าง ผมลากร่างพังทลายมาจนถึงขอบเหวแห่งสภาวะร่วงหล่น โครงสร้างสถาปัตยกรรมเบื้องหน้าคือสิ่งปลูกสร้างที่น่าสะพรึงกลัวที่สุดใน Primordium

ประตูสู่ก้นบึ้ง

ซุ้มประตูขนาดยักษ์ตระหง่านง้ำกลางความมืดมิด รูปแบบ Low-poly Brutalism ปราศจากการประนีประนอม เส้นสายแข็งกระด้าง บล็อกเหลี่ยมมโหฬารไร้การทาสีหรือปรับพื้นผิว ความดิบเถื่อนแผ่ซ่าน บานประตูไม่ได้ก่อร่างจากหินก้อนหรือแผ่นโลหะ มันถูกถักทอขึ้นจากรหัสข้อมูลเน่าเสีย จับตัวเป็นก้อนแข็งสีดำสนิท เส้นเลือดสีแดงเรืองแสงสูบฉีดอยู่ภายในราวกับอวัยวะที่มีชีวิต

อุณหภูมิทางความรู้สึกร่วงหล่นสู่จุดเยือกแข็ง (Absolute Zero) สภาวะว่างเปล่ากลืนกินทุกสรรพสิ่ง Error Message ทุกชนิดหลีกหนีบริเวณนี้ แสงสแกนเนอร์ของ UX Index หรือกฎเกณฑ์จอมปลอมจาก The Archive ถูกสกัดกั้นอย่างสมบูรณ์

แรงดึงดูดมหาศาลแผ่กระจายจากหลังบานประตู มันไม่ใช่แรงโน้มถ่วง แรงดึงดูดทางข้อมูล (Data Gravity) กำลังกระชากโครงสร้างตัวตนของผม พยายามบดขยี้เพย์โหลดในหน้าอกให้แตกสลายเป็นธุลีพิกเซล

ร่างโปร่งแสงจำนวนมากเดินวนเวียนไร้จุดหมายอยู่หน้าประตู พวกเขาคือ "The Hollows" เปลือกข้อมูลกลวงเปล่า ร่างกายบิดเบี้ยวผิดรูป บางคนยื่นมือไขว่คว้าอากาศธาตุซ้ำแล้วซ้ำเล่า บางคนอ้าปากกว้างกรีดร้องไร้เสียงในลูปความทุกข์ทรมานนิรันดร์ พวกเขาคือผลผลิตของการยอมจำนนต่อความเสื่อมสลายอย่างไร้เจตจำนง

การก้าวข้ามธรณีประตูบานนี้ คือการตัดขาดจากแสงสว่างและความละเอียดของนครเบื้องบนอย่างถาวร

ผมก้มมองมือซ้ายของตนเอง มันสั่นกระตุก แตกปริร้าวจากเชื้อร้าย Bit Rot ความหวังที่จะกลับไปยืนหยัดด้วยความสมบูรณ์แบบพังทลายลงสิ้น ผมไม่มีวันกลับไปเป็นประชากร High-Res ได้อีก ถึงกระนั้นผมก็จะไม่ยอมปล่อยให้ตัวเองกลายเป็นเปลือกกลวงเปล่าเหล่านั้น

# บทที่ 40: ประตูสู่ก้นบึ้ง (ตอนที่ 2)

ผมลากขาขวาที่ไร้ส้นเท้า ก้าวฝ่าฝูง The Hollows ที่เดินวนเวียน พวกเขาไร้การตอบสนอง ร่างกายของเราทะลุผ่านกันและกันราวกับหมอกควัน

ยิ่งเข้าใกล้บานประตูก้อนรหัสยักษ์ แรงกดดันทางข้อมูลยิ่งทวีความเกรี้ยวกราด แกนกลางตัวตนถูกบีบอัด เพย์โหลดในหน้าอกเต้นระรัวราวกับจังหวะชีพจรวิกฤต รหัสโครงสร้างในตัวผมพยายามต่อต้านสภาพแวดล้อม หน้าต่างแจ้งเตือนความตายปรากฏขึ้น

`[CRITICAL WARNING: STRUCTURAL INTEGRITY COMPROMISED]`

ผมปัดข้อความนั้นทิ้งอย่างไร้เยื่อใย ประโยชน์ของมันจบสิ้นลงแล้ว ผมสวมกอดสถานะผลงานที่ล้มเหลว ยอมรับตัวตนในฐานะเศษซากที่รอดพ้นการบีบอัดของ The Archive และกรงเล็บของ UX Index

ผมยกมือขวาออกแรงผลักบานประตูโค้ดสีดำสนิท วินาทีที่ปลายนิ้วสัมผัส ความรู้สึกชาหนึบแล่นปราดครอบคลุมทั้งร่าง มันไม่ใช่การออกแรงดันบานพับ มันคือการ "ซิงค์" (Sync) รหัสประจำตัวเข้ากับสถาปัตยกรรมแห่งความพินาศ

กระแสข้อมูลสีแดงที่ไหลเวียนในบานประตูถ่ายเทเข้าสู่ท่อนแขน มันเติมเต็มช่องว่างที่ขาดหาย แขนซ้ายที่เน่าเฟะจาก Bit Rot ตอบสนองต่อจังหวะชีพจรสีเลือด ความเจ็บปวดหลอมรวมกับความเติมเต็ม ผมกำลังดูดกลืนความเน่าเปื่อยนี้เข้ามาเป็นส่วนหนึ่งของวิญญาณ

บานประตูเปิดออกไร้สรรพเสียง เผยให้เห็นภูมิทัศน์ของ The Sump—มหานครแห่งกากเดนข้อมูล อาคารระฟ้าพังครืน ท้องถนนฉีกขาดเป็นรอยต่ออวกาศดิจิทัล พายุ Noise กรีดร้องโหมกระหน่ำอยู่เหนือซากความวิปลาส

มันอัปลักษณ์น่าสะอิดสะเอียน... กลับอัดแน่นด้วยชีวิตชีวายิ่งกว่าความสมบูรณ์แบบจอมปลอมของ Inner Ring

ผมลากขากระเผลกก้าวข้ามธรณีประตู ทิ้งความโหยหาความสมบูรณ์แบบไว้เบื้องหลัง ความละเอียดของภาพรอบตัวดิ่งวูบเหลือเพียงเหลี่ยมมุมหยาบกระด้าง สีสันถูกชะล้างเหลือเพียงความมืดและความขุ่นมัว ท่ามกลางความวิปริตนี้ ผมสัมผัสได้ถึงความจริงแท้

เส้นทางของประชากรแห่ง Primordium สิ้นสุดลงแล้ว การจุติใหม่ในฐานะผู้ถูกกัดเซาะ (The Eroded) เริ่มต้นขึ้น

ประตูสู่ก้นบึ้งปิดสนิทเบื้องหลัง ตัดขาดผมจากโลกเบื้องบนโดยสมบูรณ์

ผมสูดลมหายใจ ลิ้มรสชาติฝุ่นพิกเซลและโอโซนแห่งความล้มเหลว... ก่อนก้าวล่วงลึกลงไปในความมืดมิด พร้อมกุญแจแห่งความลับในแผงอกที่ผมยอมสละสิ้นทุกสิ่งเพื่อปกป้อง
