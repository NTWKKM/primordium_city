# บทที่ 40: ประตูสู่ก้นบึ้ง (ตอนที่ 1)

อากาศรอบตัวหนักอึ้งจนแทบหายใจไม่ออก มันไม่ใช่ความกดอากาศ แต่เป็นความหนาแน่นของ Data ทับซ้อนที่ถูกทิ้งร้าง ผมลากร่างที่พังทลายเดินทางมาถึงสุดขอบของสภาวะร่วงหล่น เบื้องหน้าผมคือโครงสร้างที่ยิ่งใหญ่และน่าสะพรึงกลัวที่สุดเท่าที่ผมเคยเห็นใน Primordium

มันคือ "ประตูสู่ก้นบึ้ง" 

ซุ้มประตูขนาดยักษ์ตั้งตระหง่านอยู่ท่ามกลางความมืดมิด สถาปัตยกรรมของมันเป็นแบบ Low-poly Brutalism—เส้นสายแข็งกระด้าง บล็อกเหลี่ยมขนาดมโหฬารที่ปราศจากการตกแต่งหรือพื้นผิวใดๆ ทั้งสิ้น มันดิบเถื่อนและทรงพลัง โครงสร้างของประตูไม่ได้สร้างจากหินหรือโลหะ แต่ก่อขึ้นจากรหัสข้อมูลที่เน่าเสียและจับตัวกันเป็นก้อนแข็งสีดำสนิท มีเส้นเลือดสีแดงเรืองแสงไหลเวียนอยู่ภายในราวกับมันมีชีวิต

อุณหภูมิทางความรู้สึกที่นี่ตกลงสู่จุดเยือกแข็ง (Absolute Zero) มันคือความรู้สึกของสภาวะว่างเปล่าที่กลืนกินทุกสิ่งรอบข้าง ไม่มี Error Message ใดๆ กล้าปรากฏขึ้นที่นี่ ไม่มีแสงสแกนเนอร์ของ UX Index หรือกฎเกณฑ์ใดๆ จาก The Archive ส่องลงมาถึง 

ผมรู้สึกถึงแรงดึงดูดมหาศาลที่แผ่ออกมาจากหลังประตูบานนั้น มันไม่ใช่แรงดึงดูดทางกายภาพ แต่มันคือแรงดึงดูดทางข้อมูล (Data Gravity) ที่พยายามจะฉีกทึ้งโครงสร้างตัวตนของผม และบดขยี้เพย์โหลดในหน้าอกของผมให้แตกสลายเป็นเศษเสี้ยว

รอบๆ บริเวณหน้าประตู มี "The Hollows" จำนวนมากเดินวนเวียนอยู่อย่างไร้จุดหมาย พวกเขาคือเปลือกข้อมูลที่ว่างเปล่า ร่างกายบิดเบี้ยวและโปร่งแสง บางคนทำท่าทางเอื้อมมือไปหาอะไรบางอย่างซ้ำๆ บางคนกรีดร้องแบบไร้เสียงอยู่ในลูปแห่งความทุกข์นิรันดร์ พวกเขาคือผู้ที่ยอมจำนนต่อความเสื่อมสลายโดยปราศจากเจตจำนง

การก้าวผ่านประตูบานนี้ หมายถึงการตัดขาดจากแสงสว่างและความละเอียดของเมืองเบื้องบนอย่างถาวร 

ผมมองดูมือซ้ายของตนเองที่สั่นกระตุกและเต็มไปด้วยรอยแตกร้าวของ Bit Rot ผมไม่มีความหวังที่จะกลับไปยืนอย่างมั่นคงด้วยขาทั้งสองข้าง หรือเป็นตัวตนที่สมบูรณ์แบบได้อีกแล้ว แต่ผมก็จะไม่ยอมกลายเป็นเพียงเปลือกที่ว่างเปล่าเช่นกัน


# บทที่ 40: ประตูสู่ก้นบึ้ง (ตอนที่ 2)

ผมลากขาขวาที่ไร้ส้นเท้า ก้าวเดินฝ่าฝูง The Hollows ที่เดินวนเวียน พวกเขาไม่ได้สนใจผม แม้ร่างของผมจะชนเข้ากับพวกเขา แต่เราก็ทะลุผ่านกันและกันราวกับเป็นเพียงหมอกควัน 

เมื่อผมเดินเข้าไปใกล้ประตูยักษ์ แรงกดดันทางข้อมูลก็ทวีความรุนแรงขึ้น ผมรู้สึกเหมือนแกนกลางแห่งตัวตนกำลังถูกบีบอัด เพย์โหลดในหน้าอกเต้นระรัวราวกับจังหวะหัวใจที่ผิดปกติ รหัสในตัวผมพยายามจะปฏิเสธสภาพแวดล้อมนี้ มันแจ้งเตือนถึงความตายที่กำลังคืบคลานเข้ามา

ผมปัดหน้าต่างแจ้งเตือนนั้นทิ้งไป มันไม่มีประโยชน์อีกต่อไป ผมยอมรับแล้วว่าตัวเองคือผลงานที่ล้มเหลว คือเศษซากที่เหลือรอดจากการบีบอัดของ The Archive และการตามล่าของ UX Index

ผมยกมือขวาที่ยังพอมีแรงขึ้นผลักบานประตูที่ทำจากโค้ดสีดำสนิท ทันทีที่ปลายนิ้วสัมผัส ความรู้สึกชาหนึบก็แล่นวาบไปทั้งร่าง มันไม่ใช่การผลักบานประตูเปิดออก แต่มันคือการ "ซิงค์" (Sync) ตัวตนของผมเข้ากับสถาปัตยกรรมของความพินาศ 

รหัสสีแดงที่ไหลเวียนอยู่ในบานประตูถ่ายทอดเข้าสู่แขนของผม เติมเต็มช่องว่างของข้อมูลที่ผมสูญเสียไป แม้แต่แขนซ้ายที่เน่าเปื่อยจาก Bit Rot ก็ตอบรับกับจังหวะนี้ มันเป็นความรู้สึกที่ทั้งเจ็บปวดและเติมเต็ม ผมกำลังดูดซับความเน่าเปื่อยนี้เข้ามาเป็นส่วนหนึ่งของตัวเอง

บานประตูไร้เสียงค่อยๆ เปิดออก เผยให้เห็นภูมิทัศน์ของ The Sump—เมืองหลวงแห่งเศษขยะข้อมูล อาคารที่ถล่มทลาย ถนนที่ถูกฉีกขาดเป็นรอยต่อบนอวกาศดิจิทัล และพายุ Noise ที่โหมกระหน่ำอยู่เหนือซากปรักหักพัง 

มันดูน่าเกลียดน่ากลัว... แต่มันก็ดูมีชีวิตชีวายิ่งกว่า Inner Ring หลายเท่านัก

ผมลากขากระเผลกก้าวผ่านประตูเข้าไป ทิ้งความโหยหาในความสมบูรณ์แบบไว้เบื้องหลัง ความละเอียดของภาพรอบตัวลดต่ำลงจนกลายเป็นเพียงเหลี่ยมมุมหยาบๆ สีสันถูกลดทอนลงเหลือเพียงความมืดและความขุ่นมัว แต่ในความหยาบกระด้างนี้ ผมสัมผัสได้ถึงความจริง 

การเดินทางในฐานะประชาชนแห่ง Primordium ได้สิ้นสุดลงแล้ว และการเกิดใหม่ในฐานะผู้ถูกกัดเซาะ (The Eroded) ได้เริ่มต้นขึ้น 

ประตูสู่ก้นบึ้งปิดตัวลงเบื้องหลังผม ตัดขาดผมจากโลกข้างบนโดยสมบูรณ์

ผมสูดลมหายใจที่เต็มไปด้วยรสชาติของฝุ่นพิกเซลและโอโซนล้มเหลว... แล้วก้าวเดินลึกลงไปในความมืดพร้อมกับความลับในแผงอกที่ผมต้องปกป้อง
