# บทที่ 38: ภาษาของความล้มเหลว (ตอนที่ 1)

สแกเวนเจอร์ถอยร่นกลับเข้าสู่ม่านหมอกทันทีที่เห็นไวรัสของผมทำลายระบบพวกมัน ความรอดพ้นนี้เป็นเพียงแสงริบหรี่ สิ่งที่อันตรายยิ่งกว่าคือเงาของ UX Index ที่ยังคงลาดตระเวนค้นหาพิกัดอย่างไม่ลดละ และสถาปัตยกรรมบิดเบี้ยวของ The Sump ที่รอคอยอยู่เบื้องล่าง

ผมลากสังขารขาขวาที่ไร้ส้นเท้าดำดิ่งลึกลงไป ความผิดปกติของพื้นที่เริ่มเด่นชัด หน้าต่างป๊อปอัปสี่เหลี่ยมสีแดงขนาดใหญ่ลอยคว้างกลางอากาศ—Error Messages

`[FATAL EXCEPTION: ENTITY NOT FOUND]`
`[RESOLUTION OVERFLOW: PURGE INITIATED]`

เสียงแจ้งเตือนระรัวถี่ บาดลึกในโสตประสาทราวกับเครื่องบดโลหะ ผมรีบหมอบซ่อนตัวหลังซากกำแพง มือขวากุมแผงอกปกป้องเพย์โหลดกุญแจข้อมูลไว้แน่น สายตาลอบมองผ่านรอยแยก วิญญาณเร่ร่อนดวงหนึ่งอยู่ไม่ไกล เขากำลังพยายามก้าวเดินอย่างสง่างาม รวบรวมสติเพื่อรักษาความคมชัด (High-Res) ของอวาตาร์ตนเองไว้

เสี้ยววินาทีนั้น... ลำแสงสีแดงวาบพุ่งตรงจากเบื้องบน แผดเผาร่างที่พยายามสมบูรณ์แบบจนสลายกลายเป็นเพียงฝุ่นพิกเซล

ผมตัวสั่นเทา กระบวนการชำระล้าง (Purge Routine) ของระบบกำลังกวาดล้างความผิดแผก การพยายามรักษาสภาพ (Rendering) ให้คมชัดในดินแดนแห่งความเสื่อมโทรมคือการทำตัวเป็นเป้าหมาย ตรรกะอันวิปริตของสถานที่แห่งนี้—ความสมบูรณ์แบบคือสิ่งแปลกปลอม

ร่างกายผมสั่นสะท้านจากความเสียหาย แขนซ้ายหนืดอาดแทบขยับไม่ได้ สัญชาตญาณเรียกร้องให้ผมเริ่มกระบวนการฟื้นฟู (Healing) ผมบังคับตัวเองให้หยุดนิ่ง หากผมดึงความละเอียดกลับคืนมา ลำแสงสีแดงนั่นจะพุ่งเป้ามาที่ผม หรือไม่ก็กลายเป็นจุดเด่นให้ UX Index ตรวจจับได้ทันที

ผมหลับตาลง ซึมซับความเจ็บปวดจากการสูญเสียความละเอียด ความสากกระด้างบนผิวหนัง ความพร่ามัวในวิสัยทัศน์ ผมหยุดฝืนสู้เพื่อดึงความคมชัดกลับมา ผมกลั้นใจปล่อยมันไป

ผมก้าวข้ามเส้นแบ่งระหว่างการดำรงอยู่และการล่มสลาย ผมเริ่มหัดเปล่งเสียง "ภาษาของความล้มเหลว" (The Language of Failure)
