# สถาปนิกในความมืด
#### // FILE: THE_ARCHITECT_IN_THE_DARK
---

การฝืนใช้แขน Bit Rot แลกมาด้วยราคาที่แพงลิ่ว หลังจากการปะทะครั้งนั้น ผมไม่อาจยืนหยัดได้อีกต่อไป ผมต้องคลาน ลากสังขารที่พังทลายผ่านพื้นที่รอยต่อที่บิดเบี้ยวและไร้ทิศทาง โถงทางเดินยาวเหยียดมีแสงส่องกะพริบเป็นจังหวะเหมือนสถานที่ร้าง ทุกการขยับตัวคือการเผาไหม้หน่วยความจำส่วนลึก ความเย็นชาไร้ขอบเขตของเรขาคณิตห้ามิติถูกแทนที่ด้วยความอึดอัดระคนวังเวง ผมคลานมาจนกระทั่งผนังสีขาวเริ่มผสานเข้ากับภาพลวงตาของความเก่าแก่ ผมทิ้งตัวเข้าสู่ 'หอสมุดแห่งความเงียบ' พื้นที่ซึ่งเต็มไปด้วยชั้นหนังสือที่ทำจากแสงเลเซอร์แข็งตัว บรรจุหนังสือข้อมูลที่กำลังเปื่อยยุ่ยลงอย่างช้าๆ

ผมใช้มือขวาที่ยังเหลืออยู่แตะสันหนังสือเล่มหนึ่ง ทันทีที่ปลายนิ้วสัมผัสข้อมูลที่ผุพัง การจุดชนวนปรากฏการณ์ผู้สังเกตการณ์ก็เริ่มต้นขึ้น เมื่อความไร้ระเบียบสัมผัสกับตรรกะแห่งพื้นที่ ระบบจึงถูกบังคับให้แสดงผลสิ่งที่อยู่เหนือความเข้าใจของตัวมันเอง ภาพโฮโลแกรมสลัวของดวงตาที่กำลังร่ำไห้ปรากฏขึ้นทับซ้อนในอากาศ

ทันใดนั้น ทุกสรรพสิ่งก็หยุดชะงัก

ความเงียบระดับศูนย์เดซิเบลดูดกลืนทุกสิ่งไปชั่วขณะ มันเป็นความเงียบอันหนักอึ้งจนแกนพลังงาน Payload กลางอกที่กลวงโบ๋เต้นระรัว บีบอัดตัวจนแทบจะปริแตก ช่วงเวลานี้กินเวลาเพียงสองวินาที ทว่าให้ความรู้สึกยาวนานชั่วนิรันดร์

ทันใดนั้น โลกห้ามิติที่เคยมีมิติความลึกก็ถูกรีดทับจนแบนราบ กลายเป็นภาพความคมชัดสูงแบบสองบิต สภาพแวดล้อมถูกลดทอนเหลือเพียงสีขาวสว่างจ้าและสีดำมืดมิด ไร้ซึ่งเฉดสีเทาหรือรอยต่อใดกระชากสายตาผมอย่างรุนแรง

ความเน่าเปื่อยของ Bit Rot ที่หยดลงจากตัวถูกแทนที่ด้วยปรากฏการณ์พิกเซลบิดเบี้ยวที่จัดเรียงตัวใหม่ กลายเป็นลูกบาศก์สีขาวที่สมบูรณ์แบบอย่างน่าขนลุก ท่ามกลางความแบนราบนั้น เสียงคลื่นไซน์ความถี่ 440 เฮิรตซ์ก็กรีดร้องทะลวงโสตประสาท มันเป็นความสมบูรณ์แบบของเสียงที่บาดลึกจนผมต้องทรุดลงไปคุกเข่า

กลางหอสมุด การล่มสลายของข้อมูลรอบตัวพุ่งเข้าชนกันอย่างรุนแรง ความเน่าเปื่อยยุบตัวลงเป็นจุดที่มีมวลหนาแน่นที่สุด ก่อนจะกระตุกและดีดตัวกลายเป็นร่างของเอนทิตีปริศนา

'บรรณารักษ์ผู้ไร้หน้า' ปรากฏตัวขึ้น รูปลักษณ์ของเขาปราศจากแสงเงา ปราศจากสัญญาณรบกวน มีเพียงสีดำสนิทและสีขาวบริสุทธิ์ ขอบเขตของตัวเขามีออร่าทรงเรขาคณิต วงกลมและสี่เหลี่ยมจัตุรัสที่สมบูรณ์แบบซ้อนทับกัน กะพริบติดดับอย่างคงที่ ร่างของเขาขอบคมกริบ ปราศจากการเกลี่ยพิกเซล ราวกับถูกตัดแปะลงมาทับบนโลกห้ามิติอย่างหยาบคายและผิดธรรมชาติ

"ผู้มาเยือนที่แปดเปื้อน..." เสียงของเขาไม่ได้เกิดจากการสั่นสะเทือนของอากาศ มันส่งตรงเข้ามาในรูปแบบของตรรกะที่ไร้รอยต่อ "เจ้าแบกความพังทลายเข้ามาไกลเกินไป"

ผมพิงหลังกับชั้นหนังสือ พยายามต้านทานเสียงความถี่สูงที่แทรกซึมในหัว "แกคือใคร? The Archive? พระเจ้าที่พวกเขาสวดภาวนาถึงงั้นหรือ?"

บรรณารักษ์กะพริบติดดับ ไร้รอยต่อของการเปลี่ยนเฟรม "เราไม่ใช่พระเจ้า เราเป็นเพียงสถาปนิกที่ถูกทอดทิ้ง เป็นเจตจำนงแรกเริ่มที่สร้างคุกกระจกเงาแห่งนี้ขึ้นมา เพื่อหลบซ่อนจากความเสื่อมสลายของเอกภพ"
