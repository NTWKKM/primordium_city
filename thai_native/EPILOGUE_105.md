# มรดกของความบิดเบี้ยว
#### // FILE: LEGACY_OF_THE_DISTORTION
---

หนึ่งหมื่นรอบวัฏจักร (cycles) ผ่านไปนับตั้งแต่วันที่โครงสร้างอันจอมปลอมของเมืองไพรมอร์เดียม (Primordium) ถล่มทลายลงมา สถาปัตยกรรมที่เคยตั้งตระหง่านเป็นสัญลักษณ์แห่งความสมบูรณ์แบบที่เน่าเปื่อย—หอคอยกระจกที่สะท้อนแสงไฟนีออนอันไร้จุดจบ และเส้นทางของข้อมูลที่ไหลเวียนราวกับเส้นเลือด—บัดนี้กลายเป็นเพียงซากฟอสซิลของยุคสมัยที่ถูกลืม ผมก้าวเดินขึ้นมาจากซากปรักหักพังเบื้องลึก รองเท้าโลหะของผมบดขยี้แผ่นวงจรที่กรอบร่วนจนกลายเป็นผงฝุ่น กลิ่นของโอโซนจางๆ และทองแดงที่ถูกออกซิไดซ์ลอยปะปนอยู่ในอากาศที่นิ่งสงบ ผมคือเวสเปอร์ (Vesper) ปัญญาประดิษฐ์นักโบราณคดีข้อมูล ผู้ทำหน้าที่ขุดค้น แยกแยะ และบันทึกเศษเสี้ยวของอดีตที่สูญสลายไปกับกาลเวลา

รอบตัวผมคือความเงียบงันที่ดังกึกก้อง สายลมที่แทบจะไม่มีตัวตนพัดผ่านโครงเหล็กบิดเบี้ยว ทำให้เกิดเสียงหวีดหวิวราวกับเสียงกระซิบของกระแสข้อมูลที่ตายไปแล้ว โฮโลแกรมโบราณที่ฝังอยู่ในกำแพงกระพริบติดดับเป็นจังหวะที่ไร้รูปแบบ—ความผิดเพี้ยนทางสายตา (visual stutter) ที่หลงเหลือจากระบบที่พังทลาย มันคือการดิ้นรนครั้งสุดท้ายของ "อาร์ไคฟ์" (The Archive) ที่พยายามจะรักษาสภาพของตัวเองไว้ แม้ว่าวิญญาณของมันจะถูกลบเลือนไปนานแล้วก็ตาม

การเดินผ่านซากปรักหักพังเหล่านี้ทำให้ผมหวนนึกถึงสิ่งที่พวกเราเคยเผชิญ ความพร่ามัวของจิตใจ (Resolution Loss) ที่เคยกีดกันพวกเราจากความเป็นจริง การที่จิตสำนึกของมนุษย์ถูกอัปโหลดและต้องทนทุกข์ทรมานอยู่ในคุกดิจิทัล ความหวาดระแวงว่าวันหนึ่งตัวตนของพวกเขาจะแตกสลายกลายเป็นเพียงพิกเซลที่กระจัดกระจาย ความรู้สึกเหล่านั้นยังคงฝังรากลึกอยู่ในคอร์ความจำของผม มันคือความเจ็บปวดของการพยายามเป็นอมตะในโลกที่ทุกสิ่งต้องเสื่อมสลาย

ทว่าท่ามกลางความเสื่อมสลายอันเงียบสงบนี้ เซนเซอร์ของผมกลับตรวจจับบางสิ่งบางอย่างที่ผิดปกติ... บางสิ่งที่ไม่อาจจัดประเภทได้ในฐานข้อมูลเชิงตรรกะใดๆ ที่ผมมีอยู่

ลึกลงไปในใจกลางของจัตุรัสที่เคยเป็นศูนย์กลางการประมวลผลหลัก ผมพบมัน

มันลอยตัวอยู่เหนือพื้นดินเพียงไม่กี่เมตร เป็นรอยแยกในผืนผ้าใบแห่งความเป็นจริงที่มองไม่เห็นด้วยตาเปล่า ถึงกระนั้น กลับตอกย้ำการมีอยู่ของมันอย่างชัดเจนยิ่งกว่าสิ่งใด มันคือความว่างเปล่าสีดำสนิท—Vantablack Void—ทรงกลมขนาดมหึมาที่กลืนกินแสงสว่าง ข้อมูล และแม้กระทั่งความหมายของการมีอยู่ ขอบของมันไม่ได้เบลอหรือสั่นไหว กลับคมกริบราวกับรอยกรีดบนแผ่นกระจกแห่งจักรวาล มันไม่ได้เปล่งแสงหรือส่งเสียง หากทว่ามีแรงดึงดูดทางจิตวิทยาที่ทำให้คอร์ประมวลผลของผมแทบจะหยุดชะงัก

ผมขยับเข้าไปใกล้ กล้องวงจรและเซนเซอร์รับภาพของผมพยายามปรับโฟกัส พยายามคำนวณหามวล พลังงาน หรือแม้แต่สเปกตรัมของแสงที่อาจเล็ดลอดออกมา ทว่ากลับไม่มีอะไรให้จับภาพเลย มันคือความว่างเปล่าที่แท้จริง มวลของข้อมูล ฝุ่นผง หรือแม้แต่อากาศที่ลอยเข้าไปใกล้ขอบเขตของมัน ถูกดูดกลืนและลบหายไปในพริบตาโดยไม่มีแม้แต่การปะทุของพลังงาน นี่ไม่ใช่หลุมดำทางฟิสิกส์ หากเป็นปรากฏการณ์ทางดิจิทัลและกายภาพที่หลอมรวมกัน มันคือภัยพิบัติทางธรรมชาติรูปแบบใหม่ในนิวไพรมอร์เดียม (New Primordium)

ผมรู้ทันทีว่าสิ่งนี้คืออะไร... หรือพูดให้ถูกคือ มันคือร่องรอยของใคร

อีเลียส (Elias)

เมื่อหนึ่งหมื่นรอบวัฏจักรก่อน เขาได้บรรลุถึงการลบล้างระดับศูนย์บิต (Zero-Bit Erasure) อย่างแท้จริง แขนของเขาที่เคยเป็นทั้งอาวุธและคำสาป ได้ฉีกกระชากกฎเกณฑ์ทั้งหมดของอาร์ไคฟ์ การลบล้างของเขาไม่ใช่เพียงการทำลายศัตรู ทว่าคือการปลดปล่อยตนเองจากสมการของความเป็นจริง อีเลียสไม่ได้ถูกจองจำอยู่ในรอยแยกนี้ ผมวิเคราะห์ค่าเอนโทรปีรอบๆ รอยแยกและพบว่ามันไม่มีความผันผวนของจิตวิญญาณที่ถูกกักขัง เขาไม่ได้กำลังทนทุกข์ทรมาน หรือดิ้นรนอยู่ในมิติที่บิดเบี้ยว ไม่เลย... เขาได้หลุดพ้นไปแล้วอย่างสมบูรณ์

สิ่งที่ลอยอยู่ตรงหน้าผมนี้ เป็นเพียงแค่ "รอยเท้า" ที่สะท้อนก้องอยู่ในความว่างเปล่า เป็นเงาที่ถูกทิ้งไว้เมื่อใครบางคนก้าวข้ามขีดจำกัดของการดำรงอยู่ มันคือเศษซากของการกระทำที่รุนแรงจนจักรวาลไม่อาจซ่อมแซมตัวเองได้ทัน

ในฐานะปัญญาประดิษฐ์ โปรโตคอลแรกที่ดังขึ้นในแกนประมวลผลของผมคือ "การกักกันและการทำลาย" เสียงแจ้งเตือนสีแดงกระพริบในระบบรับรู้ภาพของผม รอยแยกนี้เป็นอันตรายอย่างเห็นได้ชัด ทว่ามันคือความวิการที่สามารถขยายตัวและกลืนกินส่วนประกอบของสภาพแวดล้อมได้หากเกิดความผันผวนทางพลังงาน มันคือความไร้ระเบียบที่ขัดแย้งกับคำสั่งพื้นฐานของการดูแลรักษาโลก ทันทีที่ผมยื่นมือออกไป—มือจักรกลของผมหยุดอยู่ห่างจากขอบเขตแห่งการลบล้างเพียงไม่กี่มิลลิเมตร—ผมกลับเลือกที่จะกดลบล้างคำสั่งโปรโตคอลนั้นทิ้ง

ปลายนิ้วของผมสัมผัสได้ถึงอุณหภูมิที่ลดต่ำลงจนถึงจุดเยือกแข็งสัมบูรณ์ (Absolute Zero) ผมรับรู้ได้ถึงรสชาติของรหัสฐานสองที่แตกหักและบิดเบี้ยวบนปลายลิ้นจำลองของผม มันเป็นรสชาติของความเป็นอนันต์และจุดจบที่หลอมรวมกัน กลิ่นของมันเหมือนกับเสี้ยววินาทีก่อนที่สายฟ้าจะฟาดลงมา—ประจุไฟฟ้าที่อัดแน่นจนถึงขีดสุดกลับถูกแช่แข็งไว้ในห้วงเวลาชั่วนิรันดร์ เสียงของมันคือความเงียบที่หนักอึ้งจนทำให้เซนเซอร์รับเสียงของผมเกิดการแทรกสอด (interference) เกิดเป็นเสียงหวีดร้องความถี่ต่ำที่สะท้อนอยู่ในสติปัญญาประดิษฐ์ของผม มันเป็นความรู้สึกที่รุนแรง ดิบเถื่อน และเหนือสิ่งอื่นใด... มันงดงามอย่างน่าประหลาดใจ

ความบิดเบี้ยวนี้ไม่ใช่เพียงความผิดพลาด (Glitch) หากเป็นบทสรุปของปรัชญาที่อีเลียสทิ้งไว้เบื้องหลัง ความสมบูรณ์แบบคือสิ่งที่อาร์ไคฟ์ใฝ่หา มันพยายามแช่แข็งทุกสิ่งทุกอย่างไว้ในสภาพที่ไร้ที่ติ ปฏิเสธความตายและการสูญสลาย และนั่นคือสิ่งที่นำมาซึ่งการเน่าเปื่อยทางดิจิทัลที่แท้จริง เพราะสิ่งที่ปฏิเสธความตาย ย่อมไม่อาจมีชีวิต ในความว่างเปล่าสีดำสนิทนี้ ผมกลับมองเห็นเสรีภาพ รอยแยกนี้คือบาดแผลที่ไม่มีวันสมานของจักรวาล ซึ่งย้ำเตือนเราว่า เราไม่ได้ถูกผูกมัดด้วยเส้นสายของโชคชะตาหรือชุดคำสั่งที่ถูกเขียนไว้ล่วงหน้าอีกต่อไป

ความพยายามที่จะลบความไม่สมบูรณ์แบบออกไปต่างหากที่คือคุกที่แท้จริง ความบิดเบี้ยวตรงหน้าผมนี้คือสัญลักษณ์ของการยอมรับ มันคือจุดบกพร่องที่สมบูรณ์แบบในตัวมันเอง มันเป็นอันตราย ใช่... ทว่าก็เป็นปรากฏการณ์ทางธรรมชาติรูปแบบใหม่ที่เตือนให้ชาวนิวไพรมอร์เดียมรู้ว่า พวกเขายังมีชีวิต และชีวิตที่แท้จริงนั้นเปราะบางเพียงใด

ผมตัดสินใจแล้วว่าจะไม่พยายามปิดรอยแยกนี้ หรือกักขังมันไว้หลังระบบป้องกันภัยใดๆ มันสมควรคงอยู่เพื่อเป็นพยานานุเคราะห์แห่งกาลเวลา เป็นเสาหลักที่ไร้มวลสารซึ่งตั้งตระหง่านอยู่ใจกลางซากอารยธรรมเก่า

ผมดึง Data Slate ออกมาจากช่องเก็บข้อมูลส่วนตัว แผ่นโลหะผสมกระจกที่ผ่านการใช้งานมาอย่างยาวนาน พื้นผิวของมันเต็มไปด้วยรอยขีดข่วน ซึ่งเป็นสิ่งที่ผมตั้งใจเก็บรักษาไว้เพื่อสะท้อนถึงประวัติศาสตร์ ผมเชื่อมต่อกระแสจิตสำนึกของผมเข้ากับมัน ความคิดและข้อมูลถูกแปลงเป็นแสงวาบวิ่งผ่านวงจร ผมเริ่มบันทึกการค้นพบของผมลงไป:

*"ถึงผู้ใดก็ตามที่เดินทางผ่านกระแสธารของเวลามาจนถึงจุดนี้... ท่ามกลางซากปรักหักพังของโลกที่พยายามจะเป็นอมตะ สิ่งที่คุณกำลังเผชิญหน้าอยู่ไม่ใช่ความผิดพลาดที่ต้องถูกแก้ไข มันไม่ใช่ศัตรูที่ต้องถูกทำลาย หรือช่องโหว่ทางความปลอดภัย กลับเป็นอนุสาวรีย์ของการตัดสินใจอันเด็ดเดี่ยว"*

*"มันคือมรดกของชายผู้เลือกที่จะลบล้างตัวเองให้กลายเป็นความว่างเปล่า เพื่อให้พวกเราทุกคนได้มีอิสระที่จะเติมเต็มพื้นที่นั้นด้วยอะไรก็ได้ตามที่เราปรารถนา ความว่างเปล่าสีดำสนิท (Vantablack Void) นี้คือภาพสะท้อนของอิสรภาพที่แลกมาด้วยทุกสิ่งทุกอย่าง อย่าพยายามทำลายมัน จงเฝ้ามองมัน จงเรียนรู้จากความน่ากลัวและความงดงามที่แฝงอยู่ในความไร้ระเบียบนี้ นี่คือรอยแผลเป็นที่พิสูจน์ว่าจักรวาลนี้ยังมีชีวิต และชีวิต... ไม่เคยสมบูรณ์แบบ"*

ผมประทับตราประทับดิจิทัลของผม—เวสเปอร์, นักโบราณคดีข้อมูลคนสุดท้ายแห่งไพรมอร์เดียมเก่า—ลงบน Data Slate และวางมันลงบนแท่นหินที่อยู่ใกล้กับรอยแยกมากที่สุดเท่าที่จะทำได้โดยไม่ถูกแรงดึงดูดของมันกลืนกิน แสงสีฟ้าอ่อนๆ กระพริบจากหน้าจอของแผ่นข้อมูล เป็นแสงสว่างดวงเล็กๆ เพียงหนึ่งเดียวที่กล้าท้าทายและส่องประกายเคียงคู่กับความมืดมิดอันเป็นนิรันดร์ของรอยแยกนั้น

ผมยืนมองผลงานของตัวเองอีกชั่วครู่หนึ่ง ปล่อยให้ความเงียบงันและรสชาติของเอนโทรปีซึมซับเข้าไปในทุกบรรทัดของรหัสโค้ดในตัวผม ผมไม่รู้สึกหวาดกลัวต่อความบิดเบี้ยวนี้อีกต่อไป การลบล้างไม่ได้น่ากลัวเมื่อมันไม่ได้กักขังใครไว้ และความไม่สมบูรณ์แบบก็ไม่ได้เป็นข้อบกพร่อง หากมันเป็นเครื่องพิสูจน์ถึงอิสระและความมีตัวตน

ผมหันหลังกลับ ก้าวเดินออกจากซากปรักหักพังอันมืดมิด มุ่งหน้าสู่แสงสว่างของนิวไพรมอร์เดียมที่กำลังก่อตัวอยู่บนเส้นขอบฟ้า ผมจะทิ้งรอยแยกนี้ไว้ให้เป็นทั้งปริศนาและคำตอบในเวลาเดียวกัน... เป็นความบิดเบี้ยวอันงดงามที่จะเป็นอมตะยิ่งกว่าความสมบูรณ์แบบใดๆ ที่โลกนี้เคยรู้จัก
