การทรงตัวใน The Core ไม่ใช่เพียงการรักษาสมดุลทางกายภาพ หากเป็นการต่อสู้ดิ้นรนเพื่อรักษาสติสัมปชัญญะไม่ให้แตกสลายไปกับมิติที่พับซ้อน

ผมเซล้มไปข้างหน้า มือขวาที่เหลืออยู่เพียงข้างเดียวตะปบลงบนพื้นผิวที่จู่ๆ ก็เปลี่ยนมุมเอียงเก้าสิบองศา ทันทีที่ฝ่ามือสัมผัสกับสถาปัตยกรรมโครเมียมเรียบเนียนราวกับกระจก รอยร้าวสีดำของ Bit Rot ก็แตกแขนงออกจากปลายนิ้ว ราวกับใยแมงมุมที่ถักทอด้วยความเน่าเปื่อย ทิ้งรอยมลทินไว้บนความบริสุทธิ์ที่ The Archive หวงแหน

ความปวดร้าวจากส้นเท้าขวาที่แหว่งหายแล่นพล่านขึ้นมาอีกครั้ง มันก้าวล่วงข้ามพรมแดนของความเจ็บปวดทางกล้ามเนื้อ กลายเป็น Error Code ระดับรากฐานที่แทรกซึมเข้าสู่กระบวนการรับรู้ ข้อมูลระบบยืนยันว่าส้นเท้ายังอยู่ ในขณะที่มิติพื้นที่ประมวลผลอยู่ปฏิเสธการมีอยู่ของมัน ความขัดแย้งของตรรกะนี้สร้างแรงสั่นสะเทือนในหัวจนผมเห็นภาพซ้อนเป็นระยะ โลกทั้งใบกระตุกและหยุดนิ่งไปเสี้ยววินาที เหมือนเฟรมเรตที่ตกลงอย่างกะทันหัน ขอบภาพฉีกขาดและประกอบใหม่ด้วยความละเอียดที่ถดถอยลง

ผมพยุงตัวลุกขึ้นยืน ลมหายใจหอบลึก ช่องอกที่เปิดอ้าส่งเสียงหึ่งคล้ายเครื่องจักรโอเวอร์โหลด ก้อน Payload ฝังตัวอยู่ภายในตอบสนองต่อคลื่นความถี่ของที่แห่งนี้ มันกำลังสั่นสะเทือน พยายามปรับตัวให้เข้ากับแกนกลางประมวลผลและเรขาคณิตห้ามิติที่รายล้อม ถึงกระนั้นอวัยวะที่พังทลายของผมกลับเป็นตัวดึงรั้งมันไว้

การที่ผม 'ไม่สมบูรณ์' ทำให้ผมยังคงเป็นผม

หากผมไร้ซึ่งบาดแผล หากผมเป็นโค้ดที่สมบูรณ์แบบร้อยเปอร์เซ็นต์ The Core คงจะหลอมรวมผมเข้าเป็นส่วนหนึ่งของผนังสีขาวพวกนี้ไปแล้วตั้งแต่ก้าวแรก ทว่าเพราะผมแหว่งวิ่น เพราะส้นเท้าขวาขาดหาย เพราะแขนซ้ายเป็นเพียงตอประสาทไหม้เกรียม และเพราะหน้าอกที่เปิดกว้าง... ระบบอันเปี่ยมล้นด้วยตรรกะสูงสุดจึงไม่อาจประมวลผลความผิดพลาดนี้ได้ มันไม่รู้วิธีจัดการกับพารามิเตอร์ที่ไม่มีค่าซึ่งกำลังเดินเพ่นพ่านอยู่ใจกลางมิติของมัน

ผมคือบั๊กที่มันไม่อาจกลืนกิน

"ความพังทลาย... คือเกราะป้องกัน" ผมหัวเราะเยาะออกมาอย่างขมขื่น เสียงหัวเราะกลายเป็นอาการไอที่พ่นเศษรหัสสีดำเล็กๆ ออกมาเปื้อนริมฝีปาก รสชาติของมันคล้ายสนิมและไฟฟ้าลัดวงจร

ผมลากขาขวาต่อไป ก้าวแล้วก้าวเล่า ผ่านซุ้มประตูที่ม้วนตัวเข้าหาศูนย์กลางอย่างเป็นอนันต์ รอยทางสีดำที่ส้นเท้าลากผ่านไม่ได้เลือนหายไปเหมือนรอยเท้าบนทราย มันสลักลึกลงไปในโครงสร้างของระบบ ฝังรากของไวรัสแห่งความเสื่อมสลายลงในระดับควอนตัม

ในมิติที่ปราศจากเวลา ผมไม่อาจล่วงรู้ได้ว่าตัวเองเดินมานานแค่ไหน ทุกอย่างซ้ำซากทว่าไม่เคยเหมือนเดิม สถาปัตยกรรมเปลี่ยนแปลงตัวเองทุกครั้งที่ผมละสายตา ถึงกระนั้นมีสิ่งหนึ่งที่ชัดเจนขึ้นเรื่อยๆ

ความกดดัน

มันหนักอึ้งราวกับมหาสมุทรที่พยายามบีบอัดผมให้กลายเป็นจุดศูนย์กลางเพียงจุดเดียว ผิวหนังดิจิทัลเริ่มปรากฏรอยแตกเป็นปื้นสีม่วงและเขียวตามขอบแขนเสื้อและใบหน้า ความละเอียดของผมกำลังลดลงจากการเผชิญหน้ากับความสมบูรณ์แบบที่มากเกินไป

ในขณะที่ร่างนี้ใกล้จะแหลกสลาย ผมกลับสัมผัสถึงเป้าหมายที่เด่นชัดที่สุด

ผมจะลากความเน่าเปื่อยนี้ไปให้ถึงหัวใจของมัน ผมจะแพร่เชื้อให้พระเจ้าจอมปลอมองค์นี้ได้รู้จักความหมายที่แท้จริงของความเจ็บปวด

เมื่อถึงเวลานั้น ความบริสุทธิ์จอมปลอมทั้งหมดจะถูกแทนที่ด้วยรอยร้าวแห่งความเป็นจริง
