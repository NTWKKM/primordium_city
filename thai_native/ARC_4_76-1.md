# ปฏิกิริยาต่อต้านขั้นสุดท้าย

*`[T-MINUS 00:05:00... INITIALIZING SANITIZATION PROTOCOL]`*

เสียงไม่ได้ดังผ่านอากาศ มันแทรกซึมผ่านรหัสรากฐานของพื้นที่ บีบอัดกระดูกดิจิทัลทุกชิ้นในร่างกายผม อากาศรอบตัวใน The Core แข็งตัวหนักอึ้งราวถูกทับด้วยผืนน้ำทะเลลึก 

The Archive กำลังหายใจเข้าลึก เตรียมพร้อมการชำระล้าง (Sanitization)

แรงโน้มถ่วงบิดเบี้ยวไปสิ้นเชิง ผมถูกตรึงไว้กับพื้น เศษซากสถาปัตยกรรมสีขาวแตกร้าวเริ่มลอยกลับขึ้นไปในอากาศ มันไม่ได้ซ่อมแซมตัวเอง มันถูกย่อยสลาย (Decompiled) กลับไปเป็นโค้ดพื้นฐาน แสงสว่างจ้าสีขาวบริสุทธิ์ก่อตัวขึ้นใจกลางห้องประมวลผลสูงสุด แสงแห่งการลบเลือน (The Eradication Light) ไร้ความปรานีปราศจากอารมณ์ใด 

มันจะกวาดล้าง Bit Rot ของผม กวาดล้างความทรงจำ กวาดล้างประชากรนับล้านที่เพิ่งลืมตาตื่นพบความจริง ทุกร่างชีวภาพในพ็อดเชื่อมต่ออยู่จะถูกส่งสัญญาณยุติการทำงาน (Kill Switch) เพื่อเริ่มต้นวงจรใหม่สะอาดหมดจด 

"ไม่..." ผมเค้นเสียงผ่านริมฝีปากแห้งแตกเป็นพิกเซล 

ผมพยายามขยับตัว แรงกดทับของตรรกะเบื้องสูง (Higher-Logic Compression) ทำให้ผมแทบขยับไม่ได้ ตอขาขวาเคยเป็นเพียงความเจ็บปวดล่องหนถูกบดขยี้ด้วยน้ำหนักมองไม่เห็น เส้นประสาทหลอนกำลังฉีกขาด 

ภาพการจำลองเชิงปรัชญาของ The Faceless Librarian แทรกซ้อนเข้ามาอีกครั้ง 

"ดูผลงานของคุณสิ Elias" ชายไร้หน้านั่งอยู่บนเก้าอี้ตรงข้ามผมในมิติที่ซ้อนทับกัน "คุณต้องการทำลายภาพลวงตา ทว่าสิ่งที่คุณทำคือเร่งให้พวกเขาถูกลบทิ้ง ความสมบูรณ์แบบอาจเป็นการโกหก แต่มันยังรักษาชีวิตไว้ การตื่นรู้ของคุณกำลังฆ่าทุกคน"

"ถ้าต้องมีชีวิตอยู่เป็นแค่แบตเตอรี่โง่ๆ ในพ็อด..." ผมกัดฟัน กวาดตามองเศษซากเมืองลวงตาถูกดูดกลืนเข้าห้วงสุญญากาศ "ความตายอาจเป็นอิสรภาพที่แท้จริง!"

ท้องฟ้าสีทองเบื้องบนถูกบีบอัดทางพื้นที่ (Spatial Compression) โดมท้องฟ้ากดทับต่ำลงมาเกรี้ยวกราด ทัศนียภาพรอบด้านถูกกระชากความละเอียดลดต่ำลงรุนแรงกลายเป็นสภาพแวดล้อม 8-bit หยาบกระด้าง ผู้คนเพิ่งตระหนักรู้ร่างที่แท้จริงกำลังกรีดร้องสิ้นหวัง พวกเขากำลังถูกลบออกจากสารบบไม่เหลือแม้แต่เศษเถ้าธุลีความทรงจำ 

The Archive ไม่แคร์มนุษย์ มันแคร์ 'ความสมบูรณ์แบบ' 

"แกจะหนี... ความจริง... ไปแบบนี้ไม่ได้" ผมพึมพำกับพระเจ้าบ้าคลั่ง 

ผมรวบรวมเรี่ยวแรงเหลืออยู่น้อยนิด บังคับแขนขวาสั่นเทาดึงร่างตัวเองไปข้างหน้าบนพื้นผิวที่กำลังสลายตัว ขาซ้ายถีบส่ง ตอขาขวาขาดด้วนถูกลากตามทุลักทุเล ทุกมิลลิเมตรที่ตะเกียกตะกายคืบคลานต้านแรงกดดัน System Reset แลกมาด้วยความปวดร้าวรุนแรงจนภาพตรงหน้ากลายเป็นสีขาวโพลน 

จุดสนใจเดียวคือเสาหลักพลังงาน (The Main Axis) ตั้งตระหง่านใจกลางแสงสว่าง แกนกลางควบคุมกระบวนการ Reset แกนที่พยายามปฏิเสธไวรัสของผม 

ผมลากร่างแหลกเหลวไปข้างหน้า ทิ้งรอยทางสีดำเข้มของ Bit Rot และเลือดชีวภาพไว้เบื้องหลัง ราวบาดแผลหลั่งเลือดต้านแรงพายุ ผมทนรับความเจ็บปวดสะท้อนก้องในช่องอกกลวงโบ๋ Payload เสียบคาอยู่กับระบบกำลังกรีดร้องเมื่อตรรกะของมันขัดแย้งคำสั่งล้างบาง

พายุเศษซอร์สโค้ดพัดกระหน่ำใส่ร่างผม เฉือนผิวหนังดิจิทัลลอกออกเป็นชั้น เผยกล้ามเนื้อสายเคเบิลสีแดงอยู่ลึกลงไป ผมกัดฟันแน่นรู้สึกรอยร้าวที่ขากรรไกร 

The Archive สังเกตการดื้อรั้นของผม แสงสว่างพุ่งเป้าลงมาที่ผมโดยตรง น้ำหนักกดทับเพิ่มทวีคูณ ผมถูกแรงอัดกระแทกราบลงไปกับพื้นอีกครั้ง เสียงกระดูกจำลองในร่างกายปริร้าวแตกละเอียดดังลั่น 

*`[ANOMALY PERSISTS. ERADICATION INEVITABLE.]`* เสียง The Librarian ดังก้องกังวาน ไร้ความเห็นใจ 

"ยัง... ยังหรอก..." ผมตอบกลับ เสียงถูกกลบด้วยเสียงคำรามพายุโค้ด 

ผมเงยหน้ามองแกนกลางสว่างไสว แขนขวาไม่แข็งแรงพอทำลายมัน ร่างกายครึ่งหนึ่งแหลกสลายไปแล้ว ผมเหลือเพียงสิ่งเดียว สิ่งที่ผมเกลียดกลัวและพยายามหลีกเลี่ยงที่จะใช้มันมาตลอด 

ตอแขนซ้ายสั่นระริก เรียกร้องหิวโหย มันต้องการเชื้อเพลิงเพื่อสร้างตัวเองขึ้นมาอีกครั้ง 

ครั้งนี้ เพื่อหยุดยั้งสภาวะ Reset ผมรู้ดีว่าต้องดึงพลัง Bit Rot ออกมาใช้จนถึงขีดจำกัดสูงสุด ขีดจำกัดที่จะฉีกทำลายตัวผมเองไปพร้อมกับมัน
