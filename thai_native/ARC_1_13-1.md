# ตลาดมืดแห่งเศษเสี้ยว
#### // FILE: THE_FRAGMENT_MARKET_SHADOW
---

สุญญากาศขนาด 32K กัดกินจิตสำนึกของผม มันไม่ใช่ความหนาวเหน็บทางกายภาพ หากเป็นความว่างเปล่าที่หลอกหลอน (Phantom Void) ฝังรากลงลึกตรงตำแหน่งที่เคยเป็นความทรงจำของ 'ความโกรธเกรี้ยวเมื่อตระหนักว่าท้องฟ้าเป็นเพียงภาพลวงตา' ผมลิ้มรสขอบเขตอันเว้าแหว่งของมัน รสชาติเถ้าถ่านและสเตติกค้างเกรอะกรังในคอเหมือนโลหะขึ้นสนิม ทุกครั้งที่พยายามควานหาความอบอุ่น เสียงสะท้อนกลวงเปล่าและอาการชาหนึบจะลุกลามไปตามเครือข่ายเส้นประสาทเทียม

ผมลากสังขารลัดเลาะไปตามตรอก 'ฮอลโลว์' (Hollow) ที่เรนเดอร์ไม่เสร็จ โลกเบื้องหน้าถูกย้อมด้วยสีเขียวหม่นของจอ CRT เก่าและสีเทาด้านทึบของเดดพิกเซล แสงสว่างที่นี่ไม่เคยปรานี มันสาดส่องลงมาเป็นมุมแหลมเกรี้ยวกราด เฉือนทะลุเงามืด อากาศรอบตัวอึดอัด หนาทึบจากการอัดแน่นของโฮโลแกรม UI นับพันที่ซ้อนทับกันจนแทบไม่มีบิตว่างให้หายใจ เสียงครางต่ำ (Low-frequency hum) ดังหึ่งอยู่ในสถาปัตยกรรม แทรกซึมผ่านกะโหลก บังคับให้ฟันสังเคราะห์ของผมสั่นสะท้าน

ผมหันศีรษะเร็วเกินไป ภาพตรงหน้าเกิดอาการฉีกขาด (Screen-tearing) เป็นริ้วแนวนอน โลกทั้งใบเลื่อนศูนย์ไปชั่วขณะ ก่อนกระตุกกลับเข้าที่อย่างรุนแรง ผมกะพริบตาถี่รัว บังคับให้วิชั่นทรงตัว แขนซ้ายที่ถูกปะติดปะต่อขึ้นจากขยะโค้ด (Junk-code) กระตุกอย่างบ้าคลั่ง ประกายไฟสีฟ้าสปาร์คแลบแปลบปลาบ มันทำงานผิดพลาดอย่างควบคุมไม่ได้ ราวกับกล้ามเนื้อเน่าเปื่อยที่พยายามจำลองจังหวะเต้นของหัวใจ

NPC ตามรายทางปราศจากชีวิตจิตใจ พวกมันติดอยู่ในลูปแอนิเมชันสามเฟรมที่วนซ้ำไปมาอย่างเชื่องช้า กรอบร่างอืดอาด ดวงตาว่างเปล่าไร้อารมณ์ เอนทิตีเหล่านี้ขับเคลื่อนด้วยตรรกะการทำธุรกรรม (Transactional logic) เพียงอย่างเดียว ไม่มีสิ่งใดมากไปกว่านั้น 

สุดปลายความแคบอึดอัดของตรอก คือทางเข้ารังข้อมูล (Data-den) ผมก้าวเข้าไปในความมืดที่เหม็นคาวกลิ่นโอโซนไหม้เกรียม 

"คุณกำลังมองหาอะไรอยู่ ซากโค้ด?" เสียงแหบพร่าดังขึ้นจากเงามืดหลังแผงสายไฟระโยงระยาง

นั่นคือ 'วีฟเวอร์' (Weaver) นายหน้าค้าข้อมูล

ผมอ้าปากจะตอบ สัญชาตญาณเก่ากลับทรยศ ผมหยุดนิ่ง... รอคอยเสียงปิงใสๆ ของ UI สีทองที่จะแจ้งเตือนสถานะคู่สนทนาแบบที่อวตารเขตวิจิตร (Luminary) คุ้นเคย ไม่มีเสียงใดเกิดขึ้น มีเพียงความเงียบงันที่น่าอึดอัด

"อ่า..." วีฟเวอร์ขยับตัวเข้ามาในแสงไฟสลัว ดวงตาจักรกลหรี่ลง ท่าทีหยุดชะงักเมื่อครู่คือ 'เทล' (Tell) ที่ผมเผลอแสดงออก "รอกล่องข้อความช่วยเหลืออยู่หรือไง พ่ออวตารตกสวรรค์ แกเอาขยะยัดแขนซ่อนเมทาดาตาไว้ซะมิดชิด ถึงอย่างนั้น 'สัญชาตญาณ' ของแกมันก็ฟ้องชัดเจนยิ่งกว่าป้ายไฟเสียอีก"
