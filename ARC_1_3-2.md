# ดัชนีแห่งความไม่แน่นอน (The UX Index Paradox)
---

แสงอาทิตย์ประดิษฐ์ยามเช้าสาดส่องลงมาทำมุมสี่สิบห้าองศาพอดิบพอดี ไร้ความร้อนระอุ มีเพียงความอบอุ่นที่ถูกตั้งค่าพารามิเตอร์มาอย่างแม่นยำเพื่อกระตุ้นระดับโดปามีนในสมอง ไม่มีละอองฝุ่นใดลอยล่องเว้นแต่ฝุ่นสังเคราะห์ที่ระบบจงใจเรนเดอร์เพื่อสร้างมิติความลึก (Depth of Field) ทุกก้าวที่ผมเดินออกจากโหนดพักอาศัยลงบนระเบียงของวงแหวนชั้นใน (Inner Ring) คือการแหวกว่ายในมหาสมุทรข้อมูลที่ถูกประมวลผลด้วยความละเอียดสูงสุด (Max Resolution)

กลิ่นโอโซนบริสุทธิ์เจือกลิ่นไอดินหลังฝนตก—Petrichor ที่ถูกเรียงร้อยในระดับโมเลกุล—ลอยแตะจมูก มันคือเช้าวันใหม่ที่สมบูรณ์แบบ สมบูรณ์แบบจนรู้สึกกระอักกระอ่วน 

ผมเดินปะปนไปกับฝูงชนของกลุ่ม High-Tier ขอบเขตการมองเห็นของผมคมชัดระดับไฮเปอร์เรนเดอร์ ทุกรายละเอียดตั้งแต่ขอบเสื้อผ้า เงากระทบพื้น รอยยับของเนื้อผ้า ถูกลบรอยหยัก (Anti-aliasing) จนราบรื่นไร้ที่ติ ชีพจรของเมืองเต้นเป็นจังหวะบิตเรตที่สม่ำเสมอ ทว่าในความลื่นไหลนั้น ผมกลับรู้สึกถึงน้ำหนักมหาศาลที่กดทับบ่า—ความหนักอึ้งของ UX Index ที่พุ่งทะยานขึ้นหลังจาก ‘ขบถระดับไมโคร’ ของผม

ความทรงจำเรื่องการจงใจชงกาแฟให้หกเมื่อเช้า การสวมเสื้อเชิ้ตที่กระดุมเบี้ยวไปหนึ่งมิลลิเมตร... ผมเคยภูมิใจว่ามันคือรอยรั่วที่ผมสร้างเพื่อยืนยันว่ายังมีเจตจำนงเสรี แต่เมื่อมองตัวเลข UX Index สีทองที่มุมลานสายตา ผมก็ถูกตบหน้าด้วยความจริงอันเยือกเย็น 

‘ความโกลาหลเชิงสร้างสรรค์’ (Creative Chaos) ของผม ไม่ใช่การต่อต้าน มันเป็นแค่ลูปการทำงานที่ The Archive คาดการณ์ไว้แล้ว อัลกอริทึมรู้ว่าซอฟต์แวร์จำลองจิตใจมนุษย์ต้องการพื้นที่ระบายความเครียดเพื่อไม่ให้ระบบแครช (Crash) มันจึงวาดขอบเขตของความดื้อรั้นที่ยอมรับได้ เปลี่ยนการดิ้นรนของผมให้เป็นแค่ตัวแปรควบคุมในสมการ ผมไม่ได้แหกคอก ผมแค่ทำตามรูทีนลดความเครียดที่สถาปัตยกรรมของเมืองอนุญาต และระบบก็ตบรางวัลความว่านอนสอนง่ายนั้นด้วยการสูบฉีดความละเอียดของผมให้สูงขึ้น

ผมคือผลผลิตระดับพรีเมียมของความสิ้นหวัง

ผมพยายามมองตรงไปข้างหน้า ทว่าหางตากลับทรยศ มันจับภาพความพร่ามัวที่ริมระเบียง บริเวณขอบเขตของวงแหวนชั้นในที่เชื่อมกับโซนรอยต่อ ชายคนหนึ่งยืนพิงกระจก เขาไม่ใช่คนในระดับเดียวกับเรา เขาคือพวกที่ค่า UX Index กำลังตกต่ำ ขอบร่างของเขาเบลอและกระตุก (Stutter) เหมือนสัญญาณแบนด์วิดท์ไม่พอ สีเสื้อผ้าซีดจางและลดทอนเฉดสีลงจนเกิดเป็นแถบสีหยาบๆ (Color banding) ดวงตาของเขาเหม่อลอย แขนที่ยกขึ้นเช็ดเหงื่อขยับช้ากว่าเวลาปกติเสี้ยววินาที ราวกับเฟรมเรตในชีวิตถูกจำกัด

นั่นคือผลลัพธ์ของ Resolution Loss อย่างค่อยเป็นค่อยไป เมื่อ The Archive ประเมินว่าคุณไร้ค่า ทรัพยากรในการมีอยู่ของคุณก็จะถูกริบกลับ 

ผมรีบหันหน้าหนี กัดฟันแน่นจนปวดกราม ความกลัวแล่นพล่านในอกเหมือนไฟกระชาก ผมไม่กล้ามอง การจ้องมองความผุพังนานเกินไปอาจทำให้อัลกอริทึมตรวจจับอารมณ์ความเห็นใจ และนั่นคือจุดเริ่มต้นของการสูญเสียความละเอียด ผมรีบจัดระเบียบแพตเทิร์นลมหายใจให้กลับมาอยู่ในโหมด ‘มีประสิทธิภาพสูงสุด’ สวมหน้ากากแห่งความสงบนิ่ง ก้าวเดินต่อไปราวกับสคริปต์ที่ไร้ข้อผิดพลาด

เบื้องหน้าของผมตอนนี้คือ สำนักซินเธซิส (Synthesis Bureau) 

มันไม่ใช่อาคาร แต่เป็นวิหารแห่งยุคหลังความตายดิจิทัล โครงสร้างเรขาคณิตของมันสมบูรณ์แบบจนเย้ยหยันฟิสิกส์กายภาพ เส้นตรงทุกเส้นพุ่งตัดอากาศอย่างไร้ที่ติ กระจกทุกบานสะท้อนแสงแดดประดิษฐ์กลับมาในมุมองศาที่คำนวณมาแล้วว่าจะไม่ทำร้ายเรตินา มันดูโอ่อ่า ยิ่งใหญ่ และไร้ซึ่งจิตวิญญาณโดยสิ้นเชิง

ผมเดินผ่านประตูโฮโลแกรมบานยักษ์ที่เลื่อนเปิดอย่างไร้เสียง ภายในโถงกว้างใหญ่ระดับบรรจุสเตเดียมโบราณได้ เสียงรองเท้ากระทบพื้นหินอ่อนสังเคราะห์ดังก้องเป็นจังหวะ ทว่าไม่มีเสียงพูดคุย ทุกคนก้มหน้าทำหน้าที่ราวกับโปรแกรมแบ็กกราวด์ 

ณ ใจกลางโถงนั่นเอง จุดที่ผมได้รับคำสั่งให้มารายงานตัว... ‘จุดสะท้อนสมบูรณ์แบบ’ (The Perfect Reflection)

มันคือกระจกควอนตัมอินเทอร์เฟซบานมหึมาตั้งตระหง่านอยู่กึ่งกลาง กระจกนี้ไม่ได้ทำงานด้วยหลักการสะท้อนแสง ทว่ามันดึงข้อมูลเชิงควอนตัมและสถานะ UX Index ของบุคคลเบื้องหน้ามาเรนเดอร์เป็นภาพตัวตนในสายตาของระบบ The Archive โดยตรง

ผมหยุดยืนเบื้องหน้ากระจก ห่างสามก้าวพอดิบพอดีตามโปรโตคอล

เงาในนั้นจ้องมองกลับมา 

ภาพสะท้อนของผม... ไร้ที่ติอย่างน่าขยะแขยง เสื้อเชิ้ตที่ผมพยายามดึงให้เบี้ยวถูกระบบเรนเดอร์กลับให้ตรงเป๊ะและตึงกริบ ผิวพรรณเปล่งประกายไร้รูขุมขน ดวงตาเป็นผลึกคมกริบ ไร้ร่องรอยของความหวาดระแวง ขอบเขตตัวตนถูกขีดเส้นด้วยความละเอียดสูงสุดเท่าที่สถาปัตยกรรมของจักรวาลนี้จะประนอมให้ มันคือ High-Res Glory ที่อาบย้อมทุกอณู 

ผมกำลังมองดูรูปปั้นเรนเดอร์ของความสมบูรณ์แบบที่มีใบหน้าเหมือนผมทุกประการ 

ผมยืนอยู่ตรงนั้น สัมผัสได้ถึงสายตาของ The Archive ที่จ้องมองมาผ่านแผ่นกระจก ชื่นชมผลงานชิ้นเอกของมัน ความรู้สึกอึดอัดตีตื้นขึ้นมาในคอ ถึงกระนั้นผมไม่สามารถขยับตัว ไม่สามารถแสดงความอ่อนแอออกมาได้แม้เพียงเสี้ยวบิต ผมถูกขังอยู่ในคุกแห่งความสมบูรณ์แบบนี้ ถูกกักขังอยู่บนจุดสูงสุดของยอดเขาที่ไม่อนุญาตให้มีความผิดพลาดใดแทรกซึม
