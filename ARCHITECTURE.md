## ⚙️ The Multi-Agent Localization Pipeline

All content localization follows a strict sequential process using specialized agent personas defined in the `souls/` directory:

1.  **Lore Verification:** Verify terminology against [terminology_lexicon.md](file:///Users/ntwkkm/primordium_city/english_localization/system_documents/terminology_lexicon.md).
2.  **Sensory Instructions:** Check environmental anchors and define the visual look of the sector.
3.  **Prose Drafting:** Translate prose to a **poetic/existential** tone.
4.  **Narrative Editing:** Verify pacing, character development milestones, and tension curves.
5.  **Micro-Continuity QA:** Proofread syntax and verify that physical indicators (e.g. Elias's limbs, wounds, or system levels) match the active chapter state.
